PHP 인터페이스 PHPDoc 주석에 대한 기본 제공 템플릿. #parse 지시문
을 이용하여 파일 템플릿 (템플릿 탭) 에 포함될 수 있는 코드 조각을 포함합니다. 템플릿은 편집할 수 있습니다. 정적 텍스트, 코드 및 주석과 마찬가지로, 매크로처럼 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${NAME}   인터페이스 이름.
${NAMESPACE}   앞에 슬래시가 없는 인터페이스 네임스페이스의 정규화된 이름
${DS}   달러 표시, '$' 기호로 표시됩니다.
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름 첫 세글자. 예: Jan, Feb 등등.
${MONTH_NAME_FULL}   현재 월의 이름 전체. 예: January, February 등등.
${DAY}   현재 일
${HOUR}   현재 시
${MINUTE}   현재 분
${SECOND}   현재 초
${PROJECT_NAME}   현재 프로젝트 이름
${USER}   현재 사용자
${CARET}   주석이 추가된 후 캐럿이 이동해야하는 위치를 표시합니다.
