ejecución celular

Ejecute rápidamente celdas de código usando estos atajos inteligentes:

  • Ctrl + Intro: Ejecuta la celda actual.

  • Mayús + Intro: Ejecuta la celda actual y selecciona las celdas debajo de ella.

  • Ctrl + Mayús + Alt + Intro: Ejecuta todas las celdas de código en Nnotebook.
