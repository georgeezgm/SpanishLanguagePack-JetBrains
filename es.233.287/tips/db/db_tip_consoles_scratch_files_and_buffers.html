데이터베이스 콘솔, 스크래치 파일 및 스크래치 버퍼

데이터베이스 콘솔 은 데이터 소스에 이미 연결된 SQL 파일입니다. 데이터 소스를 생성하면 데이터베이스 콘솔이 자동으로 생성됩니다. 콘솔을 열려면 데이터 소스를 마우스 오른쪽 버튼으로 클릭하고 탐색 | 쿼리 콘솔로 이동 | 디폴트 콘솔 열기 를 선택합니다. 새로운 콘솔을 생성하려면 데이터 소스를 마우스 오른쪽 버튼으로 클릭하고 새로 만들기 | 쿼리 콘솔 을(를) 선택합니다.

스크래치 파일 은 데이터 소스에 연결되지 않은 SQL 파일입니다. 프로젝트 컨텍스트 외부의 일부 코드 아이디어에 대해서는 스크래치 파일을 임시 노트 또는 초안으로 참조합니다. 스크래치 파일을 만들려면 파일 | 새로 만들기 | 스크래치 파일 로 이동합니다(또는 &shortcut:NewScratchFile; 누르기).

스크래치 버퍼 는 코딩 지원 기능이 없는 간단한 텍스트 파일입니다. 스크래치 버퍼는 간단한 작업 목록 및 참고 노트용으로 사용할 수 있습니다. 새 스크래치 버퍼를 생성하는 액션의 전용 메뉴 항목은 없지만, 액션 찾기 팝업(&shortcut:GotoAction; )을 사용하여 새 스크래치 버퍼 액션을 실행할 수 있습니다.
