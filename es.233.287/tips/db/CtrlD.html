Duplicate a code block or a line

Press &shortcut:EditorDuplicate; in the editor to duplicate the selected code block, or the current line when no block is selected.

If you want to have a line break between duplicated part and the original text, press &shortcut:GotoAction; and type Duplicate Entire Lines. You can assign a shortcut to this action in settings &shortcut:ShowSettings; in the Keymap menu item.
