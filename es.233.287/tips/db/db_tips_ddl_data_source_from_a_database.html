Volcar a fuente de datos DDL

Una fuente de datos DDL es una fuente de datos virtual con un esquema basado en varios scripts SQL. Almacenar estos archivos en un sistema de control de versiones (VCS) es una forma de mantener su base de datos bajo el VCS.
