Escriba la finalización del código coincidente

Las sugerencias de código de coincidencia de tipos lo ayudan a analizar la expresión completa para el tipo requerido y encontrar métodos y variables aplicables al contexto actual. Esta característica se puede utilizar en asignaciones, listas de argumentos de llamadas a métodos y otros lugares.devolver Funciona después de las palabras clave.&shortcut:SmartTypeCompletion; (código | Finalización de código | Tipo de coincidencia ) para obtener una lista filtrada de finalizaciones de código.
