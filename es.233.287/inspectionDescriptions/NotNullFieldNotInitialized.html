생성자에서 초기화되지 않았으며 NotNull 어노테이션이 추가된 필드를 보고합니다.

예:

public class MyClass {
  private @NotNull String value;

  public void setValue(@NotNull String value) {
    this.value = value;
  }

  public @NotNull String getValue() {
    return value;
  }
}

이러한 필드는 not-null 제약에 위배될 수 있습니다. 상기 예시에서 setValue 매개변수에 NotNull 어노테이션이 추가되었지만, setter가 호출되지 않은 경우 getValue가 null을 반환할 수 있습니다.

검사 구성:

  • 묵시적으로 초기화될 수 있는 필드 무시 옵션을 사용하여 필드가 묵시적으로 초기화될 경우(예: 종속성 삽입을 통해) 경고 표시 여부를 제어합니다.
  • setup() 메서드에서 초기화된 필드 무시 옵션을 사용하여 테스트 케이스 setUp() 메서드에서 필드가 작성된 경우 경고 표시 여부를 제어합니다.
