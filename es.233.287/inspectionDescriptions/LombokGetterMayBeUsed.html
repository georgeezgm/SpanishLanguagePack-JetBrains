lombok @Getter 어노테이션으로 대체될 수 있는 표준 getter 메서드를 보고합니다.

예:


  import java.util.Date;

  public class MyClass {
    /**
     * 처리 날짜.
     */
    private Date processDate;

    /**
     * 날짜를 반환함.
     *
     * @return The date
     */
    public Date getProcessDate() {
      return processDate;
    }
  }

빠른 수정/정리 적용 후:


  import lombok.Getter;
  import java.util.Date;

  @Getter
  public class MyClass {
    /**
     * 처리 날짜.
     * -- GETTER --
     * 날짜를 반환함.
     *
     * @return The date
     */
    private Date processDate;
  }

lombok 라이브러리가 구성될 때만 보고합니다. lombok을 해제하려는 경우 lombok 사이트를 참조하세요.
