이스케이프 클로저에서 필요하지 않은 한정자로서 self의 사용 위치를 보고합니다.

불필요한 self를 제거하는 것이 좋습니다.

예: 


  struct URL {
      func loadAsync() {
          print(self.path)

          DispatchQueue.global().async {
              self.loadSync()
          }
      }
  }

빠른 수정 적용 후: 


  struct URL {
      func loadAsync() {
          print(path)

          DispatchQueue.global().async {
              loadSync()
          }
      }
  }
