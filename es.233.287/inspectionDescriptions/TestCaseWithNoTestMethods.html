테스트 메서드를 사용하지 않는 비 abstract 테스트 사례를 보고합니다. Such test cases usually indicate unfinished code or could be a refactoring leftover that should be removed.

예:


  public class CrucialTest {
    @Before
    public void setUp() {
      System.out.println("setting up");
    }
  }

테스트 메서드를 사용하는 상위 클래스가 있는 테스트 사례 무시 옵션을 사용하여 테스트 메서드가 포함된 상위 클래스를 가진 테스트 사례를 무시합니다.
