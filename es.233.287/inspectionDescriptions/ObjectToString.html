java.lang.Object의 기본 구현을 사용하는 toString() 호출을 보고합니다.

디폴트 구현은 의도적으로 사용되는 경우는 드물지만 실수로 사용될 수 있습니다.

java.lang.Object 인터페이스 또는 추상 클래스 타입의 객체에서 toString()을 호출하는 경우는 이 검사에서 무시됩니다.

예:


  class Bar {
      void foo1(Bar bar) {
          String s = bar.toString(); // 경고
          /* ... */
      }

      void foo2(Object obj) {
          String s = obj.toString(); // 경고 없음
          /* ... */
      }
  }
