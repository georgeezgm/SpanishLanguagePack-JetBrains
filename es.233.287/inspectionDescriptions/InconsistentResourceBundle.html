리소스 번들에 포함된 프로퍼티 파일의 문제를 보고합니다.
  • 누락된 번역 보고

    상위 프로퍼티 파일에 포함되어 있지만 상속에서 누락된 프로퍼티를 보고하려면 이 옵션을 사용하세요(파생 언어가 아닌 경우).

    예:

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    # Empty file
    비어 있음 프로퍼티 abc가 번역되지 않은 것으로 보고됩니다.


  • 일관되지 않은 프로퍼티 보고

    상속된 프로퍼티 파일에 포함되어 있지만 상위 항목(또는 상위 항목이 없는 경우 형제 항목)에서 누락된 프로퍼티를 보고하려면 이 옵션을 사용하세요.

    예:

    
    # messages.properties
    # 빈 파일

    # messages_fr.properties
    abc=xxx
    여기에서 프로퍼티 abc 번역은 프랑스어 이외의 언어로는 제공되지 않습니다. 따라서 (디폴트) 프로퍼티 파일 messages.properties에서 누락된 것으로 보고됩니다.


  • 같은 값으로 재정의된 프로퍼티 보고

    몇몇 프로퍼티 파일에 그대로 복사하여 붙여 넣은 프로퍼티를 보고하려면 이 옵션을 사용하세요.


    예:

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    abc=xxx
    프로퍼티 abc는 파일 messages_fr.properties에서 불필요하게 상속된 것으로 보고됩니다.


  • 서로 다른 자리표시자로 재정의된 프로퍼티 보고

    재정의된 프로퍼티에서 자리표시자 일관성을 확인하려면 이 옵션을 사용하세요.

    예:

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}

    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    프로퍼티 abc는 messages.properties에 해당되지 않는 값이 있는 메시지 형식 자리표시자가 프로퍼티에 포함된 것으로 보고됩니다.


  • 서로 다른 값의 끝부분으로 재정의된 프로퍼티 보고

    재정의된 프로퍼티에서 끝부분의 일관성을 확인하려면 이 옵션을 사용하세요.

    예:

    
    # messages.properties
    abc=xxxzzz

    # messages_fr.properties
    abc=xxx;
    프로퍼티 abc는 특수 기호(! / ? / . / : / ;)로 끝나는 것으로 보고되며 messages.properties 내의 상위 값은 그렇지 않습니다.
