기본 제공 함수에 대한 대표적인 이슈를 보고합니다.

이 검사는 잘못된 인수 개수, 유효하지 않은 키워드, 잘못된 데이터 타입 등의 이슈를 보고합니다.

예(MySQL):

CREATE TABLE foo (a INT, b INT, c INT)

SELECT IFNULL() FROM foo; -- 오류
SELECT IFNULL(a) FROM foo; -- 오류
SELECT IFNULL(a, b) FROM foo; -- 문제 없음
SELECT IFNULL(a, b, c) FROM foo; -- 오류

MySQL에서 IFNULL() 함수는 엄격히 두 개의 인수를 수락합니다. 그러므로 SELECT IFNULL(a, b) FROM foo; 쿼리만 올바릅니다.
