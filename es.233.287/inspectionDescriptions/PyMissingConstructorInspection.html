클래스 내 super 생성자 호출이 누락된 경우를 보고합니다.

예:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

Pear 클래스는 __init__ 메서드 안에 super 호출을 포함해야 합니다.

빠른 수정이 적용되면 코드는 다음과 같이 변경됩니다:


class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
