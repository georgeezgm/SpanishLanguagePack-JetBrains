텍스트 블록으로 대체하여 단순화할 수 있는 String 연결을 보고합니다.

요건:

  • \n이 2번 이상 발생
  • 텍스트 블록이 연결되지 않음

단일 문자열 리터럴에 적용 옵션을 사용하면 줄 바꿈이 포함된 단일 리터럴에 대한 수정이 제안됩니다.

예:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

빠른 수정 적용 후:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

이 검사는 프로젝트 또는 모듈의 언어 수준이 15 이상인 경우에만 보고합니다.
