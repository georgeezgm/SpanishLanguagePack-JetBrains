isNotEmpty()로 바꿔야 할 Collections/Array/String 크기 검사를 보고합니다.

isNotEmpty()를 사용하면 코드가 더 단순해집니다.

빠른 수정에서는 크기 검사를 isNotEmpty()로 바꿉니다.

예:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.size > 0
  }

빠른 수정 적용 후:


  fun foo() {
      val arrayOf = arrayOf(1, 2, 3)
      arrayOf.isNotEmpty()
  }
