Java Platform Module System module-info.java 파일의 불필요한 requires 지시문을 보고합니다. 모듈 A에 모듈 B가 필요하지만 모듈 A의 코드가 B에서 패키지나 클래스를 가져오지 않는 경우, requires 지시문이 불필요합니다. 또한 모든 모듈에는 java.base 모듈에 대한 종속성이 묵시적으로 선언되어 있으므로 requires java.base; 지시문은 항상 불필요합니다.

빠른 수정에서는 불필요한 requires 지시문을 삭제합니다. 삭제된 종속성에서 실제로 사용되는 모듈을 다시 내보낸 경우, 수정 기능은 이러한 모듈에 대한 requires 지시문을 추가합니다.

이 검사는 프로젝트 또는 모듈의 언어 수준이 9 이상인 경우에만 보고합니다.
