this 또는 super의 불필요한 한정을 보고합니다.

코드 참조의 모호함을 없애기 위해 this 또는 super에 한정자를 사용하면 자동 리팩터링 수행 시 불필요해지기 쉬우므로 명확함을 위해 삭제해야 합니다.

예:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

빠른 수정 적용 후:


  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
