문자열 연결 내에서 빈 문자열 피연산자를 보고합니다. 빈 문자열 연결을 사용하여 String이 아닌 객체나 기본 유형을 String으로 변환할 수 있으나 String.valueOf() 메서드 호출을 사용하는 것이 더 명확합니다.

빠른 수정에서는 연결을 단순화하도록 제안합니다.

예:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

빠른 수정 적용 후:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

빈 문자열을 제거하려면 다른 피연산자를 String.valueOf()로 변환해야 하는 경우를 무시하려면 빈 문자열을 다른 변경 사항 없이 제거할 수 있을 때만 보고 옵션을 사용하세요.
