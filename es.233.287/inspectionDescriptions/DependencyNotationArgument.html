잘못된 종속성 표기를 탐지합니다.

다음 타입/형식은 Gradle에서 지원됩니다.
  • Dependency의 인스턴스
  • String 또는 CharSequence (예: 'org.gradle:gradle-core:1.0')
  • Maps(예: [group: 'org.gradle', name: 'gradle-core', version: '1.0'])
  • FileCollections(예: files('some.jar', 'someOther.jar'))
  • Projects(예: project(':some:project:path'))
  • ClassPathNotation(예: gradleApi())
  • 종속성 표기의 목록(예: ['org.gradle:gradle-core:1.0'])
  • (Gradle 7.0+) 버전 카탈로그 접근자(예: libs.groovy.core)
Gradle 문서를 참조하세요.

예:


dependencies {
  implementation(1) // reports '1'
}
