lombok @Setter 어노테이션으로 대체될 수 있는 표준 setter 메서드를 보고합니다.

예:


  import java.util.Date;

  public class MyClass {
    /**
     * 처리 날짜.
     */
    private Date processDate;

    /**
     * 날짜를 설정합니다.
     *
     * @param The date
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

빠른 수정/정리 적용 후:


  import lombok.Setter;
  import java.util.Date;

  @Setter
  public class MyClass {
    /**
     * 처리 날짜.
     * -- SETTER --
     * 날짜를 설정합니다.
     *
     * @param The date
     */
    private Date processDate;
  }

lombok 라이브러리가 구성될 때만 보고합니다. lombok을 해제하려는 경우 lombok 사이트를 참조하세요.
