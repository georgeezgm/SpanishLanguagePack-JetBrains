잘못된 @Profile 표현식을 보고합니다.

  • Spring 프로파일은 공백일 수 없습니다
  • Spring 프로파일 표현식에서 '&' 및 '|' 연산자는 괄호 없이 함께 사용할 수 없습니다

예:


  // 올바르게 구성됨
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  // empty 
  @Configuration
  @Profile() // "프로파일 표현식은 텍스트를 포함해야 합니다"라고 보고됩니다
  public class MyConfiguration {...}

  // mixed operators without  parentheses
  @Configuration
  @Profile("a & b | c") // "형식이 잘못된 프로파일 표현식"이라고 보고됩니다
  public class MyConfiguration {...}
