바로 다음 return 문 또는 다른 변수와 똑같은 복사본에서만 사용하는 지역 변수를 보고합니다.

그러한 변수는 안전하게 인라인화하여 코드를 더 명확하게 만들 수 있습니다.

예:


  fun sum(a: Int, b: Int): Int {
      val c = a + b
      return c
  }

빠른 수정 적용 후:


  fun sum(a: Int, b: Int): Int {
      return a + b
  }

검사 구성:

즉시 반환되거나 던져지는 변수 보고 옵션을 사용하면 즉시 반환되는 변수를 보고합니다. 변수의 이름을 명시적으로 지정하면 코드 가시성이 개선되는 경우가 있기 때문에 이 옵션은 기본적으로 비활성화되어 있습니다.
