contentEquals()로 바꿔야 하는 배열에 대한 == 또는 != 연산자의 사용을 보고합니다.

==와 != 연산자는 배열의 내용이 아닌 배열 참조를 비교합니다.

예:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // 참조 비교
  }

빠른 수정 적용 후:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
