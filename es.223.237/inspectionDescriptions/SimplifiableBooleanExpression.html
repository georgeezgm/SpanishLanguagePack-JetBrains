단순화할 수 있는 bool 표현식을 보고합니다.

예:


  void f(boolean foo, boolean bar) {
    boolean b = !(foo ^ bar);
  }

빠른 수정 적용 후:


  void f(boolean foo, boolean bar) {
    boolean b = foo == bar;
  }

예:


  void f(boolean foo, boolean bar) {
    boolean b = (foo && bar) || !foo;
  }

빠른 수정 적용 후:


  void f(boolean foo, boolean bar) {
    boolean b = !foo || bar;
  }
