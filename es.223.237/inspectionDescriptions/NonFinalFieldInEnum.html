열거형 타입의 비 final 필드를 보고합니다. 이러한 필드는 거의 불필요하며 전역 가변 상태를 제공하기 때문입니다.

예:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

빠른 수정 적용 후:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
빠른 수정에서 final로 만들 수 있는 필드만 강조 표시하려면 '빠른 수정을 사용할 수 없는 경우 필드 무시' 체크박스를 구성합니다.
