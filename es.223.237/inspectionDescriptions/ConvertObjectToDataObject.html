데이터 객체로 변환할 수 있는 객체를 보고합니다.

객체가 java.io.Serializable로 어노테이션되어 있는 경우 데이터 객체는 toString, equals, hashCode 및 readResolve를 자동으로 생성합니다.

객체를 데이터 객체로 변환하는 것을 고려해야 하는 경우는 크게 두 가지입니다. 첫 번째 경우는 사용자 지정된 toString이 클래스의 이름을 반환할 때입니다. 두 번째 경우는 객체가 java.io.Serializable를 구현할 때입니다.

예:


    object Foo {
        override fun toString(): String = "Foo"
    }

빠른 수정 적용 후:


    data object Foo

이 검사는 프로젝트 또는 모듈의 Kotlin 언어 수준이 1.8 이상인 경우에만 보고합니다
