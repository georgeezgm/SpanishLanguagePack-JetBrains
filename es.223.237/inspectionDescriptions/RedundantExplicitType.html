val f: Foo = Foo()와 같이 명백하여 불필요하게 명시적으로 지정된 지역 변수의 타입을 보고합니다.

예:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

빠른 수정 적용 후:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
