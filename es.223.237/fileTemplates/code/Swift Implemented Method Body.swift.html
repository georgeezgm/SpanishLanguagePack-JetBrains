구현된 메서드가 Code(코드) | Generate(생성) | Implement Methods(메서드 구현) 액션 등으로 생성될 때 해당 메서드의 본문을 입력합니다.
이 템플릿은 편집할 수 있습니다. Swift 표현식 및 주석과 함께 그 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${RETURN_TYPE}   생성된 메서드의 반환 타입
${DEFAULT_RETURN_VALUE}   제안된 디폴트 반환 값(빈 값일 수 있음)
${SIGNATURE}   메서드의 시그니처
Swift File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다. (예: ${USER}, ${DATE} 등)
