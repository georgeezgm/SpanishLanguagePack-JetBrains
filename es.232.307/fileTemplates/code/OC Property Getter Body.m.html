예를 들어 Code(코드) | Override Method(메서드 재정의) 액션을 호출할 때 생성되는, 프로퍼티에 대한 재정의된 getter 메서드의 본문을 생성합니다.
이 템플릿은 편집할 수 있습니다. Objective-C 표현식 및 주석과 마찬가지로, 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음 값을 사용합니다.
${CALL_SUPER}   void 메서드에 대한 super 메서드 호출 — 매개변수가 있거나 없는 [super methodName] 그 외 메서드에 대한 호출 - 매개변수가 있거나 없는 return [super methodName]
${RETURN_TYPE}   생성된 메서드의 반환 타입
${DEFAULT_RETURN_VALUE}   메서드에 의해 기본적으로 반환되는 값
${CONTAINING_CLASS}   메서드가 재정의된 클래스의 이름
${IVAR_IS_AVAILABLE}   프로퍼티에 인스턴스 변수가 있는 경우 "true"
${IVAR}   인스턴스 변수의 이름
${SETTER_SEMANTICS}   ARC 모드: copy 속성이 설정되었다면 "copyArc", 아니면 "assign"
비 ARC 모드: copy 속성이 설정되었다면 "copy", retain 속성이 설정되었다면 "retain", 아니면 "assign"
C File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다 (${USER}, ${DATE} 등)
