Editar varias filas a la vez

Seleccionar columna Para activar el modo&shortcut:EditorToggleColumnMode; Prensa . Este modo le permite aplicar el mismo cambio a varias líneas. Esto es útil, por ejemplo, si desea representar un texto como una lista y necesita incluir varias líneas en la misma etiqueta. Puede utilizar el mismo atajo para desactivar ese modo.
