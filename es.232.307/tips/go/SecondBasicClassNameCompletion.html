Finalización de código

A medida que escribe, la finalización del código sugiere opciones accesibles en la posición actual del cursor.&shortcut:CodeCompletion; Haga doble clic para encontrar todas las funciones adecuadas que acepten un valor como primer argumento.

Por ejemplo,"/hola/mundo". y luego escribe&shortcut:CodeCompletion; Al hacer doble clic se muestra una lista de funciones que toman una cadena como primer argumento.
