Las revisiones ortográficas y gramaticales están habilitadas de forma predeterminada. Estas comprobaciones las proporciona el complemento Grazie, que se incluye con el IDE. Para configurar los ajustes de revisión, abra la configuración presionando&shortcut:MostrarConfiguración; y navegar aRedactor | Lenguajes Naturales | Gramática.

Todo el texto, incluidos comentarios, cadenas de texto y literales, se compara con un diccionario predefinido o un diccionario personalizado. Los errores tipográficos se resaltan para que pueda elegir la palabra correcta, usar la ortografía actual o desactivar la verificación.

Para corregir errores ortográficos o gramaticales.&shortcut:ShowIntentionActions; Haga clic para ver la lista de acciones disponibles.
