Abrir tabla en el editor de datos

Base de datos (Base de datos) Ventana de herramientas [Ventana | Ventanas de herramientas | Base de datos Cuando hace doble clic en una tabla en [Ventana | Herramienta Windows | Base de datos], la tabla se abre en el editor de datos.

En el editor de datos, puede crear y eliminar elementos nuevos, modificar elementos existentes, filtrar datos por columnas, ver datos en un visor independiente y comparar dos tablas.
