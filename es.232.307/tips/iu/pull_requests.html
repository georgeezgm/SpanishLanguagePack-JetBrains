Administre las solicitudes de extracción entrantes de GitHub directamente desde IntelliJ IDEA. Desde el menú principalGit | GitHub | Mostrar solicitud de extracción Seleccionar . Con IntelliJ IDEA, puede asignar y fusionar solicitudes de extracción, mostrar una línea de tiempo y comentarios en línea, enviar comentarios y revisiones y aceptar cambios sin tener que salir del IDE.
