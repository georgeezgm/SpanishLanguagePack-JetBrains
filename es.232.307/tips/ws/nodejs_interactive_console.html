Consola depuradora interactiva de Node.js

Interactivo durante la depuración de aplicaciones Node.jsconsola depuradora Puede ejecutar fragmentos de código JavaScript en pestañas. La consola muestra indicaciones y proporciona la finalización del código a medida que el usuario escribe comandos.

La consola muestra una vista previa del resultado del comando, que puede expandir si es necesario.
