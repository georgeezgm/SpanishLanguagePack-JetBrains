데이터베이스 객체의 DDL, 데이터 및 Database Explorer 내 위치

선택한 객체의 DDL을 열려면 &shortcut:GotoDeclaration; 을(를) 누릅니다. 예를 들어, SELECT 문에 있는 테이블의 이름에서 이 단축키를 누르면 이 테이블의 DDL(CREATE TABLE 문)이 표시됩니다.

&shortcut:EditSource; 을(를) 눌러 객체 데이터를 엽니다. 예를 들어, SELECT 문에 있는 테이블의 이름에서 이 단축키를 누르면 테이블의 내용이 표시됩니다.

Alt+Shift+B 을(를) 눌러 Database Explorer 도구 창에서 객체로 이동합니다.
