barra de navegación

La barra de navegación esproyecto Es un reemplazo rápido de las ventanas de herramientas que le permite ver la estructura de su proyecto, abrir archivos específicos y navegar a elementos de código específicos en el archivo actual.

&shortcut:ShowNavBar; Presione para mostrar la barra de navegación y use las teclas de flecha para cambiar entre archivos y carpetas.
