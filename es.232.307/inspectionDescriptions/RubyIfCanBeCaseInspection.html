case 문으로 바꿀 수 있는 if 문을 보고합니다.

최소 브랜치 수 필드를 사용해 결과 case 문을 위한 when 브랜치의 최소 개수를 지정하십시오.

예:

fruit = gets

if fruit == 'apple' # 단순화 가능한 'if' 문
  puts 'This is apple!'
elsif fruit == 'peach'
  puts 'This is peach!'
elsif fruit == 'orange'
  puts 'This is orange!'
elsif fruit == 'banana'
  puts 'This is banana!'
end
