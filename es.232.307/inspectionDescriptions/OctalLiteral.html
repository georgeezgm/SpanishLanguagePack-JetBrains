8진수 정수 리터럴을 보고합니다. 일부 코딩 표준에서는 10진수 리터럴과 혼동될 수 있으므로 8진수 리터럴의 사용을 금지합니다.

예:


  int i = 015;
  int j = 0_777;

이 검사는 두 가지 빠른 수정을 제공합니다. Convert octal literal to decimal literal(8진수 리터럴을 10진수 리터럴로 변환) 빠른 수정을 적용한 후 코드는 다음과 같이 변경됩니다.


  int i = 13;
  int j = 511;

Remove leading zero to make decimal(선행 0을 제거하여 10진수 만들기) 빠른 수정을 적용한 후 코드는 다음과 같이 변경됩니다.


  int i = 15;
  int j = 777;
