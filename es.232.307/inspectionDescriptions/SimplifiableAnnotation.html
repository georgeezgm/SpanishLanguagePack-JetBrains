단일 요소 또는 마커 줄임 양식으로 단순화할 수 있는 어노테이션을 보고합니다.

보고된 문제:

  • 이름-값 페어 내의 불필요한 value=
  • 값이 하나 뿐인 배열 값 주위의 불필요한 중괄호
  • @ 기호와 어노테이션 이름 사이의 불필요한 공백
  • 어노테이션 이름 및 매개변수 목록 간의 불필요한 공백
  • 매개변수가 없는 어노테이션 내의 불필요한 소괄호

예:


  @interface Foo { String[] value(); }

  @ Foo({"foo"})
  public String name;

빠른 수정 적용 후:


  @interface Foo { String[] value(); }

  @Foo("foo")
  public String name;
