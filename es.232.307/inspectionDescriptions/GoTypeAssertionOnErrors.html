err.(*MyErr) 혹은 switch err.(type)와 같은 오류 타입 어설션 혹은 타입 switch를 보고하고 대신 errors.As를 사용하도록 제안합니다.

Go 1.13부터 에러는 %w 동사로 fmt.Errorf 함수를 사용하여 래핑할 수 있습니다. 따라서 오류 타입 어설션 혹은 타입 switch는 래핑된 오류에서 실패할 수 있습니다. Go 1.13 이후 표준 라이브러리의 errors.Is 함수를 사용하여 특정한 오류를 검사하는 것이 선호됩니다.

빠른 수정이 오류 타입 어설션 및 타입 switch를 errors.As 호출로 대체합니다.

예


errFoo, ok := err.(*ErrFoo)


switch err.(type) {
  case *ErrBar:
}

빠른 수정 적용 후:


var errFoo *ErrFoo
ok := errors.As(err, &errFoo)

var errBar *ErrBar
switch {
case errors.As(err, &errBar):
}
