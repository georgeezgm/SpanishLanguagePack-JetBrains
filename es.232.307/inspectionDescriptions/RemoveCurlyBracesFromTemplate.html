간단한 식별자 주변의 문자열 템플릿에서 중괄호의 사용을 보고합니다.

불필요한 중괄호를 제거하려면 '중괄호 제거' 빠른 수정을 사용하세요.

예:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== 불필요
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== 문제 없음
  }

빠른 수정 적용 후:


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== 업데이트됨
      val y = "${x.length}"
  }
