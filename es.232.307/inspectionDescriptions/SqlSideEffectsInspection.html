읽기 전용 연결 중에 데이터베이스를 변경할 수 있는 구문을 보고합니다.

연결에 대한 읽기 전용 모드를 활성화하려면 데이터베이스 도구 창(뷰 | 도구 창 | 데이터베이스)에서 데이터 소스를 오른쪽 클릭하고 프로퍼티를 선택하세요. 드라이버 및 데이터 소스 및 드라이버 대화상자에서 옵션 탭을 클릭하고 읽기 전용 체크박스를 선택하세요.

예(MySQL):

CREATE TABLE foo(a INT);
INSERT INTO foo VALUES (1);

CREATE TABLE 및 INSERT INTO 구문은 데이타베이스의 내용을 변경하기 때문에, 읽기 전용 연결 모드에서는 강조 표시됩니다.
