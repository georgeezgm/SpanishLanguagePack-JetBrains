[...] 배열 리터럴로 바꿀 수 있는 arrayOf 호출을 보고합니다.

예:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(arrayOf("alpha", "beta", "omega")) // 바꿀 수 있는 'arrayOf()'
  class MyClass

빠른 수정 적용 후:

  annotation class MyAnnotation(val strings: Array<String>)

  @MyAnnotation(["alpha", "beta", "omega"])
  class MyClass
