super(A, B) 호출이 다음 요구 사항을 충족하지 않는 경우를 보고합니다.

  • B는 A의 인스턴스입니다
  • B는 A의 하위 클래스입니다

예:


class Figure:
    def color(self):
        pass


class Rectangle(Figure):
    def color(self):
        pass


class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square는 Rectangle의 인스턴스 또는 하위 클래스가 아닙니다

수정 방안으로 Square를 Rectangle의 인스턴스로 만들 수 있습니다.
