참조 내의 불필요한 한정자를 보고합니다. 일부 한정자는 use 절이나 네임스페이스가 존재하는 경우 불필요합니다.

불필요한 한정자를 제거하기 위해 빠른 수정을 사용할 수 있습니다.

예:

use Enumerator::Generator
class Example
  # 'Enumerator::Generator' can be shortened
  def call: -> Enumerator::Generator
end

빠른 수정 적용 후:

use Enumerator::Generator
class Example
  def call: -> Generator
end
