추상 프로퍼티 또는 메서드의 일부만 하위 클래스에 정의된 경우를 보고합니다.

예:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_triangle(self):
        pass

빠른 수정이 적용되면 IDE는 Triangle 클래스의 추상 메서드를 구현합니다:


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_figure(self):
        pass

    def do_triangle(self):
        pass
