Bean 클래스에서 누락되거나 모호한 종속성이 있는 삽입점을 보고합니다.

예:


  interface Engine {}

  @Singleton
  class GasEngine implements Engine {}

  @Singleton
  class ElectricEngine implements Engine {}

  @Singleton
  class Car {

    @Inject
    Engine engine; // Dependencia ambigua: varios beans coinciden con el punto de inserción
  }
