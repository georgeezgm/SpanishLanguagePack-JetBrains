분산이 허용하지 않는 위치에서 타입 변수가 인수로 사용되는 경우 unchecked로 표시되어 있지 않는 한 보고합니다. 타입 변수는 공변성(out)이거나, 반공변성(in)이거나, 무공변성(out도 in도 아님)일 수 있습니다.

다음의 규칙이 사용되어 속성과 메서드 선언에 사용된 타입 인수를 확인합니다.

  • unchecked로 표시된 타입 인수는 무시됩니다.
  • 공변적 타입 인수는 값을 생성하는 위치에서만 사용될 수 있습니다(예: 반환 타입 혹은 블록 매개변수).
  • 반공변적 타입 인수는 인수를 소비하는 위치에서만 사용될 수 있습니다(예: 매개변수와 블록 반환 타입).
  • 무공변적 타입 인수는 값을 생성하는 위치와 인수를 취하는 위치 모두에서 사용될 수 있습니다.

예:

class Example[in A, out B]
  # 클래스 'Example'의 공변성 타입 변수 'B'가 비공변성 위치에 사용됩니다 ...
  def call: (B) -> A
end
