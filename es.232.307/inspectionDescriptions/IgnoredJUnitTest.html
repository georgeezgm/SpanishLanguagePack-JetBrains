JUnit 4의 @Ignore 또는 JUnit 5의 @Disabled 어노테이션의 사용을 보고합니다. 특히 이유가 지정되지 않은 경우 코드 스멜에 이러한 어노테이션이 추가된 테스트가 오래 있었다고 간주됩니다.

예:


  @Ignore
  public class UrgentTest {

    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

검사 구성:

  • 이유 없이 사용된 어노테이션만 보고 옵션을 사용하여 어노테이션의 value 속성으로 이유가 지정되지 않은 사례만 보고합니다.
