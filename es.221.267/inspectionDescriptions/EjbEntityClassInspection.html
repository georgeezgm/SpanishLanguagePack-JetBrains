다음과 같은 오류 수준의 EJB 사양 위반 사례를 탐지합니다.
  • Finder 및 Selector 메서드가 올바르게 정의되어 있지 않은 경우
  • CMP 및 CMR 필드 접근자가 올바르게 정의되어 있지 않은 경우

예:


public class SessionEJB implements javax.ejb.SessionBean {
  // 오류: 'ejbCreateXXX' 메서드는 public이어야 합니다
  protected void ejbCreateExample() { ... }

  // 오류: 'ejbCreateAnotherExample' 메서드는 'void'를 반환해야 합니다
  public int ejbCreateAnotherExample() { ... }
}
