default case가 다른 case 앞에 오는 switch 문 또는 식을 보고합니다.

이 구문은 불필요하게 혼동을 줄 수 있습니다. default case를 마지막 위치로 이동하는 빠른 수정이 있습니다. 이 수정은 주어진 브랜치의 끝에 break/yield가 있는 경우에만 제공됩니다.

예:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

빠른 수정 적용 후:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
