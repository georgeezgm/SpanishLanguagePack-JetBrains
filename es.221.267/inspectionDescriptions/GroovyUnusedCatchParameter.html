대응하는 블록에서 사용되지 않는 catch 매개변수를 보고합니다. 이 검사는 이름이 "ignore" 또는 "ignored"인 catch 매개변수를 보고하지 않습니다.

예:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ex) {
      println('Catching the exception')
  }

여기서 ex 매개변수는 catch 블록에서 사용되지 않습니다.

빠른 수정 적용 후:


  try {
      def arr = new int[3]
      arr[5] = 5
  } catch(Exception ignored) {
      println('Catching the exception')
  }
