첫 번째 인수가 정적으로 null이 아닌 것으로 알려진 Objects.equals(a, b) 호출을 보고합니다.

인수 모두가 기본 타입인 경우 이러한 호출은 a.equals(b) 또는 a == b로 안전하게 바꿀 수 있습니다.

예:


  String defaultName = "default";
  boolean isDefault = Objects.equals(defaultName, name);

빠른 수정 적용 후:


  String defaultName = "default";
  boolean isDefault = defaultName.equals(name);
