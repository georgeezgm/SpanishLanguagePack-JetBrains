오류 문자열의 서식 지정 이슈를 보고합니다.

예:

err := fmt.Errorf("Cannot read the file!")
log.Printf("Reading %s: %v", file, err)

golang.org의 Error Strings에 따르면 오류 문자열은 다른 컨텍스트에 나타날 수도 있으므로 대문자를 사용하거나 문장 부호로 끝나서는 안 됩니다.

서실을 수정하려면 Fix error string format(오류 문자열 서식 수정) 빠른 수정을 적용할 수 있습니다. 빠른 수정이 적용된 후 오류 문자열은 다음과 같습니다.

err := fmt.Errorf("cannot read the file")
log.Printf("Reading %s: %v", file, err)
