JSP의 텍스트(예:
  <html><body>
    Hello!
  </body></html>)를
국제화된 JSP 사용자 지정 태그로 대체합니다(예:
  <html><body>
    <bean:message key="HelloKey"/>
  </body></html>)
이 기본 제공 템플릿은 편집할 수 있습니다. JSP 컨텍스트에서 허용되는 표현식과 마찬가지로, 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${PROPERTY_KEY}   해당 프로퍼티 파일에 지정된 프로퍼티 키 이름입니다.
${PROPERTY_VALUE}   해당 프로퍼티 파일에 지정된 프로퍼티 값입니다. 원본 JSP 텍스트 선택 값입니다.
