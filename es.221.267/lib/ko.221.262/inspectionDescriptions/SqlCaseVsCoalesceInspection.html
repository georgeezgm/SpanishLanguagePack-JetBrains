CASE 및 COALESCE 호출을 서로 교체할 수 있는 경우를 보고합니다. 이 검사는 'COALESCE' 호출로 바꾸기 인텐션 액션 및 그 반대인 CASE 식으로 바꾸기 인텐션 액션을 포함합니다.

예(MySQL):

SELECT
  -- 이 CASE는 COALESCE으로 바꿀 수 있음
	CASE
		WHEN C1 IS NOT NULL THEN C1
		ELSE 0
		END
FROM dual;

이 예에서, CASE 식은 동일한 결과를 출력하는 SELECT COALESCE(C1, 0)로 바꿀 수 있습니다.

CASE 식을 사용하고 싶은 경우는, 검사 페이지에서 COALESCE 함수보다 CASE 식 선호 옵션을 선택하세요.
