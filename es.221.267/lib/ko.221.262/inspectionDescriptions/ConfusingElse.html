if—else 에서 불필요한 else 키워드 구문 및 구문 체인을 보고합니다.

이전 분기가 return, throw, break 또는 continue 문으로 끝나서 else 키워드에 도달하지 않는 경우, 이 키워드는 불필요합니다. 이 경우 else 분기의 구문은 if 문 뒤로 이동할 수 있으며 else 키워드는 제거될 수 있습니다.

예:


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

빠른 수정 적용 후:


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

if—else구문이 코드 블록의 마지막 구문인 경우를 무시하려면 'if'문 다음에 구문이 없는 경우 보고를 비활성화하세요.
