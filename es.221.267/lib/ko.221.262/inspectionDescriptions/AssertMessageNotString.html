java.lang.String 타입이 아닌 assert 메시지를 보고합니다.

문자열을 사용하면 실패 또는 어설션 이유를 진단하는 데 더 많은 정보가 제공됩니다.

예:


  void foo(List<Integer> myList) {
    assert myList.isEmpty() : false;
  }

assert 메시지 타입이 boolean 또는 java.lang.Boolean인 경우에만 경고하려면 assert 메시지 타입이 'boolean' 또는 'java.lang.Boolean'인 경우에만 경고 옵션을 사용하세요. boolean 상세 메시지는 어설션 실패에 관한 추가 정보를 제공할 가능성이 낮으며 실수로 & 대신 :를 입력할 수 있습니다.
