패키지를 문서화하기 위해 사용하는 package.html 파일을 보고합니다.

JDK 1.5 버전부터는 패키지 어노테이션까지 포함 가능한 package-info.java 파일을 사용하는 것이 좋습니다. 이러한 package-info.java는 패키지 수준의 어노테이션과 문서화를 위한 유일한 저장소가 됩니다.

예: package.html


<html>
  <body>
    Documentation example.
  </body>
</html>

빠른 수정 적용 후: package-info.java


/**
 * Documentation example.
 */
package com.sample;
