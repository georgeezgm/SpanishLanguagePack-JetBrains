Boolean 객체의 생성을 보고합니다.

Boolean 객체를 새롭게 생성하는 것은 거의 불필요하며, 빈번히 생성할 경우 성능 문제가 발생할 수 있습니다. 또한 Java 9부터 Boolean 생성자가 사용되지 않으므로 이후 Java 버전에서는 제거되거나 액세스가 불가능할 수 있습니다.

예:


  Boolean b1 = new Boolean(true);
  Boolean b2 = new Boolean(str);

빠른 수정 적용 후:


  Boolean b1 = Boolean.TRUE;
  Boolean b2 = Boolean.valueOf(str);
