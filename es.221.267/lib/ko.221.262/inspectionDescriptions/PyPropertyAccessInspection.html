프로퍼티로의 올바르지 않은 접근을 보고합니다:
  • 읽기 전용 프로퍼티가 설정됨
  • 쓰기 전용 프로퍼티가 읽힘
  • 삭제 불가한 프로퍼티가 삭제됨

예:

class MyClass:
    @property
    def read_only(self): return None

    def __write_only_setter(self, value): pass

    write_only = property(None, __write_only_setter)


a = MyClass()
a.read_only = 10 # 프로퍼티를 설정할 수 없음
del a.read_only # 프로퍼티를 삭제할 수 없음
print(a.write_only) # 프로퍼티를 읽을 수 없음
