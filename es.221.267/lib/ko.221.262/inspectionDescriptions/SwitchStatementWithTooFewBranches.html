case 라벨 수가 너무 적은 switch 구문을 보고하고, 이를 if 및 else if 구문으로 다시 작성하도록 제안합니다.

Example (minimum branches == 3):


  switch (expression) {
    case "foo" -> foo();
    case "bar" -> bar();
  }

빠른 수정 적용 후:


  if ("foo".equals(expression)) {
    foo();
  } else if ("bar".equals(expression)) {
    bar();
  }

검사 구성:

최소 분기 수 필드를 사용하여 case 라벨의 최소 개수를 지정하세요.

패턴 브랜치가 있는 switch 구문 및 식을 보고하지 않으려면 패턴 switch 구문을 보고하지 않음 옵션을 사용합니다. 예:


  String result = switch(obj) {
    case String str -> str.trim();
    default -> "none";
  };

instanceof 구문을 사용하는 것보다 단일 패턴 브랜치로도 switch를 유지하는 것이 더 선호될 수 있습니다.
