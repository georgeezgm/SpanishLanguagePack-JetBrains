CharSequence가 인수로 사용된 String.equals() 호출을 보고합니다.

String.equals()는 String 인수에 대해서만 true를 반환할 수 있습니다. String을 내용을 String이 아닌 CharSequence 인수와 비교하려면 contentEquals() 메서드를 사용하세요.

예:


  boolean equals(String s, CharSequence ch) {
    return s.equals(ch);
  }

빠른 수정 적용 후:


  boolean equals(String s, CharSequence ch) {
    return s.contentEquals(ch);
  }
