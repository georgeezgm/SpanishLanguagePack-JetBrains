Extraer variables de instancia es una refactorización de Objective-C que convierte la expresión seleccionada en una variable de instancia de una clase.

Coloque el símbolo de intercalación en una variable local o seleccione la expresión que desea refactorizar,&shortcut:IntroduceField; Prensa (Refactorización | Extraer variables de instancia).

Para crear una propiedad para una variable o declararla en una interfaz, seleccione la casilla de verificación correspondiente en la ventana emergente que se abre.
