La vista previa integrada de WebStorm muestra el resultado de su código HTML en una pestaña del editor separada. WebStorm guarda los cambios en su archivo HTML o archivos CSS y JavaScript vinculados y recarga automáticamente la vista previa.

Para abrir la vista previa, coloque el mouse sobre el código HTML para abrir una ventana emergente del navegador y luego haga clic en el ícono WebStorm.
