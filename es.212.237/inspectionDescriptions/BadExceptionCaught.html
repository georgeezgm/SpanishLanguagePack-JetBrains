부적절한 예외를 포착하는 catch 절을 보고합니다.

예를 들어, java.lang.NullPointerException 또는 java.lang.IllegalMonitorStateException 같은 일부 예외는 프로그래밍 오류를 나타내므로 프로덕션 코드에서 거의 확실하게 포착되어서는 안 됩니다.

예:


  try {
    return component.getMousePosition(true) != null;
  }
  catch (NullPointerException e) { // 경고: 금지된 예외 'NullPointerException'이 포착되었습니다
    return false;
  }

금지된 예외 목록을 사용하여 보고 대상의 예외를 지정합니다.
