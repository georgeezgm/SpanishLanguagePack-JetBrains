단일 매개변수가 있는 equal() 선언을 보고합니다. 일반적으로 이는 오타이며, 실제로 의도한 것은 equals()입니다.

메서드 이름을 'equals'로 바꾸는 빠른 수정을 사용하는 것이 좋습니다.

예:

  
    class Main {
      public boolean equal(Object obj) {
        return true;
      }
    }

빠른 수정 적용 후:

  
    class Main {
      public boolean equals(Object obj) {
        return true;
      }
    }
