재정의 메서드의 일치하는 매개변수와 다른 매개변수 이름을 보고합니다. Java에서는 허용되지만, 일관성 없는 이름은 혼란을 야기할 수 있으며 적절한 명명법에 따른 문서화의 장점이 약화될 수 있습니다.

예:


  class Person {
    Person(String fullName) {}
  }
  class Child extends Person {
    Child(String name) { super(name); }
  }

빠른 수정 적용 후:


  class Person {
    Person(String fullName) {}
  }
  class Child extends Person {
    Child(String fullName) { super(fullName); }
  }

옵션을 사용하여 단일 문자이거나 라이브러리 메서드에서 재정의된 매개변수명을 무시할 수 있는지 표시합니다. 두 가지 옵션 모두 라이브러리에서 사용된 애매모호한 이름 생성 규칙을 따르기 원치 않는 경우 유용합니다.
