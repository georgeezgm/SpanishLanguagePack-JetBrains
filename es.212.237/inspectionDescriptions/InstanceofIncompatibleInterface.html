비교된 타입이 인터페이스이며 비교된 표현식이 비교된 인터페이스를 구현하지 않고 비교된 인터페이스를 구현하는 볼 수 있는 하위 클래스가 없는 클래스 타입을 포함하는 경우 instanceof 표현식을 보고합니다.

의도된 경우도 있으나 그러한 구문은 보통 오류일 가능성이 높으며 instanceof 표현식이 항상 false로 평가됩니다.

예:


  interface I1 {}

  interface I2 {}

  interface I3 extends I1 {}

  static class Sub1 implements I1 {}

  static class Sub2 extends Sub1 implements I2 {
    void test(Sub1 sub1) {
      if (sub1 instanceof I3) { // 여기서 'I3'는 호환되지 않는 인터페이스임
      }
    }
  }
