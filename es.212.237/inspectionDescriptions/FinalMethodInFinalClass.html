final 클래스의 final 메서드를 보고합니다.

final 클래스는 상속될 수 없으므로 메서드를 final로 표시하는 것은 불필요하며 혼란을 야기합니다.

예:

record Bar(int a, int b) {
  public final sum() { 
     return a + b;
  }
}

빠른 수정 적용 후:

record Bar(int a, int b) {
  public sum() { 
     return a + b;
  }
}

예시에서 볼 수 있듯, 클래스는 명시적 혹은 묵시적으로 final로 표시될 수 있습니다.
