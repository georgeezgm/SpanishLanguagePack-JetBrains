잘못된 Bean 정의 규칙을 보고합니다.

최상위 Java 클래스는 다음 경우에 단순한 Bean이 될 수 있습니다:

  • 매개변수화된 타입이 아닌 경우
  • static이 아닌 내부 클래스가 아닌 경우
  • 구체 클래스이거나 @Decorator 어노테이션이 추가된 클래스인 경우
  • 적절한 생성자를 포함합니다: 클래스에 매개변수가 없는 생성자가 있거나 클래스가 @Initializer 어노테이션이 추가된 생성자를 선언하는 경우
  • 다음 인터페이스 중 하나를 구현하지 않는 경우: javax.servlet.Servlet, javax.servlet.Filter, javax.servlet.ServletContextListener, javax.servlet.http.HttpSessionListener, javax.servlet.ServletRequestListener, javax.ejb.EnterpriseBean
  • javax.faces.component.UIComponent를 확장하지 않는 경우
