다음을 포함해 객체를 구성하는 동안 this를 포함하는 안전하지 않은 연산을 보고합니다.
  • 클래스를 초기화하는 동안 비 final 프로퍼티에 액세스: 생성자 또는 프로퍼티 초기화에서
  • 클래스를 초기화하는 동안 비 final 함수 호출
  • 비 final 클래스의 생성자에서 this를 함수 인수로 사용

다른 클래스가 주어진 클래스를 상속받는다면 안전하지 않은 연산이 수행되는 순간에 완전히 초기화되지 않을 수 있습니다.

예:


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }

  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }

  fun testIt() {
      println(Derived(42).code) // 예상 값: 42, 실제 값: 0
  }
