Java 8부터 DocLint에서 보고된 Javadoc 주석의 동일한 HTML 이슈를 보고합니다.

이 검사는 다음과 같은 이슈를 발견합니다.

  • 닫는 태그가 불필요한 태그, 닫히지 않은 태그, 알 수 없는 태그, 위치가 잘못된 태그 또는 빈 태그
  • 알 수 없는 속성 또는 잘못된 속성
  • 잘못 배치된 텍스트

예:
