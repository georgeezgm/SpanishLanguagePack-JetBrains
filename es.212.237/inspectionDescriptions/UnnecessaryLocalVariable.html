메서드의 이해 가능성에 아무 것도 추가하지 않는 불필요한 지역 변수를 보고하며 다음을 포함합니다.
  • 즉시 반환되는 지역 변수.
  • 다른 변수로 즉시 할당되어 더는 사용되지 않는 지역 변수.
  • 다른 지역 변수 또는 매개변수와 항상 같은 값을 가지는 지역 변수.

예:


  boolean yes() {
    boolean b = true;
    return b;
  }

빠른 수정 적용 후:


  boolean yes() {
    return true;
  }

검사 구성:

  • 즉시 반환되거나 발생하는 변수 무시 옵션을 사용하여 즉시 반환되거나 발생하는 변수를 무시합니다. 일부 코드 스타일은 명확도와 디버그 용이성을 위해 그러한 변수를 사용할 것을 제안합니다.
  • 어노테이션을 가진 변수 무시 옵션을 사용하여 주석이 추가된 변수를 무시합니다.
