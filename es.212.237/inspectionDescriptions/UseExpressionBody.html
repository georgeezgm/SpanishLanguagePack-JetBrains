표현식 본문 구문으로 바꿀 수 있는 return 표현식(한 줄 또는 when)을 보고합니다.

스타일 가이드에서는 표현식 본문 구문을 권장합니다.

표현식 본문으로 변환 빠른 수정을 사용해 코드를 자동으로 수정할 수 있습니다.

예:


  fun sign(x: Int): Int {
      return when { // <== 단순화할 수 있음
          x < 0 -> -1
          x > 0 -> 1
          else -> 0
      }
  }

빠른 수정 적용 후:


  fun sign(x: Int): Int = when {
      x < 0 -> -1
      x > 0 -> 1
      else -> 0
  }
