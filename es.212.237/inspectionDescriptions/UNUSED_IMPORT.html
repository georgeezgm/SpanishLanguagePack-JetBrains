불필요한 import 문을 보고합니다.

소스 파일의 가져온 클래스 및 패키지를 사용하지 않을 경우 일반 import 문은 불필요합니다. 소스 파일에서 사용되지 않는 가져온 static 필드 및 메서드도 마찬가지입니다.

예:


  import java.util.ArrayList;
  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }

빠른 수정 적용 후: 


  public class Example {
    public static void main(String[] args) {
      System.out.println("Hello World!");
    }
  }
