JUnit 5로 자동으로 마이그레이션될 수 있는 JUnit 4 테스트를 보고합니다. 기본 러너는 자동으로 변환할 수 있지만, 사용자 지정 러너, 메서드 규칙 및 필드 규칙은 자동으로 변환할 수 없으며 수동으로 변경해야 합니다.

예:


  import org.junit.Assert;
  import org.junit.Test;

  public class RelevantTest {

    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

빠른 수정 적용 후:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;

  public class RelevantTest {

    @Test
    public void testIt() {
      Assertions.assertEquals("expected", "actual");
    }
  }

이 검사를 수행하려면 클래스 경로에서 JUnit 5 라이브러리를 사용할 수 있고 프로젝트에 대해 JDK 1.8 이상이 구성되어 있어야 합니다.
