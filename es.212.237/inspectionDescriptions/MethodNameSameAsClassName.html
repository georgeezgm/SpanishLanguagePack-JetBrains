클래스와 이름이 동일하게 지정된 메서드를 보고합니다.

Java 언어에서 이와 같은 이름 지정이 허용되긴 하지만, 대개는 생성자를 생성하려는 의도였으나 실수한 경우에 해당합니다. 이러한 이름 지정은 나쁜 사례입니다.

예:


  class MyClass {
    int val;

    // 포함하는 클래스와 이름이 동일하게 지정된 MyClass 메서드
    // 'void'도 실수로 추가되었을 수 있습니다
    void MyClass(int val) {
      this.val = val;
    }
  }

적절한 경우 빠른 수정은 메서드를 생성자로 변환합니다.


  class MyClass {
    int val;

    MyClass(int val) {
      this.val = val;
    }
  }

메서드 이름을 변경하는 다른 빠른 수정도 있습니다.
