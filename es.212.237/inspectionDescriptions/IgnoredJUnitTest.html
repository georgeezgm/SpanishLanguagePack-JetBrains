JUnit 4의 @Ignore 또는 JUnit 5의 @Disabled 어노테이션 사용을 보고합니다. 이러한 어노테이션이 장기간 특별한 이유 없이 추가된 테스트는 코드 스멜입니다.

예:


  @Ignore
  public class UrgentTest {

    @Test
    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }

검사 구성:

  • 이유없이 사용된 어노테이션만 보고 옵션을 사용하여 어노테이션의 value 속성으로 이유가 지정되지 않은 케이스만 보고합니다.
