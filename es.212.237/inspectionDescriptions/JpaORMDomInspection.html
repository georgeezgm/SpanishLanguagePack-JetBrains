`orm.xml` 구성 파일 내 모든 데이터베이스 객체로의 해결되지 않은 참조를 보고합니다:
  • 테이블
  • 열
  • 스키마
  • 카탈로그
  • 시퀀스

데이터베이스 테이블 예:



CREATE TABLE DATA_TABLE (
  ID INTEGER,
  DATA VARCHAR(100)
);

orm.xml 파일 예:



<entity-mappings xmlns="http://java.sun.com/xml/ns/persistence/orm" version="1.0">
    <package>example</package>
    <entity name="JavaEntity" class="JavaEntity">
        <table name="DATA_TABLE"/>
        <attributes>
          <id name="id">
            <column name="UNKNOWN_ID" nullable="false" length="20"/> // 오류: 해결되지 않은 열 이름
          </id>
          <basic name="data">
            <column name="DATA" length="12"/> // 좋음
          </basic>
        </attributes>
    </entity>
</entity-mappings>
