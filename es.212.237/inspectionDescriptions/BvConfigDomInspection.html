Bean Validation <validation-config> 파일 내 잘못된 요소를 보고합니다:

  • 인스턴스화할 수 없는 클래스 참조
  • 필수 클래스를 확장하지 않는 클래스 참조
  • 부적절한 스코프를 가진 클래스 참조
  • 빈 태그 및 속성 값
  • 요구되는 패턴(예: Java 식별자)과 일치하지 않는 태그 및 속성 값
  • 필요한 하위 태그 또는 속성을 포함하지 않는 태그
  • 이름이 중복된 객체를 정의하는 태그

예:


<validation-config>
  <default-provider>foo.UnknownProvider</default-provider>  <!-- 클래스 또는 패키지 'UnknownProvider'를 해결할 수 없음  -->
  <message-interpolator>my.Interpolator</message-interpolator>
  <constraint-mapping>META-INF/ConstraintMappings.xml</constraint-mapping>
  <constraint-mapping>META-INF/unknown.xml</constraint-mapping>
</validation-config>
