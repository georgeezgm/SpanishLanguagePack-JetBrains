선언된 매개변수와 실제 인수 사이의 불일치 및 잘못된 인수(예: 이름이 중복된 인수)와 잘못된 인수 순서를 보고합니다.

예:

class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__() # 채워지지 않은 매개변수
bar(5, "#") # 예기치 않은 인수

올바른 코드 조각은 다음과 같습니다:

class Foo:
    def __call__(self, p1: int, *, p2: str = "%"):
        return p2 * p1


bar = Foo()
bar.__call__(5)
bar(5, p2="#")
