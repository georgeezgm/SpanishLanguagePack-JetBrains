continue 구문이 루프에서 도달할 수 있는 마지막 구문인 경우를 보고합니다. 이러한 continue 문은 불필요하며 안전하게 제거할 수 있습니다.

예:


  for (String element: elements) {
    System.out.println();
    continue;
  }

빠른 수정 적용 후:


  for (String element: elements) {
    System.out.println();
  }

이 검사는 JSP 파일을 분석하지 않습니다.

완전한 if-else 문의 then 브랜치에 위치한 continue 문을 무시하려면 'else' 브랜치가 있는 'if' 문의 then 브랜치에서 무시 옵션을 사용하세요.

예:


  for (String element: elements) {
    if(element.isEmpty()) {
      continue;
    } else {
      //...
    }
  }
