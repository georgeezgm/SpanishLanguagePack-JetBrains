Collection<T>.count() 호출을 보고합니다.

이 함수 호출은 .size로 바꿀 수 있습니다.

.size 양식은 연산이 O(1)이며 여분의 객체가 대입되지 않도록 보장하는 반면 count()는 O(n)이며 개입이 가능한 Iterable<T>.count()와 혼동할 수 있습니다.

예:

  fun foo() {
      var list = listOf(1,2,3)
      list.count() // 바꿀 수 있는 'count()'
  }

빠른 수정 적용 후:

  fun foo() {
      var list = listOf(1,2,3)
      list.size
  }
