포함된 클래스 이름으로 한정된 static import 문의 사용을 보고합니다.

static import 문은 구성원명이 직접 접근할 수 있기 때문에 이 한정자는 필요하지 않으며 삭제할 수 있습니다.

예:

  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

빠른 수정 적용 후:

  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
