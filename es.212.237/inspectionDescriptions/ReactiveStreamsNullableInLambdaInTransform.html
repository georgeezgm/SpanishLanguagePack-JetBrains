Reactive Stream 체인에서 null을 반환할 수 있는 변환 연산자를 보고합니다.

Reactive Stream은 그러한 코드가 실패하는 원인을 제공하는 null 값을 지원하지 않습니다.

예:

repository.findWithTailableCursorBy()
    .map(e -> (Person)null)
    .doOnNext(System.out::println)

Reactive Stream 체인 코드 안에서 null 가능 값을 사용하지 않는 것이 좋지만 필요하다면 Reactor의 mapNotNull과 같은 특수 처리를 사용하세요.

map을 mapNotNull로 교체하는 빠른 수정이 있습니다. mapNotNull이 null 값을 허용하므로 예외를 생략합니다.

빠른 수정 전:

repository.findWithTailableCursorBy()
    .map(e -> (Person)null)
    .doOnNext(System.out::println)

빠른 수정 적용 후:

repository.findWithTailableCursorBy()
    .mapNotNull(e -> (Person)null)
    .doOnNext(System.out::println)
