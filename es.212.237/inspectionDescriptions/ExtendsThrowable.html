java.lang.Throwable를 직접 확장하는 클래스를 보고합니다.

java.lang.Throwable을 직접 확장하는 것은 좋지 않은 사례로 일반적으로 간주됩니다. 일반적으로 java.lang.RuntimeException, java.lang.Exception 또는 - 특별한 케이스에서 - java.lang.Error를 확장하는 것으로 충분합니다.

예:


  class EnigmaThrowable extends Throwable {} // 경고: 클래스 'EnigmaThrowable' 은 'java.lang.Throwable'을 직접 확장합니다.
