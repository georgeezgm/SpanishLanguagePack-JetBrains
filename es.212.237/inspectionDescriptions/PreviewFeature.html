미리보기 기능 API의 사용 위치를 보고합니다. Preview Feature API는 @PreviewFeature로 주석이 추가된 요소 모임입니다.

미리보기 기능은 지정과 구현이 완료된 Java 언어, Java 가상 머신, 또는 Java SE API의 새로운 기능이지만 아직 정식으로 도입되지 않았습니다. 미리보기 기능의 개념은 JEP 12 에서 정의됩니다.

코드가 미리보기 기능에 의존하는 경우 해당 기능이 긍정적인 피드백을 받지 못해 제거되거나 변경되면 이후 JDK 버전에서 컴파일이 중지될 수 있습니다.

현재 프로젝트의 언어 수준이 미리보기 수준인 경우에만 활성화됩니다.
