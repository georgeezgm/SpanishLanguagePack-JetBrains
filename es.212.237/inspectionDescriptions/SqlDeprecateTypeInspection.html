이후 버전의 DBMS에서 사용할 수 없게 될 타입의 사용 위치를 보고합니다.

보고된 타입:

  • Oracle: LONG(docs.oracle.com에서 지원이 중단되어 곧 사용할 수 없게 될 기능을 참고하세요).
  • Microsoft SQL Server: TEXT, NTEXT, IMAGE(docs.microsoft.com에서 SQL Server 2016에서 사용되지 않는 데이터베이스 엔진 기능을 참고하세요).

예(Oracle):

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS IDENTITY,
b LONG NOT NULL
);
