&&, ||, ==, != 또는 ?: 연산자를 사용하는 조건으로 바꿀 수 있는 if 문을 보고합니다.

얻어진 결과는 일반적으로 더 짧지만, 항상 더 명확하지는 않으므로, 모든 경우에 적용하는 것은 권장하지 않습니다.

예:


  if (condition) return true; else return foo;

빠른 수정 적용 후:


  return condition || foo;

검사 구성:

  • '?:' 연산자를 제안 안 함 옵션을 사용하여 ?: 연산자가 제안되는 경우 경고를 끄세요. 이 경우, &&, ||, == 및 != 제안만 강조 표시됩니다. 에디터에서 빠른 수정을 계속 사용할 수 있습니다.
  • 체인 'if' 문 무시 옵션을 사용하여 if-else 체인에 대한 경고를 끌 수 있습니다. 에디터에서 빠른 수정을 계속 사용할 수 있습니다.
