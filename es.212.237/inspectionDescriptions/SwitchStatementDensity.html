실행문 대비 switch 라벨의 비율이 너무 낮은 switch 구문 또는 표현식을 보고합니다.

그러한 switch 문은 혼란을 일으킬 수 있으므로 리팩터링되어야 할 수 있습니다.

예:


    switch (i) { // 하나의 case와 5개의 실행문 -> 20% 밀도
        case 1:
            System.out.println("1");
            System.out.println("2");
            System.out.println("3");
            System.out.println("4");
            System.out.println("5");
            break;
    }

최소 브랜치 밀도 필드를 사용하여 허용되는 실행문 대비 switch 필드 비율을 지정하세요.
