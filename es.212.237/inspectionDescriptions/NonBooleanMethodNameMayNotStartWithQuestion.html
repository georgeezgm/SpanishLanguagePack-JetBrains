이름이 의문사로 시작하는 부울이 아닌 메서드를 보고합니다. 이러한 메서드 이름은 혼란을 일으킬 수 있습니다.

라이브러리 메서드를 재정의하는 부울이 아닌 메서드는 이 검사에서 무시됩니다.

예:

  public void hasName(String name) { assert names.contains(name); }

그러한 메서드의 이름을 변경하는 빠른 수정은 에디터에서만 사용할 수 있습니다.

검사 구성:

  • 부울 메서드 이름 접두사 목록을 사용하여 부울 메서드에만 사용되어야 하는 의문사를 지정합니다.
  • 반환 타입이 java.lang.Boolean인 메서드를 무시하려면 반환 타입이 'java.lang.Boolean'인 메서드 무시 옵션을 사용합니다.
  • 상위 메서드가 있는 메서드를 무시하려면 상위 메서드를 재정의/구현하는 메서드 무시 옵션을 사용합니다.
