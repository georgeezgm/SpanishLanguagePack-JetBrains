catch 블록이 일치하는 try 블록에 따라 발생하는 예외보다도 일반적인 매개변수를 포함하는 것을 보고합니다.

예:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  }
  catch (Exception ex) { // 경고: 'Exception' 의 'catch'는 범위가 너무 광범위하고 'RuntimeException'  예외가 마스크됨
    return defaultFilePath;
  }

빠른 수정 적용 후:


  try  {
    File file = new File(pathToFile);
    return file.getAbsolutePath();
  }
  catch (RuntimeException ex) {
    return defaultFilePath;
  }

검사 구성:

  • RuntimeException,Exception,Error 또는 Throwable 에 대한 경고 옵션을 사용하여 이 검사로 가장 일반적인 예외에 대해서만 경고합니다.
  • 다른 예외를 감추지만 자체적으로 발생하는 예외 무시 옵션을 사용하여 다른 예외를 감추지만 여전히 발생할 가능성이 있기 때문에 기술적으로 너무 광범위하지 않은 예외를 무시합니다.
