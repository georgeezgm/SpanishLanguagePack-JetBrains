자동으로 추론할 수 있는 유형 인수를 포함하는 함수 호출을 보고합니다. 그러한 유형 인수는 불필요하며 안전하게 생략할 수 있습니다.

코드를 정리하려면 '명시적 유형 인수 제거' 빠른 수정을 사용하세요.

예:


  // 여기서 'String' 유형을 추론할 수 있음
  fun foo(): MutableList<String> = mutableListOf<String>()

  // 여기서는 'String'을 추론할 수 있으며 유형 인수가 필요합니다.
  fun bar() = mutableListOf<String>()

빠른 수정 적용 후:


  fun foo(): MutableList<String> = mutableListOf() <== Updated

  fun bar() = mutableListOf<String>()
