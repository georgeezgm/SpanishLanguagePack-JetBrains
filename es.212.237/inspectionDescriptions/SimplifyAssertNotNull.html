선언된 변수의 null이 아닌 값을 검사하는 assert 호출을 보고합니다.

!! 또는 ?:를 사용하면 코드가 더 단순해집니다.

빠른 수정이 변수 이니셜라이저에서 assert를 !! 또는 ?: 연산자로 바꿉니다.

예:


  fun foo(p: Array<String?>) {
      val v = p[0]
      assert(v != null, { "Should be not null" })
  }

빠른 수정 적용 후:


  fun foo(p: Array<String?>) {
      val v = p[0] ?: error("Should be not null")
  }
