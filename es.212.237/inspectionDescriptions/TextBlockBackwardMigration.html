일반 문자열 리터럴로 대체 가능한 텍스트 블록을 보고합니다.

Java 15 이전의 Java 버전으로 코드 마이그레션을 수행한 결과로 해당 텍스트 블록이 표시될 수 있습니다.

예:


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

빠른 수정 적용 후:


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

이 검사는 구성된 언어 수준이 15 이상인 경우에만 보고합니다.
