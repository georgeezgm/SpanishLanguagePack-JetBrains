축약될 수 있는 정규화된 클래스 이름을 보고합니다.

빠른 수정은 정규화된 이름을 축약하고 필요한 경우 import 문을 추가합니다.

예:


  class ListWrapper {
    java.util.List<Integer> l;
  }

빠른 수정 적용 후:


  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

검사 구성:

Java 9 provides 및 uses 모듈 구문 내에서 정규화된 이름을 무시하려면 Java 9 모듈 구문 무시 옵션을 사용하세요.

검사를 구성하려면 설정 | 에디터 | 코드 스타일 | Java | 가져오기에서 다음 옵션을 사용하세요:

  • 내부 클래스에 대한 참조가 외부 클래스와 정규화되어야 하는 경우 내부 클래스에 대한 import 문 삽입 옵션을 사용하세요.
  • JavaDoc에서 정규화된 클래스 이름 사용 옵션을 사용하여 Javadoc에서 정규화된 이름을 허용하세요.
