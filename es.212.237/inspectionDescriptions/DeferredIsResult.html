kotlinx.coroutines.Deferred 결과를 포함하는 함수를 보고합니다.

Deferred를 반환 유형으로 사용하는 함수는 Async 접미사를 포함하는 이름을 가지고 있어야 합니다. 그렇지 않으면 함수를 suspend로 표시하고 내부에 있는 Deferred를 언래핑하는 것을 추천합니다.

예:


  fun calcEverything(): Deferred<Int> {
      return CompletableDeferred(42)
  }

수정 적용 후(Async 접미사 추가):


  fun calcEverythingAsync(): Deferred<Int> {
      return CompletableDeferred(42)
  }

수정 적용 후(함수를suspend 함수로 변환):


  suspend fun calcEverything(): Int {
      return CompletableDeferred(42).await()
  }
