static으로 선언되거나, public으로 선언되지 않거나, void를 반환하지 않거나 매개변수가 있는 JUnit 테스트 메서드를 보고합니다.

해당 메서드는 실수로 생성되기 쉽지만 JUnit 테스트 러너를 통해 실행되지 않습니다.

예:


  @Test
  void test() {
    assertEquals(1, 1);
  }

빠른 수정 적용 후:


  @Test
  public void test() {
    assertEquals(1, 1);
  }

검사 옵션을 사용하여 어노테이션을 지정합니다. 해당 어노테이션 중 하나가 추가된 매개변수는 보고되지 않습니다.
