지역 변수에서 val f: Foo = Foo() 같이 명백하고 따라서 불필요한 명시적 유형을 보고합니다.

예:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

빠른 수정 적용 후:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
