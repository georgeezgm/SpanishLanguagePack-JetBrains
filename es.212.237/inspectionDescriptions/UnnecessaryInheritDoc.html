{@inheritDoc} 태그만을 포함하는 Javadoc 주석을 보고합니다. 주석이 없는 경우 Javadoc은 슈퍼 클래스의 주석을 복사하므로 {@inheritDoc} 만을 포함하는 주석은 어느 것도 추가하지 않습니다.

또한 유효하지 않은 위치(예: 필드)의 {@inheritDoc} 사용을 보고합니다.

불필요한 Javadoc 주석의 제거를 권장합니다.

예:

  class Example implements Comparable<Example> {
    /**
     * {@inheritDoc}
     */
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }

빠른 수정 적용 후:

  class Example implements Comparable<Example> {
    @Override
    public int compareTo(Example o) {
      return 0;
    }
  }
