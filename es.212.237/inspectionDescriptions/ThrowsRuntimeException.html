메서드의 throws 절에 체크하지 않은 예외(java.lang.RuntimeException 또는 하위 클래스 중 하나)의 선언을 보고합니다.

체크하지 않은 예외의 선언은 필요하지 않으며 삭제하거나 Javadoc의 @throws 태그로 이동할 수 있습니다.

예:


  public class InvalidDataException extends RuntimeException {}

  class TextEditor {
    void readSettings() throws InvalidDataException {} // 경고: 'throws' 절에서 선언된 체크하지 않은 예외
  }
