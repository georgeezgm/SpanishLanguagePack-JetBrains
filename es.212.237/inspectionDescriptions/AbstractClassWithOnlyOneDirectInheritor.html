직접 상속자가 하나만 있는 추상 클래스를 보고합니다. 이러한 클래스는 메모리가 제한적이거나 대역폭이 제한된 환경에서 훌륭한 디자인 명확도를 제공할 수 있지만, 애플리케이션의 총 사용 공간을 불필요하게 증가시킵니다. 추상 클래스를 해당 상속자와 병합하는 것을 고려하세요.

이 검사는 Java ME 및 리소스가 많이 제한된 기타 환경을 위한 것입니다. 이 검사 결과를 고려하지 않고 적용하면 코드 명확도와 디자인에 부정적인 영향을 미칠 수 있습니다.

예:


  abstract class Base {} // 보고됩니다

  class Inheritor extends Base {}
