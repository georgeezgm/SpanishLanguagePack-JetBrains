잘못된 Spring 컨텍스트 구성을 보고합니다.

  • locations 속성 및 @ContextConfiguration 메타 어노테이션에 해당하는 별칭 속성의 확인되지 않은 파일 및 디렉터리를 보고합니다.
  • 누락된 기본 애플리케이션 컨텍스트 파일

자세한 내용은 @ContextConfiguration을 참조하십시오.

예:


  @ContextConfiguration(locations = "classpath:META-INF/unknown-context.xml") // reports "Cannot resolve file 'unknown-context.xml'"
  class MyTests {...}
