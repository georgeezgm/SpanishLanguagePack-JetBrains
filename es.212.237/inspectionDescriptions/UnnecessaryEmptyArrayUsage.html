배열 요소 타입의 클래스에 상수가 있을 경우 알려진 길이가 0인 배열의 대입을 보고합니다. 길이가 0인 배열은 불변이므로 동일한 배열 인스턴스를 재사용하여 메모리를 절약할 수 있습니다.

예:


  class Item {
    // 재사용 가능한 길이가 0인 배열의 Public 상수
    public static final Item[] EMPTY_ARRAY = new Item[0];
  }
  class EmptyNode {
    Item[] getChildren() {
      // 길이가 0인 배열의 불필요한 생성
      return new Item[0];
    }
  }

빠른 수정 적용 후:


  class EmptyNode {
    Item[] getChildren() {
      return Item.EMPTY_ARRAY;
    }
  }
