빈 기본 생성자를 묵시적으로 사용할 수 있는 경우 그러한 생성자를 보고합니다.

어노테이션 또는 가시성 제어자가 없는 기본 생성자는 불필요하며 생략해도 안전합니다. 코드를 정리하려면 '빈 기본 생성자 제거' 빠른 수정을 사용하세요.

예:


  class MyClassA constructor() //  불필요, 'class MyClassA'로 바꿀 수 있음

  annotation class MyAnnotation
  class MyClassB @MyAnnotation constructor() //  어노테이션 때문에 필요함

  class MyClassC private constructor() // 가시성 제어자 때문에 필요함
