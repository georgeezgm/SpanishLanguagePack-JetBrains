컬렉션 및 String에 대한 부정 isEmpty() 및 isNotEmpty() 또는 String에 대한 isBlank() 및 isNotBlank()을 보고합니다.

대응하는 함수를 사용하면 코드가 더 단순해집니다.

빠른 수정이 부정 호출을 표준 라이브러리에서 대응하는 호출로 바꿉니다.

예:


  fun main() {
      val list = listOf(1,2,3)
      if (!list.isEmpty()) {
          // 뭔가 함
      }
  }

빠른 수정 적용 후:


  fun main() {
      val list = listOf(1,2,3)
      if (list.isNotEmpty()) {
          // 뭔가 함
      }
  }
