조건이 일정하여 표현식을 단순화할 수 있는 if 표현식을 보고합니다.

의도적일 때가 가끔 있지만 이러한 구조는 혼란을 초래할 수 있으며 종종 오타가 난 결과입니다.

예:


  fun example() {
      if (true) {
          throw IllegalStateException("Unexpected state")
      }
  }

빠른 수정을 위해 if 조건을 제거하도록 제안합니다.


  fun example() {
      throw IllegalStateException("Unexpected state")
  }
