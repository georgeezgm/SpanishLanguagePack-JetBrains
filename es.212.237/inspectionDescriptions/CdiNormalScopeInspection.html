디폴트 @Dependent 스코프 안에 있지 않은 public 인스턴스 필드를 포함하는 매니지드 Bean을 보고합니다. CDI 사양에 따르면, 컨텍스트 스코프(@RequestScoped, @ApplicationScoped, @SessionScoped, ConversationScoped)는 public 인스턴스 필드를 포함하는 매니지드 Bean에 허용되지 않습니다.

예:


  @ApplicationScoped // 경고: 디폴트가 아닌 스코프를 가진 Bean은 public 필드를 포함할 수 없음
  class Car {
    public String engine;
  }
