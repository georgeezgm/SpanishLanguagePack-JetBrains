finally 블록 내에서 사용되는 return, throw, break, continue 및 yield 구문을 보고합니다. 이로 인해 finally 블록이 정상적으로 완료되지 않고 갑자기 완료됩니다. 동일한 try-catch 구문의 try 및 catch 블록에서 던져진 모든 예외는 억제됩니다.

예:


  void x() {
    try {
      throw new RuntimeException();
    } finally {
      // bar()가 true를 반환하는 경우 RuntimeException이 억제됩니다
      if (bar()) return;
    }
  }
