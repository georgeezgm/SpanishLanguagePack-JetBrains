다른 클래스를 너무 많이 참조하는 메서드를 보고합니다. 참조된 각 클래스는 참조 횟수에 관계없이 한 번만 집계됩니다.

결합이 너무 지나친 메서드는 매우 취약할 수 있으며, 더 작은 메서드로 분할해야 할 수 있습니다.

검사 구성:

  • 메서드 결합 제한 필드를 사용하여 메서드에 허용되는 최대 결합을 지정합니다.
  • Java 시스템 클래스에 결합 포함 옵션을 사용하여 java 또는 javax 패키지의 클래스에 대한 참조를 집계합니다.
  • 라이브러리 클래스에 결합 포함 옵션을 사용하여 타사 라이브러리 클래스에 대한 참조를 집계합니다.
