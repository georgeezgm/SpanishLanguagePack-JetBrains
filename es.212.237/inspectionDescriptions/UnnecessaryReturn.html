void를 반환하는 생성자 및 메서드 끝에 위치한 return 문을 보고합니다. 이러한 구문은 불필요하며 안전하게 제거할 수 있습니다.

이 검사는 JSP 파일에서 보고하지 않습니다.

예:


  void message() {
    System.out.println("Hello World");
    return;
  }

빠른 수정 적용 후: 


  void message() {
    System.out.println("Hello World");
  }

else 브랜치도 포함하는 if 문의 then 브랜치 내 return 문을 무시하려면 'else' 브랜치가 있는 'if' 문의 then 브랜치에서 무시 옵션을 사용하세요.
