val 또는 var를 제거할 수 있는 기본 생성자 매개변수를 보고합니다.

생성자에 선언된 클래스 프로퍼티는 추가 메모리를 사용합니다. 매개변수 값이 생성자에서만 사용되는 경우 일반적으로 매개변수 값을 저장할 이유가 없습니다.

참조된 객체는 조기에 쓰레기 수집될 수 있습니다.

예:


  class Task(val name: String) {
      init {
          print("Task created: $name")
      }
  }

빠른 수정을 위해 추가 val 또는 var 키워드를 제거하도록 제안합니다.


  class Task(name: String) {
      init {
          print("Task created: $name")
      }
  }
