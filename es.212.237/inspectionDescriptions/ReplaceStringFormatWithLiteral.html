문자열 템플릿으로 바꿀 수 있는 String.format 호출을 보고합니다.

문자열 템플릿을 사용하면 코드가 더 단순해집니다.

빠른 수정이 호출을 문자열 템플릿으로 바꿉니다.

예:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = String.format("%s_%s_%s", id, date, id)
  }

빠른 수정 적용 후:


  fun main() {
      val id = "abc"
      val date = "123"
      val s = "${id}_${date}_$id"
  }
