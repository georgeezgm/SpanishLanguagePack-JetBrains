다음과 같은 Javadoc 주석 결함을 보고합니다.
  • Javadoc이 필요한 곳에 없음
  • 필수 태그 누락
  • 잘못되거나 불완전한 태그
  • 누락되거나 불완전한 Javadoc 설명

예:


  class Main {
  }

빠른 수정 적용 후:


  /**
   *
   */
  class Main {
  }

예:


  String id(String value) {
    return value;
  }

빠른 수정 적용 후:


  /**
   * @param value
   * @return
   */
  String id(String value) {
    return value;
  }
탭 및 해당하는 검사 옵션을 사용하여 Javadoc이 필요한 가시성 범위 및 클래스, 메서드, 필드 및 내부 클래스에 필요한 Javadoc 태그를 정의합니다.
