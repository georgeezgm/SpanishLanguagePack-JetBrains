불필요한 코드 블록이 있는 switch 구문 또는 switch 표현식의 라벨 지정된 규칙을 보고합니다.

예:


  String s = switch (n) {
    case 1 -> { yield Integer.toString(n); }
    default -> "default";
  };

빠른 수정 적용 후:


  String s = switch (n) {
    case 1 -> Integer.toString(n);
    default -> "default";
  };

이 검사는 프로젝트 또는 모듈의 언어 수준이 14 이상인 경우에만 보고합니다.
