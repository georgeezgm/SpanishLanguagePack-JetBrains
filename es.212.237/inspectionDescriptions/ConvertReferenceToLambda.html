함수 리터럴(람다)로 바꿀 수 있는 함수 참조 표현식을 보고합니다.

람다를 전달하는 것이 더 직관적이며 나머지 코드와 비교해 더 일관성이 있는 것처럼 보일 수 있습니다. 또한 간단한 호출을 더 복잡한 다른 것으로 교체해야 한다면 수정이 더 편할 수 있습니다.

예:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter(Int::isEven)
  }

수정 적용 후:


  fun Int.isEven() = this % 2 == 0

  fun example() {
      val numbers = listOf(1, 2, 4, 7, 9, 10)
      val evenNumbers = numbers.filter { it.isEven() }
  }
