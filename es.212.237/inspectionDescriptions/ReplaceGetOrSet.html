색인 연산자 []로 바꿀 수 있는 get 또는 set 함수에 대한 명시적 호출을 보고합니다.

Kotlin에서는 유형에서 사전 정의된 세트에 대한 사용자 정의 구현을 허용합니다. 연산자를 오버로드하기 위해 operator 제어자를 포함하는 함수를 표시할 수 있습니다. 


  operator fun get(index: Int) {}
  operator fun set(index: Int, value: Int) {}
위의 함수는 색인 연산자에 대응합니다.

예:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test().get(0) // 바꿀 수 있는 'get()'
  }

빠른 수정 적용 후:

  class Test {
      operator fun get(i: Int): Int = 0
  }

  fun test() {
      Test()[0]
  }
