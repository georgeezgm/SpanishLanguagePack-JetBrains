java.util.Collection 또는 java.util.Map의 메서드 호출 인수가 컬렉션 또는 맵 자체인 경우를 보고합니다. 자체적으로 컬렉션을 추가하는 경우도 이에 포함됩니다. 예를 들어, 자체 포함 컬렉션에서 hashCode()를 호출하여 java.lang.StackOverflowError가 발생하는 경우가 있습니다. 해당 상황은 복사-붙여넣기의 결과로 원시 타입이 있는 코드에서 발생할 수 있습니다.

예:


  ArrayList list = new ArrayList<>();
  list.add(list);
  return list.hashCode(); // StackOverflowException을 던집니다
