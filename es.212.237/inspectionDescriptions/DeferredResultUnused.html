반환 값을 사용하지 않는 경우 Deferred 결과 타입을 포함하는 호출을 보고합니다.

반환 값이 사용되는 경우 함수가 완료될 때까지 기다리는 것은 없습니다.

예:


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      calcEverythingAsync()
  }

빠른 수정을 위해 Deferred 이니셜라이저를 사용하여 변수를 삽입하도록 제안합니다.


  fun calcEverythingAsync() = CompletableDeferred(42)

  fun usage() {
      val answer = calcEverythingAsync()
  }
