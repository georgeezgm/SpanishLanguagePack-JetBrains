주석이 있는 메서드, 주석이 있는 메서드의 슈퍼 메서드, 또는 주석이 있는 클래스를 참조하는 Javadoc의 @see / {@link} / {@linkplain} 태그를 보고합니다.

관련 링크는 불필요하며 검사의 빠른 수정을 통해 안전하게 제거할 수 있습니다. 빠른 수정은 자체 내용을 참조하는 전체 Javadoc 주석을 제거합니다.

예:

  class Example {
    /**
     * @see Example#method
     */
    public void method() { }
  }

빠른 수정 적용 후:

class Example {
  public void method() { }
}

아래의 체크박스를 사용하여 슈퍼 메서드의 인라인 링크({@link} 및 {@linkplain})를 무시합니다. 모든 슈퍼 메서드의 링크는 Javadoc 도구에 자동으로 추가되지만, 슈퍼 메서드의 인라인 링크는 Javadoc 주석의 텍스트 내에서 필요한 경우가 있습니다.
