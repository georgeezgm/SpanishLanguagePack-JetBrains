boxed를 보고합니다. Range.start 및 Range.endInclusive 프로퍼티입니다.

이들 프로퍼티는 불필요한 호출을 피하기 위해 unboxedfirst 및 last 프로퍼티로 바꿀 수 있습니다.

빠른 수정이 start 및 endInclusive 프로퍼티를 대응하는 first 및 last로 바꿉니다.

예:


  fun foo(range: CharRange) {
      val lastElement = range.endInclusive
  }

빠른 수정 적용 후:


  fun foo(range: CharRange) {
      val lastElement = range.last
  }
