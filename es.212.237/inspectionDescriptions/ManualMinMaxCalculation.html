계산을 수동으로 수행하는 대신 Math.max() 또는 Math.min()을 사용하여 최소 또는 최대 두 개의 숫자를 계산할 수 있는 경우를 보고합니다.

예:

  public int min(int a, int b) {
    return b < a ? b : a;
  }

빠른 수정 적용 후:

  public int min(int a, int b) {
    return Math.min(a, b);
  }

float 및 double 비활성화 옵션을 사용하여 double 및 float에 대한 검사를 비활성화합니다. 이 옵션은 NaN을 처리할 경우 빠른 수정이 float/ double의 의미를 약간 변경할 수 있으므로 유용합니다. 그럼에도 이 옵션은 대부분의 경우 사용자가 NaN을 신경 쓰지 않는다면 사소한 버그를 수정합니다.
