다른 표현식 내에서 중첩된 증분 또는 감소 표현식을 보고합니다.

이러한 표현식은 혼란을 일으킬 수 있으며 모든 구문은 정확히 한 가지 작업을 수행해야 한다는 일반적인 디자인 규칙을 위반합니다.

빠른 수정은 증분 또는 감소 연산을 별도의 표현식 구문으로 추출합니다.

예:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

빠른 수정 적용 후:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
