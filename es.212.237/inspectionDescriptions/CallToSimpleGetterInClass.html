프로퍼티의 클래스 내에서 간단한 프로퍼티 getter 호출을 보고합니다.

간단한 프로퍼티 getter는 단순히 필드의 값을 반환하는 프로퍼티로 정의되어 있으며, 다른 계산은 수행하지 않습니다. 이러한 간단한 getter 호출은 빠른 수정을 사용하여 안전하게 인라인화할 수 있습니다. 또한 코드 명확성을 위해 일부 코딩 표준에서는 간단한 getter 사용이 권장되지 않습니다.

예:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return getName();
    }
  }

빠른 수정 적용 후:


  public class Salient {
    private String name;

    public String getName() {
      return name;
    }

    @Override
    public String toString() {
      return name;
    }
  }

다음 옵션을 사용해 검사를 구성하세요:

  • 매개변수로 전달된 동일한 타입의 객체가 아닌 this에 대한 getter 호출만 보고할지 여부.
  • 비 private getter를 무시할지 여부.
