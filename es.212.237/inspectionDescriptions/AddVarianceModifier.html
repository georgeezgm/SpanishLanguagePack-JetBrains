in 또는 out 변화를 가질 수 있는 유형 매개변수를 보고합니다.

in 및 out 변화를 사용하면 Kotlin에서 더 정확한 유형 인터페이스와 더 깔끔한 코드 의미를 제공합니다.

예:


  class Box<T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) {
      consumeString(box)
      consumeCharSequence(box) // 컴파일 오류
  }

빠른 수정을 위해 일치하는 분산 제어자를 추가하도록 제안합니다.


  class Box<out T>(val obj: T)

  fun consumeString(box: Box<String>) {}
  fun consumeCharSequence(box: Box<CharSequence>) {}

  fun usage(box: Box<String>) ++{
      consumeString(box)
      consumeCharSequence(box) // 괜찮음
  }
