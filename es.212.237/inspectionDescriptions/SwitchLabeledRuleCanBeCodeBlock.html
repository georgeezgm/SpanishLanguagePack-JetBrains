코드 블록으로 변환할 수 있는 switch 표현식 또는 향상된 switch문의 규칙을 보고합니다.

예:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
   ...
  };

빠른 수정 적용 후:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
   ...
  };

이 검사는 프로젝트 또는 모듈의 언어 수준이 14 이상인 경우에만 보고합니다.
