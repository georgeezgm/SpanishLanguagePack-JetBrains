두 배열을 비교하기 위한 equals() 호출을 보고합니다.

배열에서 equals()를 호출하면 ID가 비교되며 이는 ==를 사용하는 것과 동등합니다. 두 배열의 내용을 비교하려면 Arrays.equals()를 사용하세요. 혹은 다차원 배열을 비교하려면 Arrays.deepEquals()를 사용하세요.

예:

  void sample(int[] first, int[] second){
    if (first.equals(second)) return;
  }

빠른 수정 적용 후:

  void sample(int[] first, int[] second){
    if (Arrays.equals(first, second)) return;
  }
