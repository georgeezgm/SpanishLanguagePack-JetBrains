접근 제어자(package-private이라고도 부름)없이 선언되는 중첩 클래스를 보고합니다.

예:


 public class Outer {
    static class Nested {}  // 경고
    class Inner {}          // 경고
    enum Mode {}            // 설정에 따른 경고
    interface I {}          // 설정에 따른 경고
  }

검사 구성:

  • package-visible 내의 열거형 무시 옵션을 사용하여 package-private 내의 열거형을 무시합니다.
  • package-visible 내의 인터페이스 무시 옵션을 사용하여 package-private 내의 인터페이스를 무시합니다.
