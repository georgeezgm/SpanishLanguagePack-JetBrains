this 또는 super 의 필요하지 않은 자격을 보고합니다.

코드 참조를 명확하게 하는 this 또는 super 의 한정자는 자동 리팩터링을 사용하여 간단하게 불필요하게 만들 수 있으며, 명확함을 위해 삭제해야 합니다.

예:

  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      Bar.super.foo();
    }
  }

빠른 수정 적용 후:

  class Foo {
    void foo() {}
  }

  class Bar extends Foo {
    void bar() {
      super.foo();
    }
  }
