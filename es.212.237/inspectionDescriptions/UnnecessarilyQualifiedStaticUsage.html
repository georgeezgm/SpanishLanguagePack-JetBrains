클래스명으로 한정된 static 구성원의 사용을 보고합니다.

이러한 한정은 불필요하며 안전하게 생략할 수 있습니다.

예:

  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      Foo.foo();
      System.out.println(Foo.x);
    }

    static void baz() { Foo.foo(); }
  }

빠른 수정 적용 후:

  class Foo {
    static void foo() {}
    static int x;

    void bar() {
      foo();
      System.out.println(x);
    }

    static void baz() { foo(); }
  }

보고 대상을 전환하려면 검사 옵션을 사용하세요.

  • static 필드 접근:
    void bar() { System.out.println(Foo.x); }

  • static 메서드 호출:
    void bar() { Foo.foo(); }

또한 static 구문에서 구성원 사용을 보고하기 위해 검사를 구성할 수 있습니다. 이 경우 static void baz() { Foo.foo(); } 만 보고됩니다.
