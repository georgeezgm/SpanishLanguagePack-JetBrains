로깅 구문을 포함하지 않는 모든 public 메서드를 보고합니다. 이 검사는 단순한 getter 및 setter를 보고하지 않습니다.

예:


  public class Crucial {
    private static finalLogger LOG = LoggerFactory.getLogger(Crucial.class);
    public void doImportantStuff() {
      // 이 메서드에서 경고 발생
    }

    public void doOtherStuff() {
      LOG.info("do other stuff");
    }
  }

아래 테이블을 사용하여 Logger 클래스 이름을 지정하세요. 이 검사는 특정한 클래스의 인스턴스 메서드를 사용하지 않는 public 메서드를 보고합니다.
