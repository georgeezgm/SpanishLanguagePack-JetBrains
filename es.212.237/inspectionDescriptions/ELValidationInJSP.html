해결되지 않은 참조 및 유효하지 않은 EL 위치 등 가능한 EL 문제를 보고합니다.

예:


현재 네임스페이스에 정의된 함수 `exampleFun(String)`를 포함하는 방안을 고려하세요. 

<jsp:root xmlns:example="urn:jsptld:exampleNamespace.tld">
  ${ example:exampleFun("1") + example:unknownFun("2") } // 오류: 함수 'unknownFun'을 해결할 수 없음
  ${ example:exampleFun("1", "2") } // 오류: 형식 매개변수와 실제 매개변수의 개수가 다름
</jsp:root>
