@StreamListener, @SendTo, @Output 및 @Input 어노테이션의 확인되지 않은 채널 속성을 보고합니다.

잘못된 바인딩 이름을 사용하면 런타임에서 'Bean을 찾을 수 없음' 오류가 발생합니다.

예:


  @EnableBinding(Sink.class)
  public class LogSource {
    @StreamListener(Sink.class)
	  public void log1() {
    }

    @StreamListener("invalid")  // Reports 'Cannot find channel'
	  public void log2() {
    }
  }
