파일 또는 소켓 리소스에서 getChannel()를 호출하여 생성된 모든 인스턴스를 포함하여 안전하게 닫히지 않은 Channel 리소스를 보고합니다.

이 검사는 기본적으로 리소스가 이름에 'close' 또는 'cleanup'을 포함하는 모든 메서드에 의해 닫힐 수 있다고 가정합니다.

예:


  void send(Socket socket) throws IOException {
    SocketChannel channel = socket.getChannel(); // 경고
    channel.write(ByteBuffer.wrap("message".getBytes()));
  }

다음 옵션을 사용해 검사를 구성하세요:

  • 리소스를 Channel 블록 내부에서 여는 것을 허용할지 여부를 설정합니다. 이 스타일은 try 블록 앞에서 Channel을 여는 것보다 장황하기 때문에 그다지 바람직하지 않습니다.
  • 리소스가 인수로 전달된 메서드 호출에 의해 리소스가 닫힐 수 있는지 여부를 설정합니다.
