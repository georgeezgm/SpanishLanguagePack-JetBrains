메서드 매개변수로 선언된 변수에 대입한 경우를 보고합니다. 또한 변수를 증분 또는 감소하려는 시도를 보고합니다.

의도된 경우도 있으나 이 구문은 혼란스러울 수 있으므로 일부 Java 프로젝트에서 금지됩니다.

빠른 수정은 새로운 변수 선언을 추가합니다.

예:


  void printTrimmed(String s) {
    s = s.trim();
    System.out.println(s);
  }

빠른 수정 적용 후:


  void printTrimmed(String s) {
    String trimmed = s.trim();
    System.out.println(trimmed);
  }

대입이 원본 매개변수의 변형인 경우 무시 옵션을 사용하여 이전 값에 따라 매개변수 값을 수정하는 대입을 무시합니다.
