"magic" 상수로 대체 가능한 표현식을 보고합니다.

예시 1:

  // 단순 리터럴 "2"가 사용되었습니다, 경고:
  Font font = new Font("Arial", 2)

예시 2:

  // 사전 정의된 상수가 사용되었습니다, 적합:
  Font font = new Font("Arial", Font.ITALIC)

가능한 경우 빠른 수정은 사전 정의된 적절한 상수를 삽입합니다.

이 검사의 해당 동작은 org.intellij.lang.annotations.MagicConstant 어노테이션으로 제어됩니다. 일부 표준 Java 라이브러리 메서드는 어노테이션이 미리 추가되어 있으나 코드에서 이 어노테이션을 사용할 수도 있습니다.
