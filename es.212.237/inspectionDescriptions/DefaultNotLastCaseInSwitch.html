default 케이스가 다른 케이스 앞에 오는 switch 구문 또는 표현식을 보고합니다.

이 구문은 불필요한 혼란을 일으킬 수 있습니다. default 케이스를 마지막 위치로 이동하는 빠른 수정이 있습니다. 주어진 브랜치의 끝에 break/yield이 있는 경우에만 수정이 제공됩니다.

예:


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

빠른 수정 적용 후:


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
