불필요한 단항 마이너스를 보고합니다. 이 표현식은 이해하기 어려울 수 있으며 오류를 포함할 수 있습니다.

예:

void unaryMinus(int i) {
    int x = - -i;
  }

다음과 같은 빠른 수정을 권장합니다.

  • i 변수 앞의 - 연산자를 제거합니다.

    void unaryMinus(int i) {
    int x = i;
  }

  • + 연산자를 접두사 감소 연산자와 치환합니다.

    void unaryMinus(int i) {
    int x = --i;
  }

다른 예:

void unaryMinus(int i) {
    i += - 8;
  }

빠른 수정 적용 후:

void unaryMinus(int i) {
    i -= 8;
  }
