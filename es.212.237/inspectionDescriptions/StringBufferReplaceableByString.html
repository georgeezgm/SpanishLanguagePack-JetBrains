단일한 String 연결에 치환될 수 있는 StringBuffer, StringBuilder, 또는 StringJoiner 의 사용을 보고합니다.

String 연결을 사용하여 코드를 짧고 간단하게 만듭니다.

이 조사는 제안된 교체가 JVM에서 중대한 성능 문제를 야기하지 않는 경우를 보고합니다. 대부분의 경우 String 연결의 성능은 향상될 수 있습니다.

예:

  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

빠른 수정 적용 후:

  String result = "i = " + i + ";";
  return result;
