래핑된 기본 값의 명시적 언래핑인 언박싱을 보고합니다.

Java 5 이상 버전에서 언박싱은 불필요하며 안전하게 제거될 수 있습니다.

예:

  • Integer i = Integer.valueOf(42).intValue(); → Integer i = Integer.valueOf(42);
  • int k = Integer.valueOf(42).intValue(); → int k = Integer.valueOf(42);

    (진정으로 불필요한 언박싱된 표현식만 보고 옵션이 검사되지 않은 경우에만 보고합니다)

진정으로 불필요한 언박싱된 표현식만 보고 옵션을 사용하여 언박싱된 값이 즉시 묵시적 또는 명시적으로 박싱되는 진정으로 불필요한 언박싱만을 보고합니다. 이 경우 언박싱-박싱 단계 전체를 제거할 수 있습니다. 이 검사는 간단한 명시적 언박싱을 보고하지 않습니다.

이 검사는 프로젝트 또는 모듈의 언어 수준이 5 이상인 경우에만 보고합니다.
