릴리즈되지 않은 인스턴스 변수에 저장된 소유한 객체를 보고합니다.
다음의 경우 객체의 소유권을 가져갑니다.
- 이름이 alloc 또는 new로 시작하는 메서드 또는 copy를 포함하는 메서드를 사용해 생성된 객체
- 객체를 retain 메시지로 보내는 경우
- retain 또는 copy 속성을 포함하고 있으며 인스턴스 변수와 통합된 프로퍼티의 값을 설정하는 경우
객체의 소유권을 포기하려면
- 객체를 release 또는 autorelease 메시지로 전송
- 인스턴스 변수와 통합된 프로퍼티에 nil 대입
자동 참조 집계(ARC)를 사용하면 이 검사가 비활성화됩니다. 아래의 체크박스가 꺼지면 클래스의 어떤 메서드에서도 인스턴스 변수를 릴리즈할 수 있습니다. 이를 사용하면 dealloc, tearDown, applicationWillTerminate: 또는 didTurnIntoFault 메서드는 물론 이들 메서드에서 호출할 수 있는 모든 메서드에서 그들을 릴리즈할 수 있습니다.
