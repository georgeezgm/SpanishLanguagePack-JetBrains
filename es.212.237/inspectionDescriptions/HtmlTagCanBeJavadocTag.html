Javadoc 주석 내 <code> 태그의 사용 위치를 보고합니다. JDK 1.5부터 이러한 구문은 {@code...} 구문으로 바꿀 수 있습니다. 이렇게 하면 HTML 문자 엔티티 대신 주석 내에 대괄호 < 및 > 사용이 허용됩니다.

예:

  
  /**
   * @return empty <code>ArrayList&lt;Integer&gt;</code>
   */
  List<Integer> getList(){... }

빠른 수정 적용 후:

  
  /**
   * @return empty {@code ArrayList<Integer>}
   */
  List<Integer> getList(){... }
