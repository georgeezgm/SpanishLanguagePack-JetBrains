대상이 다른 메서드 호출인 메서드 호출을 보고합니다.

예:

  class X { Y foo() { return new Y(); } }
  class Y { X bar() { return new X(); } }

  void foo() {
    Y y = new X().foo(); // 체인 메서드 호출
  }

검사 옵션을 사용하여 다음과 같은 경우에 대한 경고를 전환합니다.

  • 필드 이니셜라이저의 체인 메서드 호출. 예: private Y y = new X().foo();

  • 동일한 타입에서 작동되는 체인 메서드 호출. 예: new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();.
