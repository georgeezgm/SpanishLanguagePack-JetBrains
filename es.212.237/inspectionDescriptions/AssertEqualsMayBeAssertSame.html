동등한 assertSame() 메서드로 바꿀 수 있는 JUnit의 assertEquals() 메서드 호출을 보고합니다. Object.equals() 메서드를 재정의하지 않고 객체 ID가 비교됨을 명시적으로 만드는 final 클래스의 인스턴스가 인수인 경우 이것이 가능합니다.

assertEquals()를 다음으로 대체하는 것이 좋습니다:

assertSame().

예:


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

빠른 수정 적용 후:


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
