java.io.Serial 어노테이션이 추가되기에 적합하지 않은 Serializable 및 Externalizable 클래스 내 메서드 및 필드를 보고합니다.

예:


class Test implements Serializable {
  @Serial // 어노테이션이 추가된 필드는 final이 아니므로 직렬화 메커니즘의 일부가 아님
  private static long serialVersionUID = 7874493593505141603L;

  @Serial // 어노테이션이 추가된 메서드는 private이 아니므로 직렬화 메커니즘의 일부가 아님
  void writeObject(ObjectOutputStream out) throws IOException {
  }
}


class Test implements Externalizable {
  @Serial // 어노테이션이 추가된 메서드는 Externalizable 클래스 내부에 있으므로 직렬화 메커니즘의 일부가 아님
  private void writeObject(ObjectOutputStream out) throws IOException {
  }
}

유효한 케이스에 관한 정보는 java.io.Serial 문서를 참고하세요.
