간단한 식별자 주변에 사용하는 문자열 템플릿에서 중괄호 사용 위치를 보고합니다.

불필요한 괄호를 제거하려면 '중괄호 제거' 빠른 수정을 사용하세요.

예:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== 중복
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== 괜찮음
  }

빠른 수정 적용 후:


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== Updated
      val y = "${x.length}"
  }
