serialPersistentFields 필드가 private static final ObjectStreamField[]로 선언되어 있지 않은 Serializable 클래스를 보고합니다.

serialPersistentFields 필드가 그러한 제어자로 선언되어 있지 않은 경우, 직렬화는 필드가 전혀 선언되어 있지 않은 것과 같이 동작합니다.

예:

  
    class List implements Serializable {
      List next;

      ObjectStreamField[] serialPersistentFields
                   = {new ObjectStreamField("next", List.class)};

    }
