Java 레코드에 선언된 불필요한 생성자를 보고합니다.

예시 1:


  record Point(int x, int y) {
    public Point {} // 제거 가능
  }
  
  record Point(int x, int y) {
    public Point(int x, int y) { // 제거 가능
      this.x = x;
      this.y = y;
    }
  }

빠른 수정은 불필요한 생성자를 제거합니다.

예시 2:


   // 간결한 생성자로 변환 가능
  record Range(int from, int to) {
    public Range(int from, int to) {
      if (from > to) throw new IllegalArgumentException();
      this.from = from;
      this.to = to;
    }
  }

빠른 수정은 이 코드를 간결한 생성자로 변환합니다.

이 검사는 구성된 언어 수준이 14 또는 15(테스트 버전) 이상인 경우에만 보고합니다.
