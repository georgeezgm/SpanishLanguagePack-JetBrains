전체 패키지('* imports')를 포괄하는 모든 import 문을 보고합니다.

일부 코딩 표준은 그러한 import 문을 금지합니다.

import 문 최적화 명령어로 그러한 구문을 탐지하고 수정하도록 IntelliJ IDEA를 구성할 수 있습니다. 설정 | 에디터 | 코드 스타일 | Java | import 문으로 이동하여 단일 클래스 import 문 사용 옵션이 활성화되어 있는지 확인하고 '*'가 포함된 import 문을 사용하는 클래스 수 및 '*'가 포함된 정적 import 문을 사용하는 이름 수 필드에 값을 지정하세요. 그러므로 이 검사는 주로 변경할 의도가 없는 코드 베이스에 대한 오프라인 보고에 유용합니다.
