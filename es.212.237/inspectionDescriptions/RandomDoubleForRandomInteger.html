호출에 인자 배수를 곱하고 정수형으로 변환하여 정수를 생성하기 위해 사용한 java.util.Random.nextDouble() 호출을 보고합니다.

범위 내의 무작위 정수를 생성하기 위해 java.util.Random.nextInt(int) 를 사용하는 것은 더 간단하고 효율적입니다.

예:


  int getRandomInt() {
    return (int) ((new Random()).nextDouble() * SIZE);
  }

빠른 수정 적용 후:


  int getRandomInt() {
    return (new Random()).nextInt(SIZE);
  }
