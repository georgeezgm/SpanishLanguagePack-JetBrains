@Bean 어노테이션의 initMethod 및 destroyMethod에서 확인되지 않은 메서드 참조를 보고합니다.

예:

MyBean 클래스에 'public void 'doInit'(){...}' 메서드가 포함되지 않을 경우 'doInit' 표현식에서 "Cannot resolve method(메서드 확인 불가)"가 보고됩니다. 

  public class MyBean {...}

  @Component
  public class MyComponent {
     @Bean(initMethod="doInit" )
     public MyBean myBean() {...}
  }

이 예시에서 MyBean이 doInit() 메서드를 정의하지 않을 경우 검사가 확인되지 않은 메서드 참조를 보고합니다.
