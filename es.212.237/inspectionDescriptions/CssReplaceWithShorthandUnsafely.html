본디 CSS 프로퍼티 집합을 보고합니다. 불완전한 본디 CSS 프로퍼티 집합을 단축 양식으로 교환할 것을 제안합니다. 하지만 이 경우 100% 동등하지 않습니다.

예를 들어, 2개의 프로퍼티 outline-color 및 outline-style을 하나의 outline으로 바꿀 수 있습니다. 단축 양식은 모든 누락된 하위 값을 초기 상태로 재설정하므로 그러한 교체는 100% 동등하지 않습니다. 이 예에서 outline 단축 양식으로 바꾸면 outline-width 역시 초기 값인 medium으로 재설정됩니다.

이 검사는 전체 본디 프로퍼티 집합을 다루지 않습니다(단축 양식으로 바꾸면 100% 안전합니다.). 그러한 경우, '프로퍼티를 안전하게 단축 속성으로 바꿀수 있습니다' 검사를 고려하십시오.
