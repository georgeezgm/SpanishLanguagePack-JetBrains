메서드가 실제로 발생하는 예외보다 더 일반적인 예외를 포함하는 throws 절을 보고합니다.

예:


  public void createFile() throws Exception { // 경고: 'throws Exception'이 광범위하여 'IOException' 예외 마스크됨
    File file = new File("pathToFile");
    file.createNewFile();
  }

빠른 수정 적용 후:


  public void createFile() throws IOException {
    File file = new File("pathToFile");
    file.createNewFile();

검사 구성:

  • RuntimeException,Exception,Error 또는 Throwable 에 대한 경고 옵션을 사용하여 이 검사로 가장 일반적인 예외에 대해서만 경고합니다.
  • 라이브러리 메서드를 재정의하는 메서드에 선언된 예외 무시 옵션을 사용하여 라이브러리 메서드를 재정의하는 메서드에서 너무 광범위한 throws 절을 무시합니다.
  • 다른 예외를 감추지만 자체적으로 발생하는 예외 무시 옵션을 사용하여 다른 예외를 감추지만 여전히 메서드 본문에서 발생할 가능성이 있기 때문에 기술적으로 너무 광범위하지 않은 예외를 무시합니다.
