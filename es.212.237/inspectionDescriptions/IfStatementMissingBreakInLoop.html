의미 변경 없이 break로 끝나는 if 문이 포함된 루프를 보고합니다.

이를 통해 불필요한 루프 반복을 방지합니다.

예:

  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }

빠른 수정 적용 후:

  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
