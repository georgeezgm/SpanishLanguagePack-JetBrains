예를 들어, java.lang.Exception 또는 java.io.IOException 같은 너무 일반적인 예외 등 부적절한 예외를 던지는 throw 구문을 보고합니다.

예:


  void setup(Mode mode) {
      if (mode == null)
          throw new RuntimeException("Problem during setup"); // 경고: 금지된 예외 'RuntimeException'을 던졌습니다
     ...
  }

금지된 예외 목록을 사용하여 보고 대상의 예외를 지정합니다.
