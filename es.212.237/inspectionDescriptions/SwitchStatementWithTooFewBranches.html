case 라벨 수가 너무 적은 switch 구문을 보고하고, 이를 if 및 else if 구문으로 다시 작성하도록 제안합니다.

Example (minimum branches == 3):

  switch (expression) {
    case "foo" -> foo();
    case "bar" -> bar();
  }

빠른 수정 적용 후:

  if ("foo".equals(expression)) {
    foo();
  } else if ("bar".equals(expression)) {
    bar();
  }

검사 구성:

최소 분기 수 필드를 사용하여 case 라벨의 최소 개수를 지정하세요.
