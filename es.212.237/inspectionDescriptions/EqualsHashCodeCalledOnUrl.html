java.net.URL 객체에서 호출되는 hashCode() 및 equals()를 보고합니다.

java.net.URL 클래스는 java.net.URLStreamHandler의 인스턴스를 내부적으로 사용하여 hashCode() 및 equals()를 실행합니다. java.net.URLStreamHandler가 DNS 조회를 수행하기 때문에 성능 문제가 발생할 수 있습니다. 네트워크의 가용성과 DNS 서버의 속도에 따라 상당한 지연이 발생할 수 있습니다.

이 문제는 대신 java.net.URI를 사용하여 대부분 해결할 수 있습니다.

예:

  
    int f(URL url1, URL url2) {
      if (url1.equals(url2)) return url1.hashCode();
      else return url2.hashCode();
    }
