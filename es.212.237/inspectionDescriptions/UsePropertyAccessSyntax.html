Kotlin 통합 프로퍼티로 바꿀 수 있는 Java get 및 set 메서드를 보고합니다.

프로퍼티 액세스 구문 사용 빠른 수정을 사용해 코드를 자동으로 수정할 수 있습니다.

예:


  // Java:
  public class JavaClassWithGetter {
      private final String expr = "result";

      //...

      public String getExpr() {
          return expr;
      }
  }


  // Kotlin:
  fun test(j: JavaClassWithGetter) {
      //...
      j.getExpr() // <== 빠른 수정이 표현식을 'j.expr'로 단순화
  }
