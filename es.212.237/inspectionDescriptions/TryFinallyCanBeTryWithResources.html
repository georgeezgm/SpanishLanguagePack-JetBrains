오류 발생 가능성이 낮은 Java 7 Automatic Resource Management를 사용 가능한 try-finally 문을 보고합니다.

try-finally 문을 try-with-resources 문으로 변환하는 빠른 수정을 사용할 수 있습니다.

예:


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

원인을 생성자로 전달하는 빠른 수정이 제공됩니다.

  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }

이 검사는 프로젝트 또는 모듈의 언어 수준이 7 이상인 경우에만 보고합니다.
