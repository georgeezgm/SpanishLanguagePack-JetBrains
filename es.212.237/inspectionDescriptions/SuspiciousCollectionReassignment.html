읽기 전용 Collection에 있는 증강된 대입(+=) 표현식을 보고합니다.

읽기 전용 Collection의 증강 대입(+=) 표현식은 대상 컬렉션을 수정하지 않습니다. 이러한 표현식은 뒷단에서 오해의 여지가 있으며 성능 문제를 야기할 수 있는 새 표현식을 만듭니다.

유형을 가변으로 변경 빠른 수정을 사용해 코드를 자동으로 수정할 수 있습니다.

예:


  fun test() {
      var list = listOf(0)
      list += 42 // 새 목록이 생성됨, 'list' 변수는 여전히 '0'만 포함
  }

빠른 수정 적용 후:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
