도달할 수 없는 구문을 보고합니다. 이는 구문이 무한 루프, return, break 또는 continue 구문 뒤에 해당 구문이 있을 때 발생합니다.

예:


  void foo (int n) {
      if (n < 1) {
          return
          print('This statement is unreachable')
      }
      while (true){
          print ('Hello, world!')
      }
      print('This statement is unreachable too')
  }
