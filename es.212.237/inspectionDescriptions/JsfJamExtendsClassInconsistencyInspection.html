필수 인터페이스를 구현하지 않는, 특정 JSF 어노테이션이 추가된 클래스를 보고합니다.

  • @FacesComponent는 <javax/jakarta>.faces.component.FacesComponent를 확장해야 함
  • @FacesRenderer는 <javax/jakarta>.faces.render.Renderer을 확장해야 함
  • @FacesConverter는 <javax/jakarta>.faces.convert.Converter를 확장해햐 함
  • @FacesValidator는 <javax/jakarta>.faces.validator.Validator를 확장해햐 함

예:



@FacesConverter(value="...")  // 클래스는 'javax.faces.convert.Converter'를 확장해야 함
public class FooConverter {
}

@FacesRenderer(rendererType="..", componentFamily="..")  // 클래스는 'javax.faces.render.Renderer'를 확장해야 함
public class FooRenderer {
}

@FacesValidator(value="...")  // 클래스는 'javax.faces.validator.Validator'를 확장해야 함
public class FooValidator {
}
