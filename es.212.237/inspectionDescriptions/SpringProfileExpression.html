잘못된 @Profile 표현식을 보고합니다.

  • Spring 프로파일은 공백일 수 없습니다
  • Spring 프로파일 표현식에서 '&' 및 '|' 연산자는 괄호 없이 결합할 수 없습니다

예:


  
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  
  @Configuration
  @Profile() // reports "Profile expression must contain text"
  public class MyConfiguration {...}

  
  @Configuration
  @Profile("a & b | c") // reports "Malformed profile expression"
  public class MyConfiguration {...}
