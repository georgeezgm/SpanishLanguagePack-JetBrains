다음과 같은 EJB 사양 위반 사례를 보고합니다:
  • 삽입된 프로퍼티의 타입이 유효하지 않은 경우
  • EJB 참조가 `ejb-jar.xml`와 일치하지 않는 경우

예:


@LocalBean
@Stateless
public class ExampleBean {

  private AnotherBean anotherBean;

  @Inject
  public void getAnotherBean() { // 오류: 필드 또는 setter 메서드로의 삽입이 수행되어야 함
    return anotherBean;
  }

 ...
}
