String 을 빈 문자열과 비교하여 호출하는 equals() 를 보고합니다. 이 경우 확인 대상을 명확하게 보여주기 때문에 .isEmpty() 를 사용하는 것이 좋습니다.

예:


  void checkString(String s){
    if ("".equals(s)) throw new IllegalArgumentException();
  }

빠른 수정 적용 후:


  void checkString(String s){
    if (s != null && s.isEmpty()) throw new IllegalArgumentException();
  }

"".equals(str) 는 str 이 null 인 경우 false 를 반환합니다. 안전을 위해 검사의 빠른 수정은 equals() 인수가 nullable 인 경우 명시적인 null 확인을 삽입합니다. 검사 시 이러한 경우를 무시하도록 옵션을 사용합니다.
