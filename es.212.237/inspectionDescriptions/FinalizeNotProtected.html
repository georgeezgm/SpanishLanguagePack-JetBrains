public으로 선언된 Object.finalize() 메서드의 모든 구현을 보고합니다.

Object.finalize()의 컨트랙트에 따라 가비지 컬렉터만 이 메서드를 호출합니다. 이 메서드를 public으로 만드는 것은 메서드가 사용자에 의해 사용될 수 있음을 의미하므로 혼란을 일으킬 수 있습니다.

빠른 수정은 메서드를 protected로 만들어 다른 클래스에서 명시적으로 호출되는 것을 방지합니다.

예:


  class X {
    public void finalize() {
      /*... */
    }
  }

빠른 수정 적용 후:


  class X {
    protected void finalize() {
      /*... */
    }
  }
