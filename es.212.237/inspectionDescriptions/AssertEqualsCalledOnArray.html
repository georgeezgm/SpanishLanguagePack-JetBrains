배열 타입 인수를 사용하는 JUnit assertEquals() 메서드 호출을 보고합니다. 이러한 메서드는 배열의 콘텐츠 대신 배열의 ID를 비교합니다.

assertArrayEquals() 메서드 중 하나를 사용하여 배열의 콘텐츠를 비교하는 것이 좋습니다.

예:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertEquals(new int[] {0, 56, 248, 496}, actual);
  }

빠른 수정 적용 후:


  @Test
  public void testSort() {
    int[] actual = {248, 496, 0, 56};
    Arrays.sort(actual);
    Assert.assertArrayEquals(new int[] {0, 56, 248, 496}, actual);
  }
