의심스러운 System.arraycopy() 호출을 보고합니다.

이 검사는 다음과 같은 경우를 보고합니다:

  • 배열 타입이 아닌 소스 또는 대상
  • 타입이 다른 소스 및 대상
  • 복사된 청크 길이가 src.length - srcPos보다 항상 큰 경우
  • 복사된 청크 길이가 dest.length - destPos보다 항상 큰 경우
  • 소스와 대상이 같은 배열일 때 범위가 항상 교차하는 경우

예:

  void foo() {
    int[] src = new int[] { 1, 2, 3, 4 };
    System.arraycopy(src, 0, src, 1, 2); // 경고: 범위가 교차하는 같은 배열로 복사
  }
