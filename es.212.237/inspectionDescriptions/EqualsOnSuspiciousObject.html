StringBuilder 또는 StringBuffer과 같은 특정 클래스에서 equals()가 호출되는 경우를 보고합니다.

해당 클래스에서 equals() 메서드는 재정의되지 않으므로 두 객체의 콘텐츠가 동일한 경우에도 false가 반환될 수 있습니다. 의도한 참조 동등성이라면 ==를 사용하여 혼동을 피하는 것이 좋습니다.

예:


  public void test(StringBuilder sb1, StringBuilder sb2) {
    boolean result = sb1.equals(sb2); // Suspicious
  }
