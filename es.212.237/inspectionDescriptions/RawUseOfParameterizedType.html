타입 매개변수가 누락된 제네릭 클래스를 보고합니다. 그러한 제네릭 원천 타입의 사용은 Java에서 유효하지만 타입 매개변수의 목적을 무효화하며 버그를 가릴 수 있습니다. 이 검사는 javac의 rawtypes 경고와 같이 동작합니다.

예:

  • 
      List list = new ArrayList<String>(); // 경고: 매개변수화된 클래스 'List'의 원천 타입 사용
      list.add(1);                         // 문자열 목록이 생성되지만 Integer 또한 허용되어 런타임 예외를 초래함
   
  • 
      IntFunction<List<?>[]> fun = List[]::new; // 배열 생성 도중 타입 인수를 제공할 수 없으므로 경고 없음

검사 구성:

  • 객체 생성에 사용된 원시 타입을 무시하려면 새 객체 생성 무시 옵션을 사용하세요.
  • 타입 변환에 사용된 원시 타입을 무시하려면 타입 변환 무시 옵션을 사용하세요.
  • 타입 매개변수가 컴파일되지 않는 위치 무시 옵션을 사용하여 타입 매개변수가 컴파일되지 않는 경우를 무시합니다(예: 배열 생성 또는 라이브러리 메서드 재정의 시).
  • 재정의 메서드의 매개변수 타입 무시 옵션을 사용하여 재정의 메서드의 매개변수에서 사용된 타입 매개변수를 무시합니다.
  • 자동 빠른 수정을 사용할 수 없는 경우 무시 옵션을 사용하여 빠른 수정이 지원되지 않는 경우를 무시합니다.
이 검사는 프로젝트 또는 모듈의 언어 수준이 5 이상인 경우에만 보고합니다.
