본문에 하나의 synchronized 문이 포함된 메서드를 보고합니다. synchronized 문의 이 잠금 표현식은 인스턴스 메서드의 this 또는 static 메서드의 [ClassName].class와 동등해야 합니다.

해당 메서드의 가독성을 개선하려면 synchronized 래퍼를 제거하고 메서드를 synchronized로 표시해야 합니다.

예:


  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

빠른 수정 적용 후:


  public synchronized int generateInt(int x) {
    return 1;
  }
