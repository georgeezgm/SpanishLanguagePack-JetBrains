ReactorDebugAgent.init()이 호출된 경우 Hooks.onOperatorDebug()이 사용된 위치를 보고합니다.

ReactorDebugAgent가 클래스 경로에 있고 spring.reactor.debug-agent.enabled가 true인 경우(기본 값) 시작 시 ReactorDebugAgent.init()가 자동으로 호출됩니다. 또한 Hooks.onOperatorDebug() 및 ReactorDebugAgent.init()로 인해 Reactor에서 생성되는 디버그 스택 프레임이 두 배로 증가합니다. 그뿐 아니라 Hooks.onOperatorDebug()는 성능 오버헤드를 초래할 수 있습니다.

빠른 수정은 Hooks.onOperatorDebug()의 호출을 제거합니다.

예:


  public void hook() {
    Hooks.onOperatorDebug(); // Reports 'Call Hooks.onOperatorDebug() while ReactorDebugAgent is initialized'
  }

빠른 수정 적용 후:


  public void hook() {
  }

이 검사는 프로젝트 또는 모듈의 Spring Boot 버전이 2.2.0 이상일 경우에만 트리거됩니다.
