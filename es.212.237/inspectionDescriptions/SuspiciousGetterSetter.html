이름에 의해 기대되지 않는 필드에 액세스하는 getter 또는 setter 메서드를 보고합니다. 예를 들어, getY()가 x 필드를 반환하는 경우를 말합니다. 이는 보통 복사-붙여넣기 오류일 수 있습니다.

예:

class Point {
    private int x;
    private int y;

    public void setX(int x) { // 경고: setter 'setX()'가  필드 'y'를 대입함
      this.y = x;
    }

    public int getY() { // 경고: getter 'getY()'가 필드 'x'를 반환함
      return x;
    }
  }

아래 체크박스를 사용하여 클래스 내 필드의 이름이 getter 또는 setter의 이름과 일치하는 경우를 보고하세요.
