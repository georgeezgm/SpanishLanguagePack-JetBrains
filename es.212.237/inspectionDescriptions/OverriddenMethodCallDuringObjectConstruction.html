객체 생성 중 현재 클래스의 재정의 메서드 호출을 보고합니다. 호출이 다음 내부에 있는 경우 객체 생성 중에 호출됩니다.
  • 생성자
  • 비 static 인스턴스 이니셜라이저
  • 비 static 필드 이니셜라이저
  • 복제()
  • readObject()
  • readObjectNoData()
이 호출로 인해 버그가 발생할 수 있으며, 객체는 메서드 호출이 발생하기 전체 초기화하는 것을 보장하지 않습니다.

예:

  abstract class Parent {
    void someMethod() { }
  }

  class Child extends Parent {
    Child() {
      someMethod();
    }

    @Override
    void someMethod() { }
  }
이 검사는 다음 검사와 기능을 공유합니다.
  • 객체 생성 중에 호출된 추상 메서드 검사
  • 객체 생성 중에 호출된 재정의 가능한 메서드 검사
중복 경고를 방지하기 위해 동시에 한 개의 검사만을 활성화해야 합니다.
