프로퍼티 setter에서 명시적으로 지정된 매개변수를 보고합니다.

setter 매개변수는 항상 프로퍼티 유형과 일치하므로 명시적일 필요가 없습니다. '명시적 유형 사양 제거' 빠른 수정이 코드를 적절히 수정하도록 해줍니다.

예:


  fun process(x: Int) {}

  var x: Int = 0
      set(value: Int) = process(value) // <== 'Int' 사양은 안전하게 생략할 수 있음

빠른 수정 적용 후:


  fun process(x: Int) {}

  var x: Int = 0
      set(value) = process(value)
