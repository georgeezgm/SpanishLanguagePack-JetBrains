어노테이션 항목에서 불필요한 빈 괄호를 보고합니다.

코드를 정리하려면 '불필요한 괄호 제거' 빠른 수정을 사용하세요.

예:


  annotation class MyAnnotationA
  annotation class MyAnnotationB(val x: Int)
  annotation class MyAnnotationC(val x: Int = 10) // 기본값이 존재함

  @MyAnnotationA() // <== 괄호는 불필요
  fun testA() {
  }

  @MyAnnotationB() // <== 인수 누락, 괄호가 필요함
  fun testB() {
  }

  @MyAnnotationC() // <== 괄호가 불필요함
  fun testC() {
  }
