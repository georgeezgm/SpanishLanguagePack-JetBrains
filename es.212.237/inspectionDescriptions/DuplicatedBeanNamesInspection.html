애플리케이션 컨텍스트에 정의된 중복 Bean 이름을 보고합니다.

예:


  <beans profile="PROD">
    <bean class="..." id="duplicatedName"/>  // error
    <bean class="..." name="foo_name, duplicatedName"/> // 'duplicatedName' error
    <bean class="..." id="toBeAliased"/>

    <alias alias="duplicatedName" name="toBeAliased"/> // 'duplicatedName' error
  </beans>
