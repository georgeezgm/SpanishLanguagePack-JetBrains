Las declaraciones detalladas de las instantáneas HPROF están disponibles utilizando el Visor de memoria dentro de IntelliJ IDEA. Las instantáneas de memoria son útiles para localizar código que utiliza grandes cantidades de memoria y encontrar pérdidas de memoria.
