파일 템플릿에서 텍스트, 코드, 주석 및 사전 정의된 변수를 사용할 수 있습니다. 사전 정의된 변수 목록이 하단에서 제공됩니다. 템플릿에서 이러한 변수를 사용할 경우 추후 에디터에서 해당하는 값으로 확장됩니다.

또한 사용자 지정 변수를 지정하는 것도 가능합니다. 사용자 지정 변수는 다음 형식을 사용합니다: ${VARIABLE_NAME}, 여기서 VARIABLE_NAME은 변수 이름입니다 (예: ${MY_CUSTOM_FUNCTION_NAME}). IDE가 사용자 지정 변수를 사용하는 새 파일을 생성하기 전, 템플릿의 사용자 지정 변수 값을 정의할 수 있는 대화상자가 표시됩니다.

#parse 지시문을 사용하여 포함하기 탭에서 템플릿을 포함할 수 있습니다. 템플릿을 포함하려면, 템플릿의 전체 이름을 따옴표로 묶인 매개변수로 지정하세요 (예: #parse("File Header").
사전 정의된 변수 목록
${DATE}   현재 시스템 날짜
${DAY}   현재 일
${DAY_NAME_SHORT}   현재 요일의 첫 세 글자(Mon, Tue 등)
${DAY_NAME_FULL}   현재 요일 이름 전체(Monday, Tuesday 등)
${DIR_PATH}   새 파일 디렉터리 경로(프로젝트 루트에 대한 상대 경로)
${DS}   달러 표시 ('$'). 이 변수를 달러 기호($)를 이스케이프하기 위해 사용되므로 템플릿 변수의 접두사로 취급되지 않습니다.
${FILE_NAME}   새 파일 이름
${HOUR}   현재 시
${MINUTE}   현재 분
${SECOND}   현재 초
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자(Jan, Feb 등)
${MONTH_NAME_FULL}   현재 월 이름 전체(January, February 등)
${NAME}   새 엔티티 이름(파일, 유형, 인터페이스 등)
${ORGANIZATION_NAME}   프로젝트 설정에 지정된 사용자 조직의 이름
${PRODUCT_NAME}   IDE 이름
${PROJECT_NAME}   현재 프로젝트 이름
${TIME}   현재 시스템 시간
${USER}   현재 사용자의 시스템 로그인 이름
${YEAR}   현재 년도
