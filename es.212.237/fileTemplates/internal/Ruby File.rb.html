Ruby 파일에 대한 기본 제공 템플릿. 프로젝트 뷰 중 하나에서 팝업 메뉴의 새로 만들기 | Ruby 파일/클래스를 선택하여 새로운 Ruby 파일을 생성할 때마다 사용됩니다. Ruby 구문, 표현식 및 주석 뿐만 아니라 아래에 표시된 사전 정의 변수도 사용할 수 있습니다. 모든 변수는 매크로와 같이 해당하는 값으로 확장될 수 있습니다. #parse 지시문을 사용하여 원하는 템플릿의 전체 이름을 따옴표로 묶은 매개변수로 지정하여 포함 탭에서 템플릿을 포함할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${FILE_NAME}   Nombre de archivo actual
${USER}   현재 유저 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${DAY}   현재 일
${HOUR}   현재 시
${MINUTE}   현재 분
${PRODUCT_NAME}   현재 IDE 이름
${PROJECT_NAME}   현재 프로젝트 이름
