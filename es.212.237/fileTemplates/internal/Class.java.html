프로젝트 도구 창에서 새로 만들기 | Java 클래스 | 클래스를 호출하여 생성된 새 Java 클래스에 적용됩니다.
이 기본 제공 템플릿은 편집할 수 있습니다. Java 표현식 및 주석과 마찬가지로, 해당 값으로 확장되는 사전 정의된 변수(아래 나열됨)를 사용할 수 있습니다.

이 기본 제공 템플릿은 편집할 수 있습니다. 사용자 지정 변수는 다음 형식을 사용합니다: ${VARIABLE_NAME}, 여기서 VARIABLE_NAME은 변수 이름입니다 (예: ${MY_CUSTOM_FUNCTION_NAME}). IDE가 사용자 지정 변수를 사용하는 새 파일을 생성하기 전, 템플릿의 사용자 지정 변수 값을 정의할 수 있는 대화상자가 표시됩니다.

#parse 지시문을 사용하여 포함하기 탭에서 템플릿을 포함할 수 있습니다. 템플릿을 포함하려면, 템플릿의 전체 이름을 따옴표로 묶인 매개변수로 지정하세요(예: #parse("File Header.java").
사전 정의된 변수는 다음의 값을 사용합니다:
${PACKAGE_NAME}   새 클래스가 생성되는 패키지 이름
${NAME}   새 클래스 생성 대화상자에서 지정한 새 클래스 이름
${USER}   현재 사용자의 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자(Jan, Feb 등)
${MONTH_NAME_FULL}   현재 월 이름 전체(January, February 등)
${DAY}   현재 일
${HOUR}   현재 시
${MINUTE}   현재 분
${PROJECT_NAME}   현재 프로젝트 이름
