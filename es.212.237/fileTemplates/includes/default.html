#parse 지시문을 사용하여 파일 템플릿(템플릿 탭)에 포함하려는 코드 조각을 지정합니다.
여러 파일 템플릿에서 템플릿 하나를 공유할 수 있습니다.

정적 텍스트, 코드 및 주석과 마찬가지로, 매크로처럼 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${PACKAGE_NAME}   새 파일이 생성되는 패키지 이름
${USER}   현재 사용자의 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자(Jan, Feb 등)
${MONTH_NAME_FULL}   현재 월 이름 전체(January, February 등)
${DAY}   현재 일
${DAY_NAME_SHORT}   현재 요일의 첫 세 글자(Mon, Tue 등)
${DAY_NAME_FULL}   현재 요일 이름 전체(Monday, Tuesday 등)
${HOUR}   현재 시
${MINUTE}   현재 분
${PROJECT_NAME}   현재 프로젝트 이름
