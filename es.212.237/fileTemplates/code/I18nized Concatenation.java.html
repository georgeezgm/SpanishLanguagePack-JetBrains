문자열 연결(예:String s = "Hello " + name;)을 국제화된 Java 코드 표현식으로 대체합니다(예: String s = MessageFormat.format(​myResourceBundle.getString(​"hello.0"), name);).
이 기본 제공 템플릿은 편집할 수 있습니다. Java 표현식 및 주석과 마찬가지로, 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${RESOURCE_BUNDLE}   java.util.ResourceBundle 타입의 표현식으로, 해당 타입의 인스턴스를 이 컨텍스트에서 사용할 수 있습니다.
${PARAMETERS}   문자열 연결에 사용되는 변수로, 일반적으로 국제화된 표현식에 인수로 전달됩니다.
${PROPERTY_KEY}   해당 프로퍼티 파일에 정의된 프로퍼티 키 이름입니다. 일반적으로 ResourceBundle.getString() 메서드 매개변수에 사용되는 값입니다.
${PROPERTY_VALUE}   해당 프로퍼티 파일에 정의된 프로퍼티 값입니다. 기존의 Java 문자열 리터럴 값입니다.
