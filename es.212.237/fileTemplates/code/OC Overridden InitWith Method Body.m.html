예를 들어 Code | Override Method 또는 Code | Generate | -initWith 액션
을 호출할 때 생성되는 재정의된 -init 메서드의 본문을 생성합니다. 템플릿은 편집할 수 있습니다. Objective-C 표현식과 주석과 함께 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수도 있습니다.
사전 정의된 변수는 다음 값을 사용합니다.
${CALL_SUPER}   void 메서드에 대한 super 메서드 호출 — 매개변수가 있거나 없는 [supermethodName] 그 외 메서드에 대한 호출 - 매개변수가 있거나 없는 return [supermethodName]
${RETURN_TYPE}   생성된 메서드의 반환 유형
${DEFAULT_RETURN_VALUE}   메서드에 의해 기본으로 반환하는 값
${CONTAINING_CLASS}   메서드가 재정의된 클래스의 이름
${CUSTOM_CODE}   생성된 항목의 본문
C File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다 (${USER}, ${DATE}, 등).
