PHP 클래스 생성자에 대한 기본 제공 템플릿.
사전 정의된 변수는 다음의 값을 사용합니다:
${PARAM_LIST}   선택된 프로퍼티에 대해 하나씩 쉼표로 구분된 생성자 매개변수 목록.
${PARAM_DOC}   매개변수의 PHPDoc 주석.
'* @param type name' 을 포함하는 여러 줄로 생성됩니다. 만약 매개변수가 없으면, 빈 콘텐츠로 간주됩니다.
${THROWS_DOC}   예외의 PHPDoc 주석.
'* @throws type' 을 포함하는 여러 줄로 생성됩니다. 만약 예외가 없으면, 빈 콘텐츠로 간주됩니다.
${CLASS_NAME}   클래스 이름.
${BODY}   '$this->property=$property;' 대입을 포함하는 생성자 본문.
