예를 들어 Code | Generate | +sharedInstance 액션
을 호출할 때 클래스 헤더 섹션에서 +sharedInstance 메서드 선언을 생성합니다. 템플릿은 편집할 수 있습니다. Objective-C 표현식과 주석과 함께 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음 값을 사용합니다.
${$CLASS_TYPE}   메서드가 구현된 클래스의 이름
C File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다 (${USER}, ${DATE}, 등).
