생성된 경우 catch 매개변수를 채웁니다. 예: 코드 | 다음으로 둘러싸기 함수가 호출된 경우.
이 기본 템플릿은 편집할 수 있습니다. 예를 들어, 생성된 매개변수를 final로 지정하여 포착된 예외의 정규화된 이름에도 불구하고 항상 사전 정의된 이름을 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${EXCEPTION}   정규화된 이름이 Exception으로 끝나는 경우, 포착된 예외 유형 e에서 파생된 catch 매개변수로 지정된 Exception 변수의 이름.
${EXCEPTION_TYPE}   catch 매개변수 타입.
