재정의된 메서드가 코드 | 생성 | 메소드 재정의 액션 등으로 생성될 때 해당 메서드의 본문을 입력합니다.
템플릿은 편집 가능합니다. Swift 표현식과 주석과 함께 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${CALL_SUPER}   상위 메서드 호출, void 메서드: 매개변수가 있거나 없는 super.methodName(), 그 외 메서드: 매개변수가 있거나 없는 return super.methodName()
${RETURN_TYPE}   생성된 메서드의 반환 유형
${DEFAULT_RETURN_VALUE}   메서드에서 기본으로 반환되는 값
${SIGNATURE}   메서드의 시그니처
Swift File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다. (예: ${USER}, ${DATE}, 등)
