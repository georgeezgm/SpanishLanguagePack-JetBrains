PHP setter 메서드에 대한 기본 제공 템플릿. 예를 들어, "getter 및 setter 추가" 빠른 수정이 실행될 때 사용됩니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${STATIC}   프로퍼티가 정적이면, 'static' 값을 사용하고, 그렇지 않으면 빈 문자열을 사용합니다.
${NAME}   첫 번째 문자를 대문자로 하며 "set" 접두사없이 카멜 표기로 표기된 메서드 이름. 예를 들어, 프로퍼티 이름 '_my_property'는 'MyProperty'로 변환됩니다.
${CLASS_NAME}   포함하는 클래스 이름.
${SCALAR_TYPE_HINT}   언어 수준이 PHP 7.0 이상일 때 스칼라 타입 힌트를 포함합니다
${FIELD_NAME}   기존 프로퍼티 이름
${PARAM_NAME}   setter의 매개변수 이름. ${FIELD_NAME} 과 거의 같으나 첫 번째 밑줄이 있을 경우 제거됩니다. 예를 들어, 프로퍼티 이름 '_myProperty'는 'myProperty'로 변환됩니다.
${TYPE_HINT}   프로퍼티에 대한 타입 힌트. 지정되지 않은 경우, 빈 문자열.
${VOID_RETURN_TYPE}   프로젝트의 PHP 언어 수준에서 void 반환 타입이 사용 가능한지 나타냅니다.
${DS}   달러 표시, '$' 기호로 표시됩니다.
