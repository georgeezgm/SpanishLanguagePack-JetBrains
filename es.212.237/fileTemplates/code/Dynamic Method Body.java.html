FreeMarker 동적 메서드가 프로그램에 의해 생성될 때 본문을 작성하기 위해 사용하는 기본 제공 템플릿입니다.
템플릿은 수정할 수 있습니다. Java 표현식 및 주석과 마찬가지로, 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${RETURN_TYPE}   생성된 메서드의 반환 유형
${DEFAULT_RETURN_VALUE}   메서드에 의해 기본으로 반환되는 값
${METHOD_NAME}   생성된 메서드 이름
${CLASS_NAME}   메서드가 생성된 클래스의 정규화된 이름
