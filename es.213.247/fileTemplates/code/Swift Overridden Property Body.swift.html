재정의된 프로퍼티가 Code(코드) | Generate(생성) | Override Methods(메서드 재정의) 액션 등으로 생성될 때 해당 프로퍼티의 본문을 입력합니다.
이 템플릿은 편집할 수 있습니다. Swift 표현식과 주석과 함께 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${CALL_SUPER}   상위 프로퍼티 호출: super.propertyName
${RETURN_TYPE}   생성된 프로퍼티의 반환 타입
${DEFAULT_RETURN_VALUE}   프로퍼티에서 디폴트로 반환되는 값
${SIGNATURE}   프로퍼티의 시그니처
${HAS_SET}   프로퍼티에 setter가 있는 경우 'true'
${HAS_SUPER_SET}   재정의된 프로퍼티에 setter가 있는 경우 'true'
${GETTER_EFFECTS}   프로퍼티의 getter를 위한 getter 효과(async, throws)
${GET_ATTRIBUTES}   프로퍼티의 getter를 위한 속성
${SET_ATTRIBUTES}   프로퍼티의 setter를 위한 속성
Swift File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다. (예: ${USER}, ${DATE} 등)
