Puede anular la configuración de estilo del código IDE con la configuración de EditorConfig (https://editorconfig.org/).

Para utilizar la configuración de EditorConfig, instale y habilite elConfiguración del editor enchufar. Para configurar el estilo en EditorConfig, abra la configuración (&shortcut:MostrarConfiguración;), navegar aRedactor | Estilo de códigoy seleccione elHabilitar la compatibilidad con EditorConfig caja.

Para exportar la configuración de estilo de código IDE actual al.editconfig archivo, haga clicExportar.
