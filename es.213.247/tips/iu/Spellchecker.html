Las revisiones ortográficas y gramaticales están habilitadas de forma predeterminada. Estas comprobaciones las proporciona el complemento Grazie, que se incluye con el IDE. Para configurar los ajustes de revisión, abra la configuración presionando&shortcut:MostrarConfiguración; y navegar aRedactor | Lenguajes Naturales | Gramática.

Todos los textos, incluidos comentarios, cadenas textuales y literales, se inspeccionan con diccionarios predefinidos o personalizados. Los errores tipográficos se resaltan para que pueda elegir la palabra correcta, aceptar la ortografía actual o desactivar la inspección.

Para corregir un error ortográfico o gramatical, presione&shortcut:ShowIntentionActions; para ver la lista de acciones disponibles.
