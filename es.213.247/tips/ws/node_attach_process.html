Node.js 애플리케이션이 --inspect 또는 --inspect-brk 플래그를 사용하여 시작된 경우, 기본 제공 터미널, 실행 도구 창 또는 디버그 도구 창에서 디버그를 시작할 수 있습니다.

  1. node --inspect-brk <path to the starting page of your application relative to the project root> 을(를) 사용하여 터미널 에서 애플리케이션을 실행합니다.

  2. 터미널 도구 창에 정보 메시지 디버거 리스닝 <host>:<port> 이(가) 표시되고 기본 포트는 9229 입니다. 디버그를 시작하려면 Ctrl+Shift 을(를) 누른 상태에서 링크를 클릭합니다.
