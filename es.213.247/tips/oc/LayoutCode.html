Preferences 대화 상자 &shortcut:ShowSettings; 의 Editor(에디터) | Code Style(코드 스타일) 페이지에 정의된 스타일에 따라 코드 형식을 재지정하려면 메인 메뉴에서 Code(코드) | Reformat Code(코드 형식 재지정) 를 선택하거나 &shortcut:ReformatCode; 을(를) 누릅니다.

또한 Objective-C에서 코드 | import 문 최적화 를 사용하여 프로젝트 코드 스타일 설정에 따라 import 문을 정리하고 사용되지 않는 import 문을 제거할 수 있습니다.
