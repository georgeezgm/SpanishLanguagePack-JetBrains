Kotlin 1.3 또는 이후 버전과 호환성을 위해 업데이트해야 하는 Gradle의 kotlinx.coroutines라이브러리 종속 요소를 보고합니다.

예:


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.0.1'
  }

빠른 수정에서는 kotlinx.coroutines 라이브러리 버전을 Kotlin 1.3과 호환 가능한 버전으로 변경합니다.


  dependencies {
      implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:0.27.0-eap13'
  }
