매우 상세하거나 덜 효율적인 구문 대신 static Integer.compare() 메서드 또는 유사한 메서드가 사용될 수 있는 경우를 보고합니다.

x 및 y가 이미 정수로 박싱된 경우 x.compareTo(y)가 제안됩니다.

예:


  public int compare(int x, int y) {
    return x > y ? 1 : x < y ? -1 : 0;
  }

빠른 수정 적용 후:


  public int compare(int x, int y) {
    return Integer.compare(x, y);
  }

Double.compare() 및 Float.compare() 메서드는 Java 1.4에서 도입되었습니다. 다른 기본 타입의 메서드는 Java 1.7부터 지원됩니다.
