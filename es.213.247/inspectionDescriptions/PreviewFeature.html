@PreviewFeature 어노테이션이 추가된 요소 집합인 Preview Feature API의 사용을 보고합니다.

미리보기 기능은 지정과 구현이 완료된 Java 언어, Java 가상머신, 또는 Java SE API의 새로운 기능이지만 아직 정식으로 도입되지 않았습니다. 미리보기 기능의 개념은 JEP 12에서 정의됩니다.

코드가 미리보기 기능에 의존하는 경우 미리보기 기능이 긍정적인 피드백을 받지 못해 삭제 또는 변경되는 경우 향후 JDK 버전에서 컴파일이 중지될 수 있습니다.

현재 프로젝트의 언어 수준이 미리보기인 경우에만 검사를 활성화할 수 있습니다.
