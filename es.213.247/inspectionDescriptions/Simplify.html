상수 조건, 동일한 if 브랜치, 불필요한 bool 표현식 등 단순화할 수 있는 구문을 보고합니다. 분석 옵션을 제어하려면 아래 체크박스를 사용하세요.
  • "if (true)", "while(false)" 단순화: if/while 구문을 bool 리터럴로 바꾸기를 권장합니다.
  • "condition == true" 단순화: 상등 구문을 bool 리터럴로 바꾸기를 권장합니다.
  • "condition1 ? 단순화 true : condition2;": 표현식 중 하나가 bool 리터럴인 삼항 연산자를 단순화하기를 권장합니다.
  • "if (condition1) return true; return condition2;" 단순화: 조건이 true이고 다른 조건의 반환 값이 다르면 bool 리터럴이 반환되는 경우 if를 단순화하도록 권장합니다.
