요소의 디폴트 모달리티와 일치하는 모달리티 제어자를 보고합니다(대부분 요소의 경우 final, override가 있는 멤버의 경우 open).

예:


  final class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }

빠른 수정 적용 후:


  class Foo

  open class Bar : Comparable<Bar> {
      open override fun compareTo(other: Bar): Int = 0
  }
