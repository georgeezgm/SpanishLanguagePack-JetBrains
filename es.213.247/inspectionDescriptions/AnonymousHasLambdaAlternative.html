생성자 또는 람다 식 인수를 사용하는 factory 메서드 호출로 변환 가능한 익명 클래스를 보고합니다

이 검사를 통해 다음 클래스가 보고됩니다.

  • initialValue() 메서드가 포함된 ThreadLocal을 확장하는 익명 클래스(ThreadLocal.withInitial로 대체 가능)
  • run() 메서드가 포함된 Thread를 확장하는 익명 클래스(new Thread(Runnable)로 대체 가능)

예:


  new Thread() {
    @Override
    public void run() {
      System.out.println("Hello from thread!");
    }
  }.start();

빠른 수정 적용 후:


  new Thread(() -> {
    System.out.println("Hello from thread!");
  }).start();
