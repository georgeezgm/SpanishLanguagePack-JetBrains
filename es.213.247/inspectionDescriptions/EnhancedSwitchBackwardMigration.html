개선된 switch 문 또는 식을 보고합니다. 해당 문이나 식을 일반 switch 문으로 바꾸도록 제안합니다.

예:


  boolean even = switch (condition) {
    case 1, 3, 5, 7, 9 -> false;
    default -> true;
  };

빠른 수정 적용 후:


  boolean even;
  switch (condition) {
    case 1:
    case 3:
    case 5:
    case 7:
    case 9:
      even = false;
      break;
    default:
      even = true;
      break;
}
