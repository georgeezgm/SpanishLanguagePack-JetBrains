Reports unassigned properties of Entity in creating in ChangeScope with new or getOrCreate function.

You have to assign all non-nullable properties of Entity, or mark it with @Many annotation. Otherwise, it's cause runtime exception.

ì˜ˆ:


  interface A: Entity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

A quick-fix generates assignment for missing required properties


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
