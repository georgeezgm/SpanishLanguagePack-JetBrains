public 필드를 보고합니다. 상수(static 및 final로 표기된 필드)는 보고되지 않습니다.

예:


  class Main {
    public String name;
  }

빠른 수정 적용 후:


  class Main {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
  }

검사 구성:

  • 다음으로 어노테이션이 추가된 경우 무시 목록을 사용하여 무시할 어노테이션을 지정합니다. 이 검사는 해당 어노테이션이 추가된 필드를 무시합니다.
  • 열거형의 'public final' 필드 무시 옵션을 사용하여 enum 타입의 public final 필드를 무시합니다.
