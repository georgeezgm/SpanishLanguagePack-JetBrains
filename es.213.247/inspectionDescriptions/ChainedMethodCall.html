대상이 다른 메서드 호출인 메서드 호출을 보고합니다. 빠른 수정에서는 지역 변수를 삽입할 것을 제안합니다.

예:


  class X {
    int foo(File f) {
      return f.getName().length();
    }
  }

빠른 수정 적용 후:


  class X {
    int foo(File f) {
      final String name = f.getName();
      return name.length();
    }
  }

검사 옵션을 사용하여 다음과 같은 경우에 대한 경고를 전환합니다.

  • 필드 이니셜라이저 내 체인 메서드 호출(예: private final int i = new Random().nextInt();)

  • 동일한 타입에서 작동되는 체인 메서드 호출(예: new StringBuilder().append("x: ").append(new X()).append("y: ").append(new Y()).toString();).
