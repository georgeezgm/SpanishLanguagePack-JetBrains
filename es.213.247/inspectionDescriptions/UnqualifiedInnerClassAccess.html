감싸는 클래스의 이름으로 한정되지 않는 내부 클래스의 참조를 보고합니다.

예:


  import foo.Foo.Bar;

  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Bar bar) {}
  }

빠른 수정 적용 후:


  class Foo {
    class Bar {}
  }

  class Baz {
    void f(Foo.Bar bar) {}
  }

검사 설정을 사용하여 동일한 클래스 내에서 내부의 클래스를 무시합니다. 이 경우 import 문이 필요하지 않습니다.
