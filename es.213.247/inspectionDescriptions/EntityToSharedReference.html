Reports invalid reference from Entity to SharedEntity.

You cannot refer Entity from SharedEntity. You can refer SharedEntity from Entity only if property is nullable or has @CascadeDeleteBy annotation. Otherwise, it's cause runtime exception.

  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

A quick-fix generates annotation or make your type nullable


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
