SQL 예약 키워드를 쿼리 안에서 식별자 이름으로 사용하는 경우를 보고합니다.

예(Microsoft SQL 서버):

CREATE TABLE select (identity INT IDENTITY NOT NULL, order INT NOT NULL);

테이블 및 열 이름으로 select, identity, order를 사용합니다. 하지만 이러한 이름 역시 Microsoft SQL Server의 예약 키워드입니다. 따라서 이러한 식별자를 객체 이름으로 사용하려면 따옴표로 묶어야 합니다. 식별자를 따옴표로 묶기 위해 식별자에 따옴표 추가 빠른 수정을 사용할 수 있습니다.

빠른 수정 적용 후:

CREATE TABLE [select] ([identity] INT IDENTITY NOT NULL, [order] INT NOT NULL);
