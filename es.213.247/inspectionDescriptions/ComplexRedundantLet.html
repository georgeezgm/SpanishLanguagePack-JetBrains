불필요한 인수를 기반으로 하는 let 호출을 보고합니다.

호출 표현식에서 람다 매개변수가 한정자로만 사용되는 경우 let은 불필요합니다.

정규화하는 표현식에 이름을 붙여야 한다면 지역 변수를 선언하세요.

예:


  fun splitNumbers() {
      "1,2,3".let { it.split(',') }
  }

빠른 수정에서는 여분의 let() 호출을 제거합니다.


  fun example() {
      "1,2,3".split(',')
  }

또는


  fun splitNumbers() {
      val numbers = "1,2,3"
      numbers.split(',')
  }
