@deprecated Javadoc 태그가 있지만 @java.lang.Deprecated 어노테이션은 없는 모듈 선언, 클래스, 필드 또는 메서드를 보고합니다.

예:


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

빠른 수정 적용 후:


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

이 검사는 프로젝트 또는 모듈의 언어 수준이 5 이상인 경우에만 작동합니다.

@Deprecated 어노테이션이 있지만 @Deprecated Javadoc 태그에 설명이 없는 심볼에 대해 경고하려면 아래 체크박스를 사용하세요.
