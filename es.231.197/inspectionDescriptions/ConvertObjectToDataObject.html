데이터 객체로 변환할 수 있는 객체를 보고합니다.

data object는 toString, equals 및 hashCode를 자동으로 생성합니다.

이 검사는 다음 3가지 경우에 object를 data object로 변환하도록 제안합니다.

  • 사용자 지정 toString이 클래스의 이름을 반환하는 경우
  • object가 sealed class/interface를 상속하는 경우

예:


    object Foo {
        override fun toString(): String = "Foo"
    }

빠른 수정 적용 후:


    data object Foo

이 검사는 프로젝트 또는 모듈의 Kotlin 언어 수준이 1.9 이상인 경우에만 보고합니다
