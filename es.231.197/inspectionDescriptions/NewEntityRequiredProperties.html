ChangeScope에서 new 혹은 getOrCreate 함수로 생성할 때 Entity의 대입되지 않은 프로퍼티를 보고합니다.

Entity의 null이 될 수 없는 모든 프로퍼티를 대입하거나 @Many 어노테이션으로 표시해야 합니다. 그렇지 않은 경우 런타임 예외가 발생합니다.

예:


  interface A: Entity {
    var x: Int
    var y: Int?
    @Many var z: Int
    var p: Int
  }

  fun ChangeScope.foo() {
    new(A::class) {
      this.p = 0
    }
  }

빠른 수정에서는 누락된 필수 프로퍼티에 대한 대입을 생성합니다


  fun ChangeScope.foo() {
    new(A::class) {
      this.x = x
      this.p = 0
    }
  }
