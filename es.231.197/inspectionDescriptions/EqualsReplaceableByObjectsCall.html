java.util.Objects#equals 호출로 대체 가능한 표현식을 보고합니다.

예:


  void f(Object a, Object b) {
    boolean result = a != null && a.equals(b);
  }

빠른 수정 적용 후:


  void f(Object a, Object b) {
    boolean result = Objects.equals(a, b);
  }

a != null && a.equals(b)와 같은 표현식을 Objects.equals(a, b)로 대체할 경우 의미가 약간 변경됩니다. 이 동작을 활성화 또는 비활성화하려면 'a != null && a.equals(b)'와 같은 표현식을 강조 표시 옵션을 사용합니다.

이 검사는 프로젝트 또는 모듈의 언어 수준이 7 이상인 경우에만 보고합니다.
