인수 중 하나가 빈 문자열인 문자열 연결을 보고합니다. 이러한 연결은 필요하지 않습니다. 때때로 비 String 객체 또는 기본 타입을 String으로 변환하는 관용구로 사용되지만, 이 경우, String.valueOf 같은 메서드를 사용하는 것이 더 명확합니다.

빠른 수정에서는 연결을 단순화하도록 제안합니다.

예:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

빠른 수정 적용 후:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }
