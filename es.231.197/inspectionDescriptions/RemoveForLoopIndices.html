withIndex() 함수를 사용하며 색인 변수를 사용하지 않는 컬렉션을 반복 처리하는 for 루프를 보고합니다.

코드를 정리하려면 ' 'for' 루프에서 색인 제거' 빠른 수정을 사용하세요.

예:


  fun foo(bar: List<String>) {
     for ((index : Int, value: String) in bar.withIndex()) { // <== 'index'가 사용되지 않습니다
         println(value)
     }
  }

빠른 수정 적용 후:


  fun foo(bar: List<String>) {
      for (value: String in bar) { // <== '.withIndex()' 및 'index'가 제거되었습니다
          println(value)
      }
  }
