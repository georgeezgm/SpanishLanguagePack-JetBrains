Object 상등 관계를 테스트하기 위해 equals() 대신 == 또는 !=를 사용하는 코드를 보고합니다.

== 또는 !=를 사용하여 객체를 비교하는 것은 상등 대신 ID로 객체를 비교하기 때문에 보통 버그입니다. null과의 비교는 지원되지 않습니다. 배열, String 및 Number 비교는 별도의 검사에서 보고됩니다.

예:

  if (list1 == list2) {
    return;
  }

빠른 수정 적용 후:

  if (Objects.equals(list1, list2)) {
    return;
  }

검사 설정을 사용하여 이 검사에 대한 예외를 구성하세요.
