루프를 사용하여 배열의 내용을 컬렉션으로 복사하는 코드를 보고합니다.

이 작업을 더 간편하고 빠르게(컬렉션 구현에 따라 다름) 수행하려면 Collection.addAll(Arrays.asList()) 또는 Collections.addAll()을 사용하면 됩니다.

내부에 추가 구문이 없는 루프만 보고됩니다.

예:


  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

빠른 수정 적용 후:


  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
