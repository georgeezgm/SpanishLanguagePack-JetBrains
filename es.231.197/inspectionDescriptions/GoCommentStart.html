내보내기된 요소의 이름으로 시작하지 않는 주석을 보고합니다.

github.com/golang의 Comment Sentences에 따르면 이는 내보내기된 요소의 이름으로 주석을 시작하도록 협의된 규칙입니다.

예:

// 명령 실행 요청을 나타냅니다.
type Request struct {}

주석이 구조체 이름이 아닌 구조체에 대한 설명으로 시작합니다. 협의된 규칙을 따르려면, 주석에 접두사 추가 빠른 수정을 적용하세요. 빠른 수정이 적용된 후 주석은 다음과 같습니다:

// Request는 명령 실행 요청을 나타냅니다.
type Request struct {} // 개선됨
