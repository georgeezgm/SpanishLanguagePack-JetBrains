불필요하거나 지나치게 복잡한 부울 표현식을 보고합니다.

이러한 표현식은 true와 &&연산, false와 ||연산, 부울 리터럴과 상등 비교 또는 부울 리터럴 부정을 포함합니다. 이러한 표현식은 단순화할 수 있습니다.

예: 


  boolean a = !(x && false);
  boolean b = false || x;
  boolean c = x != true;

빠른 수정 적용 후:


  boolean a = true;
  boolean b = x;
  boolean c = !x;

검사 구성:

표현식이 무의미한지 판단할 때 이름이 지정된 상수를 무시하려면 무의미한 표현식 결정 시 이름 지정된 상수 무시 옵션을 사용하세요.
