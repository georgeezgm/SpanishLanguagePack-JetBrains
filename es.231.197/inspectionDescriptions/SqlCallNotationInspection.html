위치 인수가 명명된 인수 뒤에 오는 호출을 보고합니다. PostgreSQL, Oracle, Db2에서 작동합니다.

예(PostgreSQL 내):

CREATE FUNCTION foo(a int, b int, c int) RETURNS int
    LANGUAGE plpgsql AS
$$
BEGIN
    RETURN a + b + c;
END
$$;
SELECT foo(a => 1, b => 2, c => 3);
  -- '3'이 명명된 인수 뒤에 옴
SELECT foo(1, b => 2, 3);
  -- '1' 및 '3'이 명명된 인수 뒤에 옴 
SELECT foo(b => 2, 1, 3);
