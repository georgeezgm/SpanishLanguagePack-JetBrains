Gestión de solicitudes de extracción

Administre las solicitudes de extracción entrantes de GitHub directamente desde CLion. Desde el menú principalGit | GitHub | Mostrar solicitud de extracción Seleccionar . CLion le permite asignar y fusionar solicitudes de extracción, mostrar una línea de tiempo y comentarios en línea, enviar comentarios y reseñas y aceptar cambios sin tener que salir del IDE.
