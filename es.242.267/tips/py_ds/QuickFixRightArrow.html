Opciones de acción contextual

&shortcut:ShowIntentionActions; Cuando presiona para invocar una solución rápida o una acción intencional, haga clic en la flecha derecha para mostrar una lista de opciones adicionales. Dependiendo del contexto, puede seleccionar acciones como deshabilitar el análisis, solucionar todos los problemas o cambiar el perfil de análisis.
