코드 서식 다시 지정

현재 코드 스타일 설정에 따라 코드를 재정렬하려면, 메인 메뉴에서 Code(코드) | Reformat Code(코드 서식 다시 지정)을 선택하거나 &shortcut:ReformatCode;을(를) 누르세요.

기본적으로 RustRover는 내장된 서식 지정 도구를 사용합니다. Rustfmt를 선호한다면, Settings(설정) | Rust | Rustfmt로 이동하여 Use Rustfmt instead of built-in formatter(기본 제공되는 서식 지정 도구 대신 Rustfmt 사용) 체크박스를 선택하세요.
