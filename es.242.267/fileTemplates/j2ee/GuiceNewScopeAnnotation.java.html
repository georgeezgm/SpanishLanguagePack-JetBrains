새로운 Guice Scope Annotation을 생성할 때마다 사용되는 기본 제공 템플릿입니다.
#parse 지시문을 사용하면, 원하는 템플릿의 전체 이름을 따옴표로 묶은 매개변수로 지정하여 Includes 탭에서 템플릿을 포함할 수 있습니다. 예:
#parse("File Header.java")
사전 정의된 변수는 다음의 값을 사용합니다:
${PACKAGE_NAME}   새 클래스가 생성되는 패키지 이름
${NAME}   Create New Class(새 클래스 생성) 대화상자에서 지정한 새 클래스 이름
${USER}   현재 유저 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자 예: Jan, Feb 등등.
${MONTH_NAME_FULL}   현재 월의 전체 이름 예: January, February 등등.
${DAY}   현재 월의 일
${HOUR}   현재 시
${MINUTE}   현재 분
${PROJECT_NAME}   현재 프로젝트의 이름
