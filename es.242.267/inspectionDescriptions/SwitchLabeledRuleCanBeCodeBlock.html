표현식 본문을 포함하는 switch 식 또는 향상된 switch 문의 규칙을 보고합니다. 이는 코드 블록으로 변환할 수 있습니다.

예:


  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

빠른 수정 적용 후:


  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };
