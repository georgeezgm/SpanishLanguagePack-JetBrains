'reactive' 메서드의 subscribe() 호출을 보고합니다.

Publisher 타입을 반환하는 메서드(Flux 및 Mono 포함)의 경우 subscribe() 메서드를 직접 호출해서는 안 됩니다. 그러면 Reactive 호출 체인이 끊어질 수 있기 때문입니다.

subscribe() 대신 flatMap(), zip(), then() 등의 합성 연산자를 사용해보세요.

예:


  Flux<String> stringFlux(){
    Flux<String> flux = Flux.just("abc");
    flux.subscribe(); // <- 비차단 컨텍스트에서 'subscribe' 호출 차단
    return flux;
  }
