Entity에서 SharedEntity으로의 유효하지 않은 참조를 보고합니다.

Entity를 SharedEntity에서 참조할 수 없습니다. 프로퍼티가 null 가능이거나 @CascadeDeleteBy 어노테이션이 있는 경우에만 SharedEntity를 Entity에서 참조할 수 있습니다. 그렇지 않은 경우 런타임 예외가 발생합니다.

예:


  interface A: Entity {
    var x: SharedEntity
    var y: SharedEntity
  }

빠른 수정에서는 어노테이션을 생성하거나 타입을 null 가능으로 만듭니다


  interface A: Entity {
    var x: SharedEntity?
    @CascadeDeleteBy
    var y: SharedEntity
  }
