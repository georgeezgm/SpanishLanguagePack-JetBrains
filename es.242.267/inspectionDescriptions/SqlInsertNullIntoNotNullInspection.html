NOT NULL 값 이외에는 받지 않는 열에 NULL 값을 삽입하는 경우를 보고합니다.

예(Microsoft SQL 서버):

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

col1 및 col2는 NOT NULL로 정의되어 있기 때문에 이 열에는 NULL 값을 삽입할 수 없습니다. 스크립트를 그대로 실행하면 오류가 발생합니다. 이 코드를 수정하려면 VALUES 부분의 NULL을 어떤 값으로 바꾸세요(예: 42 및 'bird').

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
