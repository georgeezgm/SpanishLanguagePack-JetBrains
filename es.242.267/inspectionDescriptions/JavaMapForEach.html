Kotlin의 forEach로 대체 가능한 Java Map.forEach 메서드 호출을 보고합니다.

예:


  fun test(map: HashMap<Int, String>) {
      map.forEach { key, value ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}

빠른 수정은 소괄호를 추가합니다.


  fun test(map: HashMap<Int, String>) {
      map.forEach { (key, value) ->
          foo(key, value)
      }
  }

  fun foo(i: Int, s: String) {}
