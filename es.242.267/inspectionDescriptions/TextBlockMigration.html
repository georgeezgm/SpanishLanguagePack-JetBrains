텍스트 블록으로 대체하여 단순화할 수 있는 String 연결을 보고합니다.

요건:

  • \n이 2번 이상 발생
  • 텍스트 블록이 연결되지 않음

Report single string literals(단일 문자열 리터럴 보고) 옵션을 사용하면 줄 바꿈이 포함된 단일 리터럴이 강조 표시됩니다. 옵션이 비활성화되어 있어도 빠른 수정을 사용할 수 있습니다.

예:


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

빠른 수정 적용 후:


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
