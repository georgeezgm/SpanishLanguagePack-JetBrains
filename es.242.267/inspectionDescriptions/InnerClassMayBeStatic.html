static이 될 수 있는 내부 클래스를 보고합니다.

static 내부 클래스는 둘러싸는 인스턴스에 대한 묵시적 참조를 유지하지 않습니다. When using Java 17 or before, this prevents a common cause of memory leaks and uses less memory per instance of the class.

예:


  public class Outer {
    class Inner { // not static
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }

빠른 수정 적용 후:


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }
