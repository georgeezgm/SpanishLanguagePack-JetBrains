JVM 플랫폼에서 StringBuilder.appendRange(CharArray, startIndex, endIndex) 함수 호출로 대체되어야 하는 StringBuilder.append(CharArray, offset, len) 함수 호출을 보고합니다.

append 함수는 JVM, JS 및 Native 플랫폼에서 다르게 동작하므로 appendRange 함수를 사용하는 것이 좋습니다.

예:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          append(charArray, offset, len)
      }
  }

빠른 수정 적용 후:


  fun f(charArray: CharArray, offset: Int, len: Int): String {
      return buildString {
          appendRange(charArray, offset, offset + len)
      }
  }
