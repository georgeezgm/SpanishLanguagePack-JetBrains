다른 메서드 호출에 대한 매개변수로 사용되는 메서드 호출을 보고합니다.

빠른 수정에서는 변수를 삽입하여 코드를 단순화하고 디버그를 용이하게 합니다.

예:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

빠른 수정 적용 후:


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }
