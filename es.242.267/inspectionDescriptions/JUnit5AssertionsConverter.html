JUnit 5 테스트의 junit.framework.Assert, org.junit.Assert 또는 org.junit.Assume 클래스에 있는 메서드에 대한 호출을 보고합니다.

테스트가 적절히 작동하더라도 org.junit.jupiter.api.Assertions/org.junit.jupiter.api.Assumptions로 마이그레이션은 이전 JUnit 버전의 종속성을 줄이는 데 도움이 됩니다.

예:


  import org.junit.Assert;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      // 이전 Assert 클래스가 사용되었습니다
      Assert.assertEquals(4, 2 + 2);
    }
  }

빠른 수정 적용 후:


  import org.junit.jupiter.api.Assertions;
  import org.junit.jupiter.api.Test;

  public class MyTest {
    @Test
    public void simpleTest() {
      Assertions.assertEquals(4, 2 + 2);
    }
  }
