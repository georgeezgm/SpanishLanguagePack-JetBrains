현재 코드 스타일에 따라 정리되지 않은 import 문을 보고합니다(설정 | 에디터 | 코드 스타일 참조).

예:


  import java.util.List;
  import java.util.ArrayList;

  public class Example {
    List list = new ArrayList();
  }

'import 문 최적화' 빠른 수정 적용 후: 


  import java.util.ArrayList;
  import java.util.List;

  public class Example {
    List list = new ArrayList();
  }
