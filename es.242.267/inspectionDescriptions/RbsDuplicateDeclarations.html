클래스, 모듈, 인터페이스, 상수, 전역 변수, 타입 별칭 및 메서드의 중복 선언을 보고합니다.
  • 선언이 중복되면 DuplicatedDeclarationError가 발생합니다.
  • 메서드 정의가 중복되면 DuplicatedMethodDefinitionError가 발생합니다.
  • 인터페이스 메서드 정의가 중복되면 DuplicatedInterfaceMethodDefinitionError가 발생합니다.

중복 선언으로 이동하기 위해 빠른 수정을 사용할 수 있습니다.

예:

# 중복 인터페이스 선언
interface _Example
end

# 중복 인터페이스 선언
interface _Example
end
