엔트로피 분석 및 정규식을 사용하여 json에서 잠재적인 보안 토큰이나 비밀번호를 탐지합니다.

이 검사는 엔트로피 분석과 정규식을 활용하여 코드 베이스를 검색하여 보안 토큰과 비밀번호와 유사한 문자를 찾습니다. 개발자가 쉽게 보안 취약점을 찾고 해결할 수 있도록 발견된 결과를 강조 표시합니다. 이 검사의 효과는 구성에 정의된 패턴에 따라 달라지며, 여러 코딩 환경과 요구사항에 맞춰 변경할 수 있습니다. 


// 잠재적인 토큰으로 탐지된 문자열 예시:
'possibleToken': 'pa#sw@rd;'

// 탐지에 사용된 정규식 패턴 예시:
regexPattern = '/[0-9]+:AA[0-9A-Za-z\-_]{33}/'

이 주석 이후의 텍스트는 검사 설정에서만 표시됩니다.
