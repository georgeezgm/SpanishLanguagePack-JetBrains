분기에서 공통 부분을 추출 할 수있는 if 문을 보고합니다.

이러한 공통 부분은 조건으로부터 독립적이며 if 문을 이해하기 어렵게 만듭니다.

예:


  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

빠른 수정 적용 후:


  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();
