열거형 타입에 사용된, 누락된 것이 있는 switch 문을 보고합니다.

예:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {

      }
    }
  }

빠른 수정 적용 후:


  enum AlphaBetaGamma {
    A, B, C;

    void x(AlphaBetaGamma e) {
      switch (e) {
        case A -> {}
        case B -> {}
        case C -> {}
      }
    }
  }

default 분기가 있는 switch 문을 무시하려면 default 분기가 있는 switch 문 무시 옵션을 사용하세요.
