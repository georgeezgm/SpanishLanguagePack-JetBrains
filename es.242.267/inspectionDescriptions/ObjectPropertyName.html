이름 생성 규칙을 따르지 않는 프로퍼티를 보고합니다.

다음과 같은 프로퍼티가 보고됩니다.

  • 최상위 프로퍼티
  • 객체 및 컴패니언 객체의 프로퍼티

요구된 패턴을 검사 옵션에서 지정할 수 있습니다.

권장 이름 생성 규칙: 대문자로 시작해야 하고, 낙타 표기법을 사용해야 하며, 밑줄을 사용하지 않아야 합니다.

예:


  // 최상위 프로퍼티
  val USER_NAME_FIELD = "UserName"
  // 싱글턴 객체에 대한 참조를 유지한 최상위 프로퍼티
  val PersonComparator: Comparator<Person> = /*...*/

  class Person {
    companion object {
      // 컴패니언 객체의 프로퍼티
      val NO_NAME = Person()
    }
  }
