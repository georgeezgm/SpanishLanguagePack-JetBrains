접근 제어자가 좁혀진 필드, 메서드 혹은 클래스를 보고합니다.

예:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // private일 수 있습니다
  }

빠른 수정 적용 후:


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

검사의 옵션을 사용하여 제어자 변경 제안의 규칙을 정의합니다.
