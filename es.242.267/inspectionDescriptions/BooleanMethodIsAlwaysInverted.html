호출되는 경우 항상 부정되는 boolean 반환 타입을 가진 메서드를 보고합니다.

메서드를 반전시키고 선택적으로 이름을 바꾸는 빠른 수정이 제공됩니다. 성능상의 이유로, 문제가 있는 일부 메서드가 에디터에서 강조 표시되지 않을 수 있습니다.

예:


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

빠른 수정 적용 후:


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
