이 템플릿에서 #parse 지시문을 사용하여 파일 템플릿(파일 탭)에 포함할 코드 조각을 지정할 수 있습니다. 하나의 템플릿을 여러 개의 파일 템플릿에서 공유할 수 있습니다. Ruby 구문, 표현식 및 주석과 마찬가지로 매크로처럼 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${FILE_NAME}   Nombre de archivo actual
${USER}   현재 유저 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${DAY}   현재 일
${HOUR}   현재 시
${MINUTE}   현재 분
${PRODUCT_NAME}   현재 IDE 이름
${PROJECT_NAME}   현재 프로젝트 이름
