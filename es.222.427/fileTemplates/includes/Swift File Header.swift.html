#parse 지시문을 사용하여 파일 템플릿(Templates(템플릿) 탭)에 포함될 수 있는 코드 조각을 포함합니다.
이 템플릿은 편집할 수 있습니다. 정적 텍스트, 코드 및 주석과 함께, 매크로처럼 그에 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${FILE_NAME}   현재 프로젝트의 이름
${USER}   현재 사용자의 시스템 로그인 이름
${USER_NAME}   현재 사용자의 전체 이름
${ORGANIZATION_NAME}   Xcode 프로젝트의 조직 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자 예: Jan, Feb 등등.
${MONTH_NAME_FULL}   현재 월의 전체 이름 예: January, February 등등.
${DAY}   현재 월의 일
${DAY_NAME_SHORT}   현재 요일의 첫 세 글자 예: Mon, Tue 등.
${DAY_NAME_FULL}   현재 요일의 전체 이름 예: Monday, Tuesday 등.
${HOUR}   현재 시
${MINUTE}   현재 분
${PROJECT_NAME}   현재 프로젝트의 이름
${COMPILER_VERSION_MAJOR}   Swift 컴파일러 메이저 버전
${COMPILER_VERSION_MINOR}   Swift 컴파일러 마이너 버전
${PREFER_EXPLICIT_RETURN}   Prefer explicit return(명시적 반환 선호) 옵션의 값(Preferences(환경 설정) | Editor(에디터) | Code Style(코드 스타일) | Swift | Code Generation(코드 생성))
