프로그램을 디버그할 때 코드 실행 흐름을 검사하고 프로그램의 현재 상태를 이해해야 하는 경우가 있습니다. Go 코어 덤프는 실행 중인 프로세스의 메모리 덤프와 프로그램 수명 동안 프로그램의 상태를 포함하는 파일입니다.

GoLand에서 Go 코어 덤프를 열고 탐색할 수 있습니다. Go 코어 덤프를 열려면 실행 | 코어 덤프 열기 로 이동합니다. 실행 파일과 덤프 파일이 있는 경로를 지정합니다. 확인 을 클릭합니다.

참고: 오직 Linux 시스템에서만 Go 코어 덤프 파일을 만들 수 있습니다. GoLand을(를) 지원하는 모든 운영 체제에서 덤프 파일을 볼 수 있습니다.
