데이터베이스 상태를 자동으로 동기화하려면 Database Explorer (보기 | Tool Windows | Database Explorer )에서 Data Source Properties 을(를) 클릭하고 변경할 데이터 소스를 선택합니다. 옵션 탭에서 자동 동기화 체크박스를 선택합니다.

Auto sync 체크박스를 해제하면 새로고침 아이콘 을 클릭하거나 &shortcut:Refresh; 을(를) 누를 때만 Database Explorer (뷰 | Tool Windows | Database Explorer )의 데이터 소스 뷰가 데이터베이스의 실제 상태와 동기화됩니다.
