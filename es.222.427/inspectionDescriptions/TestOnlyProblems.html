@TestOnly 및 @VisibleForTesting 어노테이션이 추가된 메서드 및 클래스가 프로덕션 코드에서 사용된 경우를 보고합니다. 또한 @TestOnly 및 @VisibleForTesting를 동일한 요소에 적용하는 경우도 보고합니다.

그러한 메서드 또는 클래스가 다음에서 참조되는 경우에는 문제가 보고되지 않습니다.

  • Test Sources 폴더에 있는 코드
  • 테스트 클래스(JUnit/TestNG)
  • @TestOnly 어노테이션이 추가된 다른 메서드

예(프로덕션 코드):


  @TestOnly
  fun foo() { ... }

  fun main () {
    foo()
  }
