이름이 너무 짧거나 너무 길거나 지정된 정규 표현식 패턴을 따르지 않는 클래스를 보고합니다.

예: 검사가 테스트에 대해 활성화되어 있으며 클래스 이름의 최소 길이가 8(디폴트)로 지정되어 있는 경우, 다음 테스트 클래스는 이름 길이가 8보다 작은 6이므로 경고가 발생합니다. public class MyTest{}

그러한 클래스의 이름을 변경하는 빠른 수정은 에디터에서만 사용할 수 있습니다.

검사 구성:

옵션 섹션의 목록을 사용하여 검사되어야 하는 클래스를 지정하세요. 검사를 건너뛰려는 클래스의 체크박스를 선택 해제하세요.

각 클래스 타입에 대해 제공된 입력 필드를 사용하여 최소 길이, 최대 길이를 비롯해 클래스 이름에 필요한 정규 표현식을 지정하세요. 대응하는 검사를 건너뛰려면 length 필드에 0을 지정하세요.

정규 표현식은 표준 java.util.regex 서식으로 지정되어야 합니다.
