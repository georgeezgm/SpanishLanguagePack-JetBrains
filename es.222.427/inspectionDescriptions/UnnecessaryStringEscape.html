String 및 선택적으로 char 리터럴 내에서 불필요하게 이스케이프된 문자를 보고합니다.

이스케이프된 탭 문자 \t는 보고되지 않습니다. 탭 문자는 이스케이프되지 않으면 보이지 않게 됩니다.

예:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

빠른 수정 적용 후:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;
