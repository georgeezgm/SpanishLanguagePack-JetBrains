Int 유형의 식을 보고하여 Kotlin 1.9 이상에서 컴파일 오류를 발생시킵니다.

예:


  fun takeByte(x: Byte) {}

  fun foo() {
      takeByte(1 + 1) // will be resolved to Int in 1.9
  }

빠른 수정 적용 후:


  fun takeByte(x: Byte) {}

  fun foo() {
      takeByte((1 + 1).toByte()) // will be resolved to Int in 1.9
  }

Kotlin 언어 레벨은 1.7부터 검사가 가능합니다.
