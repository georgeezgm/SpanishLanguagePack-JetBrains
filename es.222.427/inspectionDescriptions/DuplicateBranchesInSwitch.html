다른 브랜치에 동일한 코드가 포함된 switch 문 또는 표현식을 보고하고 중복 브랜치 병합을 제안합니다.

예:


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

다음으로 대체 가능:


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
