대상 인수를 삽입하여 간소화할 수 있는 when 식을 보고합니다.

예:


  fun test(obj: Any): String {
      return when {
          obj is String -> "string"
          obj is Int -> "int"
          else -> "unknown"
      }
  }

빠른 수정에서는 대상 인수를 삽입합니다.


  fun test(obj: Any): String {
      return when (obj) {
          is String -> "string"
          is Int -> "int"
          else -> "unknown"
      }
  }
