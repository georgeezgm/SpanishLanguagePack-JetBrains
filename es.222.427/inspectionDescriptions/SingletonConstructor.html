비 엄격으로 선언되지 않았으며 @Singleton 어노테이션이 추가된 클래스의 생성자를 보고합니다.

예:


  @Singleton
  class Foo{
      Foo(){
      }
  }

가능한 빠른 수정은 두 개입니다. 하나는 생성자를 제거하는 것이고 다른 하나는 @Singleton을 비 엄격으로 선언하는 것입니다.

빠른 수정 적용 후:


  @Singleton
  class Foo{
  }

or:


  @Singleton(strict = false)
  class Foo{
      Foo(){
      }
  }
