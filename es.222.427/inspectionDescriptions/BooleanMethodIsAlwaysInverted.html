부정 컨텍스트에만 사용되는 boolean 반환 타입을 가진 메서드를 보고합니다.

빠른 수정을 통해 메서드의 이름을 변경하고 반전할 수 있습니다. 성능상의 이유로 일부 메서드는 에디터에서 강조 표시되지 않을 수 있습니다.

예:


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

빠른 수정 적용 후:


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
