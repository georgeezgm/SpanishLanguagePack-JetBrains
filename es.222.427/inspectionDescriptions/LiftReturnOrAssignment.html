return 문 또는 대입을 밖으로 꺼내면 표현식으로 변환할 수 있는 if, when 및 try 문을 보고합니다.

예:


  fun foo(arg: Int): String {
      when (arg) {
          0 -> return "Zero"
          1 -> return "One"
          else -> return "Multiple"
      }
  }
빠른 수정 적용 후: 

  fun foo(arg: Int): String {
      return when (arg) {
          0 -> "Zero"
          1 -> "One"
          else -> "Multiple"
      }
  }
