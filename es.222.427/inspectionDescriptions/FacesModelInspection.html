Java Server Faces faces-config.xml 파일 내 잘못된 요소를 보고합니다.

  • 인스턴스화할 수 없는 클래스 참조
  • 필요한 클래스를 확장하지 않는 클래스 참조
  • 부적절한 범위를 가진 클래스 참조
  • 빈 태그 및 속성 값
  • 필요한 패턴(예: Java 식별자)과 일치하지 않는 태그 및 속성 값
  • 필요한 하위 태그 또는 속성을 포함하지 않는 태그
  • 이름이 중복된 객체를 정의하는 태그

예:


  <faces-config >
     <application>
        <action-listener>java.lang.String</action-listener>  <!-- 'java.lang.String'을 'javax.faces.event.ActionListener'에 대입할 수 없습니다  -->
    </application>
    <navigation-rule>
        <from-view-id>myView.xhtml</from-view-id>
        <navigation-case>
            <to-view-id>unknown.xhtml</to-view-id>  <!-- 'unknown.xhtml'을 해결할 수 없습니다  -->
        </navigation-case>
    </navigation-rule>
  </faces-config>
