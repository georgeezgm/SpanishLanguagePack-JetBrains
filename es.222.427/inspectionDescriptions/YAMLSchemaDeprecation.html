YAML 파일에서 지원이 중단된 키를 보고합니다.

해당하는 YAML 파일과 연관된 JSON 스키마가 있을 경우에만 지원 중단이 검사됩니다.

지원 중단 메커니즘은 아직 JSON 스키마 사양에 정의되어 있지 않으며 이 검사는 비표준 deprecationMessage 확장을 사용합니다.

스키마 지원 중단 예시:


  {
    "properties": {
      "SomeDeprecatedProperty": {
        "deprecationMessage": "Baz",
        "description": "Foo bar"
      }
    }
  }

다음은 해당하는 경고의 예시입니다.


  SomeDeprecatedProperty: some value
