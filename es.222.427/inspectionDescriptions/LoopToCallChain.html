일련의 stdlib 연산(map, filter 등)으로 대체 가능한 for 루프를 보고합니다.

예:


fun foo(list: List<String>): List<Int> {
  val result = ArrayList<Int>()
  for (s in list) {
     if (s.length > 0)
       result.add(s.hashCode())
     }
  return result
}

빠른 수정 적용 후:


fun foo(list: List<String>): List<Int> {
  val result = list
    .filter { it.length > 0 }
    .map { it.hashCode() }
  return result
}
