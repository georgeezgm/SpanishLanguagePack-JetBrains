항상 true이거나 항상 false, 또는 불필요한 bool 표현식의 일부를 보고합니다. 이러한 bool 표현식은 단순화하여 코드의 가독성을 높일 수 있습니다. 일부 경우에 이는 다른 이슈가 있음을 나타냅니다.

예:

func isNonZero(x, y int) bool {
  // 두 번째 비교는 항상 true 이거나
  // 전혀 실행되지 않습니다
  return x > 0 && x > 0
}

x > 0 && x > 0 부분에 대해 표현식 단순화 빠른 수정을 적용할 수 있습니다. 빠른 수정 적용 후, 표현식은 다음과 같습니다: x > 0.
