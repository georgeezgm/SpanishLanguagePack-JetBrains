N 접두사 없이 내셔널 문자를 사용하는 문자열 리터럴을 보고합니다.

N 접두사가 없으면 문자열은 데이터베이스의 디폴트 코드 페이지로 변환됩니다. 이 디폴트 코드 페이지는 특정 문자를 인식하지 못할 수 있습니다. 더 자세한 내용은 docs.microsoft.com에서 nchar 및 nvarchar(Transact-SQL)를 참고하세요.

예(Microsoft SQL 서버):

SELECT 'abcde' AS a;
SELECT N'abcde' AS b;
SELECT 'абвгд' AS c;
SELECT N'абвгд' AS d;

SELECT 'абвгд' AS c;에는 N 접두사가 없기 때문에 'абвгд' 부분이 강조 표시됩니다.
