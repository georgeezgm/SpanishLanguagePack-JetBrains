일부 플랫폼 모듈에서 실제 어노테이션이 없으나 선택적으로 필요한 어노테이션을 보고합니다.

예:

// 공통 코드
@OptionalExpectation
expect annotation class JvmName(val name: String)

@JvmName(name = "JvmFoo")
fun foo() { }

// jvm 코드
actual annotation class JvmName(val name: String)

또한 이 검사는 actual annotation class JvmName이 JVM이 아닌 플랫폼(예: Native)에서 생략된 경우도 보고합니다.
