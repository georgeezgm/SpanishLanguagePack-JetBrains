반복적으로 추가되는 java.lang.String으로 선언된 변수를 보고합니다. 이러한 변수는 java.lang.StringBuffer 또는 java.lang.StringBuilder로 더욱 효율적으로 선언할 수 있습니다.

예:


  String s = "";
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s = s + name;
  }

이러한 루프는 다음으로 바꿀 수 있습니다.


  StringBuilder s = new StringBuilder();
  for (int i = 0; i < names.length; i++) {
    String name = names[i] + (i == names.length - 1 ? "" : " ");
    s.append(name);
  }

또는 다음과 바꿀 수도 있습니다.


  String s = String.join(" ", names);

옵션을 사용하여 이 검사를 통해 변수가 루프에서 추가될 때 보고할 수 있도록 합니다.
