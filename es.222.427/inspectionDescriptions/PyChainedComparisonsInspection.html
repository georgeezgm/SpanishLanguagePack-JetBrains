단순화할 수 있는 체인 비교를 보고합니다.

예:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if x >= xmin and x <= xmax:
          pass

IDE는 if x >= xmin and x <= xmax를 단순화할 것을 제의합니다. 빠른 수정이 적용되면 코드는 다음과 같이 변경됩니다:


  def do_comparison(x):
      xmin = 10
      xmax = 100
      if xmin <= x <= xmax:
          pass
