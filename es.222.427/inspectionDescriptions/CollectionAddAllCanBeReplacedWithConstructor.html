no-arg 생성자를 사용하여 컬렉션을 인스턴스화한 직후 Collection.addAll() 및 Map.putAll() 호출을 보고합니다.

이러한 구문은 매개변수화된 생성자에 대한 단일 호출로 대체할 수 있으며, 이로써 코드가 간소화됩니다. 또한 일부 대체 컬렉션은 보다 잘 작동할 수 있습니다.

예:

  Set<String> set = new HashSet<>();
  set.addAll(Arrays.asList("alpha", "beta", "gamma"));

다음으로 대체 가능:

  Set<String> set = new HashSet<>(Arrays.asList("alpha", "beta", "gamma"));

기본적으로 표준 JDK 컬렉션 클래스가 지원됩니다. 또한 검사 옵션에서 다른 클래스를 지정할 수도 있습니다.
