TestNG org.testng.annotations.Configuration 어노테이션을 보고합니다.

최신 @BeforeXXX/@AfterXXX 어노테이션으로 테스트 스위트를 구성하는 것이 더 낫습니다.

예:


  public class Test {
    @Configuration(beforeSuite = true, afterTest = true)
    public void afterBefore(){
    }
  }

빠른 수정 적용 후:


  public class Test {
    @AfterTest()
    @BeforeSuite()
    public void afterBefore(){
    }
  }
