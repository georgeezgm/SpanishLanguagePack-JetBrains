유형 시그니처에서 NilClass, TrueClass 및 FalseClass 클래스에 대한 참조를 보고합니다. 이러한 클래스의 사용은 불필요하게 장황합니다.

NilClass, TrueClass 또는 FalseClass 사용 위치를 각각 해당하는 리터럴로 교체하기 위해 빠른 수정을 사용할 수 있습니다.

예:

# 'nil' 대신 'NilClass'가 참조됨...
type example = NilClass | TrueClass | FalseClass

빠른 수정 적용 후:

type example = nil | true | false
