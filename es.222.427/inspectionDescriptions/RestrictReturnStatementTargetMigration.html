함수를 가리키지 않는 라벨을 보고합니다.

함수를 가리키지 않는 대상 라벨을 선언하는 것은 금지되어 있습니다.

빠른 수정에서는 그러한 라벨을 제거합니다.

예:


  fun testValLabelInReturn() {
      L@ val fn = { return@L }
      fn()
  }

빠른 수정 적용 후:


  fun testValLabelInReturn() {
      L@ val fn = { return }
      fn()
  }

이 검사는 프로젝트 또는 모듈의 언어 버전이 1.4 이상인 경우에만 보고합니다.
