포함된 클래스 이름으로 한정된 정적으로 가져온 멤버의 사용을 보고합니다.

정적으로 가져온 멤버는 멤버 이름으로 바로 액세스할 수 있기 때문에 그러한 한정은 불필요하며 삭제할 수 있습니다.

예:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(Test.WIDTH);
    }
  }

빠른 수정 적용 후:


  import static foo.Test.WIDTH;

  class Bar {
    void bar() {
      System.out.println(WIDTH);
    }
  }
