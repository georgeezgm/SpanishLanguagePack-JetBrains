코드에 선언되었지만 전혀 사용하지 않는 전역 변수를 보고합니다.

사용하지 않는 변수가 있는 경우 코드는 컴파일되지 않습니다. 사용하지 않는 변수 및 가져오기에 대한 더 자세한 내용은 golang.org의 Unused imports and variables(을)를 참고하세요. 

func main() {
 a := 422
}

예시의 코드는 컴파일되지 않습니다. 따라서 오류로 강조 표시됩니다. 이러한 경우 두 가지의 빠른 수정을 적용할 수 있습니다: 변수 삭제 및 _(으)로 이름 변경. 첫 번째 빠른 수정은 변수를 삭제하며 두 번째 빠른 수정은 변수를 빈 식별자로 변환합니다.

_(으)로 이름 변경 빠른 수정 적용 후:

func main() {
 _ := 422
}
