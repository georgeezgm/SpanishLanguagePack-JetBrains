변수 타입이 형 변환 타입으로 좁혀진 경우 제거할 수 있는 형 변환을 보고합니다.

예:


  Object x = "  string  ";
  System.out.println(((String)x).trim());

여기서 x 타입을 String으로 변경하는 것은 불필요한 형 변환입니다. 제안된 빠른 수정을 통해 변수 타입을 업데이트하고 해당 변수에 대한 모든 불필요한 형 변환을 제거할 수 있습니다. 


  String x = "  string  ";
  System.out.println(x.trim());
