1.9 이후 컴파일 오류를 유발하는 함수 상위 타입에 대한 생성사 호출을 보고합니다.

동기 유형:

  • 구현이 게시된 사양 또는 문서를 준수하지 않습니다.

자세한 정보: KT-46344: 상위 타입 목록의 함수 인터페이스에서 상위 클래스 생성자 호출에 대한 오류가 없습니다.

빠른 수정으로 생성자 호출을 제거합니다.

예:


  추상 클래스 A : () -> Int()

빠른 수정 적용 후:


  추상 클래스 A : () -> Int

이 검사는 프로젝트 또는 모듈의 Kotlin 언어 수준이 1.7 이상인 경우에만 보고합니다.
