상위 클래스 var를 숨기는 클래스 매개 변수를 보고합니다.

예:

class Animal {
  var name: String = ""
}

class Cat(name: String) extends Animal {
  // 'name'을 var를 숨기는 private[this] val으로 만듭니다.
  def usage = name
}

private이 아닌 var로 클래스를 확장하면, 사용자는 하위 클래스의 클래스 매개변수가 상위 클래스에서 var를 재정의하게 될 수 있지만, 실제로는 숨길 뿐입니다. 때때로 사소하지 않은 오류를 초래할 수도 있다.

강조 표시는 실제로 값을 사용하는 경우에만 표시됩니다.
