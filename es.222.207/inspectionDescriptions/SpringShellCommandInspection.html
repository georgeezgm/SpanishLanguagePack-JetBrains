올바르지 않은 Spring Shell 명령 선언을 보고합니다.

  • 명령 설명 누락
  • 명령이 org.springframework.shell.standard.@ShellComponent 안에 없습니다

예:


// @ShellComponent가 없으므로 모든 메서드가 보고됩니다
class MathCommands {

  @ShellMethod("")
  public int sum(int a, int b) { // 설명이 비어 있으므로 메서드가 보고됩니다
    return a + b;
  }

  @ShellMethod
  public int mul(int a, int b) { // 설명이 비어 있으므로 메서드가 보고됩니다
    return a * b;
  }
}
