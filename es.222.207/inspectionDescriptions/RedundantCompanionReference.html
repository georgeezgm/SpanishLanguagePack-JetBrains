불필요한 Companion 참조를 보고합니다.

예:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.Companion.create()
  }

빠른 수정 적용 후:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.create()
  }
