Javadoc 주석의 빈 줄을 보고합니다.

Javadoc의 빈 줄은 텍스트를 다른 단락으로 분할하려는 인텐션 신호일 수 있습니다. 하지만 Javadoc 도구 및 IntelliJ IDEA는 문서 주석을 렌더링할 때 이를 무시합니다.

빠른 수정에서는 빈 줄을 단락 태그(<p>)로 교체할 것을 제안합니다.

예:


  class Main {
    /**
     * 아무 작업도 하지 않습니다.
     *
     * 아무 작업도 하지 않습니다
     */
    void foo() {}
  }

빠른 수정 적용 후:


  class Main {
    /**
     * 아무 작업도 하지 않습니다.
     * <p>
     * 아무 작업도 하지 않습니다
     */
    void foo() {}
  }
