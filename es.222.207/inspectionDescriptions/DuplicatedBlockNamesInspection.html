Django 템플릿 내 중복된 블록 이름을 보고합니다.

예:

<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="style.css">
    <title>{% block title %}My amazing site{% endblock %}</title>
</head>

<body>
    <div id="sidebar">
        {% block title %}
        <ul>
            <li><a href="/">홈</a></li>
            <li><a href="/blog/">블로그</a></li>
        </ul>
        {% endblock %}
    </div>

    <div id="content">
        {% block content %}{% endblock %}
    </div>
</body>
</html>
