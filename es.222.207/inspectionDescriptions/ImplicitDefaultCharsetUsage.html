플랫폼 디폴트 문자 집합을 묵시적으로 사용하는 메서드 및 생성자를 호출을 보고합니다. 이러한 호출은 다른 기본 문자집합을 사용하는 시스템에서 결과가 달라질 수 있으며 예기치 않은 동작이 일어날 수도 있습니다.

예:

void foo(byte[] bytes) {
  String s = new String(bytes);
}

해당하는 오버로드된 메서드를 사용할 수 있는 경우 명시적인 UTF-8 문자 집합을 지정하는 빠른 수정을 사용할 수 있습니다. 빠른 수정 적용 후:

void foo(byte[] bytes) {
  String s = new String(bytes, StandardCharsets.UTF_8);
}
