for 루프 내에서 Collection.indices 또는 컬렉션에 대한 반복으로 바꿀 수 있는 until 및 rangeTo 연산자를 보고합니다.

문법적 설탕을 사용하면 코드를 단순화할 수 있습니다.

빠른 수정에서는 수동 범위를 그에 대응하는 구조로 바꿉니다.

예:


  fun main(args: Array<String>) {
      for (index in 0..args.size - 1) {
          println(args[index])
      }
  }

빠른 수정 적용 후:


  fun main(args: Array<String>) {
      for (element in args) {
          println(element)
      }
  }
