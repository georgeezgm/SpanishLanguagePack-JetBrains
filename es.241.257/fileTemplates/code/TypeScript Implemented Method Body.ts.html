기본 제공 템플릿은 예를 들어, 사용 위치에서 메서드 생성 인텐션 동작
과 같이 메서드가 생성될 때 메서드의 본문 작성에 사용됩니다. 템플릿은 편집할 수 있습니다. TypeScript 표현식 및 주석과 함께, 사전 정의된 변수를 사용할 수도 있습니다. 이러한 변수는 템플릿이 적용될 때 해당 값으로 바뀝니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${RETURN_TYPE}   생성된 메서드의 반환 타입
${DEFAULT_RETURN_VALUE}   메서드에서 기본으로 반환되는 값
${METHOD_NAME}   생성된 메서드 이름
${SEMICOLON}   코드 스타일 옵션에 따라 세미콜론 또는 비어 있음
파일 헤더 템플릿 (포함 탭)의 모든 사전 정의된 변수도 사용 가능합니다. (예: ${USER}, ${DATE}, 등.)
