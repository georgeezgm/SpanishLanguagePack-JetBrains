예를 들어 Add Javadoc(Javadoc 추가) 인텐션 액션이 적용될 때 클래스, 인터페이스 또는 레코드에 대한 JavaDoc을 생성하는 데 사용되는 템플릿을 정의합니다.

Java 표현식 및 주석과 함께, 매크로처럼 해당 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.

#parse 지시문을 사용하여 Includes(포함) 탭의 템플릿을 포함시킬 수 있습니다. 템플릿을 포함하려면, 템플릿의 전체 이름을 따옴표로 묶인 매개변수로 지정하세요(예: #parse("File Header.java").
사전 정의된 변수는 다음의 값을 사용합니다:
${RECORD_COMPONENTS}   레코드 매개변수 목록
${TYPE_PARAMS}   타입 매개변수 이름 목록
${CONTAINING_CLASS}   생성된 클래스 이름
${INTERFACE}   JavaDoc 소유자가 인터페이스이면 true를 반환합니다
${ENUM}   JavaDoc 소유자가 열거형이면 true를 반환합니다
${RECORD}   JavaDoc 소유자가 레코드면 true를 반환합니다
${PACKAGE_NAME}   새 열거형이 생성되는 패키지 이름
${NAME}   새 클래스 생성 대화상자에서 지정한 새 열거형 이름
${USER}   현재 사용자의 시스템 로그인 이름
${DATE}   현재 시스템 날짜
${TIME}   현재 시스템 시간
${YEAR}   현재 년도
${MONTH}   현재 월
${MONTH_NAME_SHORT}   현재 월 이름의 첫 세 글자(Jan, Feb 등)
${MONTH_NAME_FULL}   현재 월 이름 전체(January, February 등)
${DAY}   현재 일
${HOUR}   현재 시
${MINUTE}   현재 분
${PROJECT_NAME}   현재 프로젝트 이름
