예를 들어 Code(코드) | Override Methods(메서드 재정의) 액션을 사용할 때 생성되는 재정의된 함수의 본문을 생성합니다.
템플릿을 편집할 때 C++ 표현식, 주석 및 사전 정의된 다음 변수를 사용할 수 있습니다.
${CALL_SUPER}   void 함수에 대한 기본 함수 호출 — 매개변수가 있거나 없는 BaseClass::functionName() 그 외 함수 - 매개변수가 있거나 없는 return BaseClass::functionName()
${RETURN_TYPE}   함수의 반환 타입입니다.
${DEFAULT_RETURN_VALUE}   함수의 디폴트 반환 값입니다.
C File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다 (예: ${USER} 및 ${DATE})
