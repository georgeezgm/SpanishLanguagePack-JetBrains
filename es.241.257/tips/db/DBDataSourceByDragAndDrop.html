Utilice un archivo SQL como fuente de datos DDL

Se puede utilizar un archivo SQL que contenga una declaración CREATE como fuente de datos. De esta forma, puede crear un diagrama y hacer referencia a todas las tablas, columnas y otros objetos definidos en ese archivo en el editor.

Para crear una fuente de datos DDL, abra el archivo DDL en el Administrador de archivos.explorador de bases de datos (ver | Ventana de herramientas |explorador de bases de datos ) y arrástrelo.
