Finalización de código

A medida que escribe, la finalización del código sugiere opciones accesibles en la posición actual del cursor.&shortcut:CodeCompletion; Al hacer doble clic se expande la lista de sugerencias.

Cuando utiliza la finalización de código para recuperar un nombre de clase, si esta clase aún no se ha importado, se genera automáticamente una declaración de importación.
