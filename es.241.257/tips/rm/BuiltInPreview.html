Vista previa incorporada

La vista previa incorporada de RubyMine muestra el resultado de su código HTML en una pestaña del editor separada. RubyMine guarda los cambios en archivos HTML o archivos CSS y JavaScript vinculados y recarga automáticamente la vista previa.

Para abrir la vista previa, coloque el cursor sobre su código HTML para abrir una ventana emergente del navegador y luego haga clic en el icono de RubyMine.
