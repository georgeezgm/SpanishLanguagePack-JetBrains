보통 self라고 이름이 지정되는 첫 번째 파라미터가 없는 메서드를 보고합니다.

예:


class Movie:

   def show():
       pass

빠른 수정이 적용되면 코드는 다음과 같이 변경됩니다:


class Movie:

   def show(self):
       pass

이 검사는 클래스 메서드 내 이름 지정 이슈 또한 보고합니다.

예:


class Movie:
    @classmethod
    def show(abc):
        pass

클래스 메서드의 첫 번째 매개변수는 cls여야 하므로 IDE는 해당 매개변수 이름을 변경하는 빠른 수정을 제공합니다.
