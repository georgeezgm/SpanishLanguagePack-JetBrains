열거형 타입에서 final이 아닌 필드를 보고합니다. final이 아닌 필드는 일반적으로 권장되지 않는 전역 가변 상태를 도입합니다.

예:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

빠른 수정 적용 후:


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
''final'로 만들 수 없는 필드 무시' 옵션을 사용하면 빠른 수정을 사용하여 final로 만들 수 있는 필드만 경고합니다.
