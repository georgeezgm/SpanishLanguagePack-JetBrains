단순화할 수 있는 부정된 bool 표현식을 보고합니다.

빠른 수정에서는 bool 표현식을 단순화합니다.

예:


    fun test(n: Int) {
        !(0 == 1)
    }

빠른 수정 적용 후:


    fun test(n: Int) {
        0 != 1
    }

IEEE-754 NaN 값이 포함된 경우, 이 액션에 의해 코드 의미가 변경될 수 있습니다.


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

빠른 수정 적용 후:


    fun main() {
        println(Double.NaN < 0) // false
    }
