List.contains() 메서드로 바꿀 수 있는 모든 List.indexOf() 표현을 보고합니다.

예:


  boolean hasEmptyString(List<String> list) {
    // 경고: 단순화될 수 있습니다
    return list.indexOf("") >= 0;
  }

제공된 빠른 수정은 indexOf 호출을 contains 호출로 바꿉니다:


  boolean hasEmptyString(List<String> list) {
    // 빠른 수정이 적용되었습니다
    return list.contains("");
  }
