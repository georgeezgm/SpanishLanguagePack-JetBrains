지역 클래스를 보고합니다.

지역 클래스는 코드 블록 내에 선언되었으며 이름이 지정된 중첩 클래스입니다. 지역 클래스는 일반적이지 않으므로 혼동될 수 있습니다. 또한 일부 코드 표준은 지역 클래스 사용을 권장하지 않습니다.

예:


  class Example {
    void test() {
      class Local { // here
      }
      new Local();
    }
  }

빠른 수정 적용 후:


  class Example {
    void test() {
      new Local();
    }

    private static class Local { // here
    }
  }
