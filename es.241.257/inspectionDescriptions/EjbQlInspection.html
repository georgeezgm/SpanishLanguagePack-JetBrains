EJB QL 쿼리 내부의 다음과 같은 오류를 보고합니다.
  • 일치하지 않는 표현식 타입
  • 잘못된 매개변수
  • 빈 조건 또는 상수 조건
  • 해결되지 않은 심볼

쿼리 예:


SELECT OBJECT(e) FROM Employee e WHERE SQRT(ABS(e.firstName - ?1)) > 50000; // 오류: 타입 불일치. 숫자 타입이 필요합니다

SELECT e.firstName FROM Employee e WHERE SUBSTRING(e.firstName, ?1, ?2) = ?4; // 오류: 실제 쿼리 매개변수가 3개 밖에 없습니다
