do-while 루프로 작성하면 더 효율적일 수 있는 while 루프를 보고합니다. 반복문 전의 코드와 반복문 본문의 코드가 동일한 while 반복문이 있습니다. do-while 반복문으로 대체하면 중복 코드를 제거할 수 있습니다. 이러한 중복 코드가 없는 while 반복문을 작성하기 위한 빠른 수정이 강조 표시 없이 에디터에 제안됩니다.

예:


  foo();
  while (x) {
      foo();
  }

다음으로 대체 가능:


  do {
    foo();
  } while (x);
