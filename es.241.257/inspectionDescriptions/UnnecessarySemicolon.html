불필요한 세미콜론 및 클래스 멤버 사이, 블록 구문 내부, 또는 클래스 정의 다음에 사용된 세미콜론을 보고합니다.

이러한 세미콜론은 Java에서 유효하지만, 불필요하며 제거될 수 있습니다.

예:


  class C {
    ;
    void m() throws Exception {
        try (AutoCloseable r1 = createAutoCloseable();) {
          ;
        }
    }
    ;
  }

빠른 수정 적용 후:


  class C {
    void m() throws Exception {
      try (AutoCloseable r1 = createAutoCloseable()) {
      }
    }
  }
