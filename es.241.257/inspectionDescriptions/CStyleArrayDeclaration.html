배열 대괄호가 변수 이름 뒤에 또는 메서드 매개변수 목록 뒤에 오는 C 스타일 구문으로 작성된 배열 선언을 보고합니다. 대부분의 코드 스타일은 배열 대괄호가 타입 이름 뒤에 있는 Java 스타일 배열 선언을 선호합니다.

예:


  public String process(String value[])[] {
    return value;
  }

빠른 수정 적용 후:


  public String[] process(String[] value) {
    return value;
  }

검사 구성:

메서드 반환 타입의 C 스타일 배열 선언만 보고하려면 변수의 C 스타일 선언 무시 옵션을 사용하세요.
