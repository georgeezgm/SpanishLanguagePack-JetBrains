배치 코드 검사가 실행된 결과에서 파일 내 중요한 문제(예: 구문 오류)를 보고합니다. 이러한 문제는 에디터에서 항상 강조 표시되며 검사와 달리 구성할 수 없습니다. 해당 옵션은 이 검사가 수행하는 확인의 범위를 제어합니다.
  • '구문 오류 보고' 옵션: 파서 관련 문제를 보고합니다.
  • '특정 언어의 어노테이터에서 발생한 문제 보고' 옵션: 관련 언어에 구성된 어노테이터에서 발견된 문제를 보고합니다. 자세한 내용은 사용자 지정 언어 지원: 어노테이터를 참조하세요.
  • '기타 강조 표시 문제 보고' 옵션: 현재 파일의 언어에 한정된 문제를 보고합니다(예: 타입 불일치 또는 보고되지 않은 예외). 자세한 내용은 사용자 지정 언어 지원: 강조 표시를 참조하세요.
