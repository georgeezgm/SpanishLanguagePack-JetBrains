배열 반복 처리 앞에 오는 불필요한 배열 길이 검사를 보고합니다. 배열 길이가 0인 경우 반복 처리를 건너뛰므로, 명시적으로 길이를 검사할 필요가 없습니다.

예:


  void f(String[] array) {
    if (array.length != 0) { // 불필요한 검사
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

빠른 수정에서는 길이 검사를 제거하거나 래핑 해제하도록 제안합니다.


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
