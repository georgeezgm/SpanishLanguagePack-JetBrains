ISNULL 함수 내 문자열 인수 자르기를 보고합니다.

ISNULL 구문은 ISNULL(check_expression, replacement_value)입니다.

docs.microsoft.com의 ISNULL에 따르면 replacement_value이 check_expression보다 긴 경우 replacement_value이 잘립니다.

예(Microsoft SQL 서버):

DECLARE @name1 VARCHAR(2) = NULL;
DECLARE @name2 VARCHAR(10) = 'Example';
DECLARE @name3 VARCHAR(2) = 'Hi';

  -- '@name2'은 VARCHAR(10)이므로 잘립니다
SELECT ISNULL(@name1, @name2);

  -- '@name3'은 '@name1'와 같이 VARCHAR(2)이므로 잘리지 않습니다
SELECT ISNULL(@name1, @name3);
