소괄호 스타일이 잘못된 메서드 정의를 보고합니다. 매개변수가 없는 메서드 정의와 반대로 매개변수가 있는 메서드 정의에는 소괄호가 필요합니다.

예:

# 나쁜 사례
def example_method(); end
def example_method_with_parameters param1, param2; end
소괄호를 제거하거나 각각 추가하기 위해 빠른 수정을 사용할 수 있습니다. 빠른 수정이 적용된 후 결과는 다음과 같습니다: 
# 좋은 사례
def example_method; end
def example_method_with_parameters(param1, param2); end
