Javadoc 주석의 일반 텍스트 링크를 보고합니다.

빠른 수정에서는 <a> 태그로 링크를 래핑하도록 제안합니다.

예:


  class Main {
   /**
    * https://en.wikipedia.org/
    */
    void foo() {}
  }

빠른 수정 적용 후:


  class Main {
   /**
    * <a href="https://en.wikipedia.org/">https://en.wikipedia.org/</a>
    */
    void foo() {}
  }
