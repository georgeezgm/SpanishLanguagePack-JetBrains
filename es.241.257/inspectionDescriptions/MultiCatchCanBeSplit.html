여러 개의 catch 섹션을 보고하고 각각의 catch 블록으로 분리할 것을 제안합니다.

예:


  try {
    int i = getIndex();
  } catch (NullPointerException|IndexOutOfBoundsException e) {
    e.printStackTrace();
  }

빠른 수정 적용 후:


  try {
    int i = getIndex();
  } catch (NullPointerException e) {
    e.printStackTrace();
  } catch (IndexOutOfBoundsException e) {
    e.printStackTrace();
  }
