junit.framework.Assert 클래스에서 이루어지는 모든 메서드 호출을 보고합니다. 이 클래스는 사용이 중단되었으며 이 호출은 org.junit.Assert 클래스의 메서드 호출로 바꿀 수 있습니다.

예:


  import org.junit.*;
  public class NecessaryTest {
    @Test
    public void testIt() {
      junit.framework.Assert.assertEquals("expected", "actual");
    }
  }

빠른 수정이 적용된 후 결과는 다음과 같습니다.


  import org.junit;
  public class NecessaryTest {

    public void testIt() {
      Assert.assertEquals("expected", "actual");
    }
  }
