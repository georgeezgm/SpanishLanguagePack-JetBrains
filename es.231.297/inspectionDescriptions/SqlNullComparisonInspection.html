IS NULL 또는 IS NOT NULL 연산자와 바꿀 수 있는 NULL과의 비교를 보고합니다.

예(Microsoft SQL 서버):

CREATE TABLE foo ( id int );

SELECT * FROM foo WHERE NULL = NULL;
SELECT * FROM foo WHERE NULL != NULL;

NULL = NULL은 IS NULL로 바꿀 수 있으며, NULL != NULL은 IS NOT NULL로 바꿀 수 있습니다. 이 교체를 실행하기 위해 IS NULL 연산자 사용 또는 IS NOT NULL 연산자 사용 빠른 수정을 사용할 수 있습니다.

SELECT * FROM foo WHERE NULL IS NULL;
SELECT * FROM foo WHERE NULL IS NOT NULL;
