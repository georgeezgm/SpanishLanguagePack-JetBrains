배열의 상등 관계를 테스트하기 위해 사용된 연산자 == 및 !=를 보고합니다. 대부분의 경우 배열 내용의 상등 관계 테스트는 의도된 것으로, java.util.Arrays.equals() 메서드를 사용하여 실행할 수 있습니다.

빠른 수정은 ==를 java.util.Arrays.equals()로 바꿀 것을 제안합니다.

예:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

빠른 수정 적용 후:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
