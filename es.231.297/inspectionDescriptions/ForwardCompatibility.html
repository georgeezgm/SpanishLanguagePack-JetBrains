이후 Java 버전에서 컴파일이 불가할 수 있는 Java 코드 구문을 보고합니다.

다음과 같은 문제가 보고됩니다.

  • 식별자로 assert, enum 또는 _이 사용된 경우
  • 타입 이름으로 var, yield 또는 record가 제한된 식별자가 사용된 경우
  • 정규화되지 않은 yield() 메서드 호출
  • module-info.java 내부의 requires java.base 문에 있는 제어자

예:


  // 이 클래스는 이전에는 유효했지만 'yield' 식별자가 금지되면서,
  // Java 14로 컴파일되지 않습니다.
  public class yield {}

해당 이슈를 제때 수정하면 향후 Java 버전으로의 마이그레이션이 간소화될 수 있습니다.
