JSON 배열 양식에서 홑따옴표로 둘러싸인 스트링을 보고합니다.

JSON 배열 양식은 반드시 홑따옴표(')가 아닌쌍따옴표(")로 단어를 둘러싸야 합니다. 그렇지 않으면 Docker 빌드는 실패합니다.

예:


  # 아래 명령어 모두 실패함
  RUN ['/bin/bash', '-c', 'echo hello']
  ADD ['binaryA.jar', 'binary2.jar', 'destination/']
  COPY ['binaryA.jar', 'binary2.jar', 'destination/']

빠른 수정 적용 후:


  RUN ["/bin/bash", "-c", "echo hello"]
  ADD ["binaryA.jar", "binary2.jar", "destination/"]
  COPY ["binaryA.jar", "binary2.jar", "destination/"]
