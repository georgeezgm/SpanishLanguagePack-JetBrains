타입 변수 개수 간의 불일치로 인해 충돌하는 부분 컨테이너 선언을 보고합니다. 타입 변수의 수가 다른 부분 컨테이너를 선언하는 것은 오류입니다.

빠른 수정을 사용하여 선언된 타입 변수의 수를 수정하거나 충돌하는 선언으로 이동할 수 있습니다.

예:

# 클래스 'Example'에 다른 선언과 충돌하는 타입 변수 1개가 있습니다
class Example[A]
end

# 클래스 'Example'에 다른 선언과 충돌하는 타입 변수 2개가 있습니다
class Example[Y, Z]
end

빠른 수정 적용 후:

class Example[A, B]
end

class Example[Y, Z]
end
