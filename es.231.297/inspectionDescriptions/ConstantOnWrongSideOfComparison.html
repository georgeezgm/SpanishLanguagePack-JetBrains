잘못된 쪽에 상수 값이 있는 비교 연산을 보고합니다.

일부 코딩 규칙에서는 비교의 특정한 쪽(왼쪽 또는 오른쪽)에 상수를 배치해야 한다고 지정합니다.

예:


  boolean compare(int x) {
    return 1 > x; // 비교 왼쪽에 상수 '1'이 있습니다
  }

빠른 수정 적용 후:


  boolean compare(int x) {
    return x < 1;
  }

비교할 상수의 위치를 선택하고, null 리터럴이 잘못된 위치에 있으면 경고할지 여부를 선택하려면 검사 설정을 사용합니다.
