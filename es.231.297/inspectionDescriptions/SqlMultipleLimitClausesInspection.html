하나의 쿼리 안에서 행 수를 제한하는 절이 여러 개 사용된 경우를 보고합니다.

예(Microsoft SQL 서버):

create table foo(a int);
select top 1 * from foo order by a offset 10 rows fetch next 20 rows only;

오직 1개의 레코드가 반환되어야 함을 지정하기 위해 SELECT TOP 절이 사용되었습니다. FETCH 절은 OFFSET 절이 처리된 후 반환할 행의 개수를 지정합니다. 하지만 SELECT TOP 제한 절이 이미 있으므로 FETCH 절은 불필요합니다.
