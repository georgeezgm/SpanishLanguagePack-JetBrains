XML 구성 파일 내의 다음과 같은 문제를 보고합니다.
  • 인스턴스화할 수 없는 클래스 참조
  • 필요한 클래스를 확장하지 않는 클래스 참조
  • 부적절한 범위를 가진 클래스 참조
  • 요구되는 패턴(예: Java 식별자)과 일치하지 않는 태그 및 속성 값
  • 필요한 하위 태그 또는 속성을 포함하지 않는 태그
  • 이름이 중복된 객체를 정의하는 태그

예:



<persistence>
    <persistence-unit name="example-persistence-unit">
        <provider>org.example.SomeRandomClass</provider> // 오류: provider 클래스는 PersistenceProvider 인터페이스를 구현해야 합니다
    </persistence-unit>
</persistence>
