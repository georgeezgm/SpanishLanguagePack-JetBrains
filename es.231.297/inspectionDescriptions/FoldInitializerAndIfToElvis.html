이니셜라이저의 elvis 연산자로 변환 가능한 변수가 초기화 직후 null인지 아닌지 검사하는 if 식을 보고합니다.

예:


  fun test(foo: Int?, bar: Int): Int {
      var i = foo
      if (i == null) {
          return bar
      }
      return i
  }

빠른 수정에서는 이니셜라이저가 포함된 if 식을 elvis 식으로 변환합니다.


  fun test(foo: Int?, bar: Int): Int {
      var i = foo ?: return bar
      return i
  }
