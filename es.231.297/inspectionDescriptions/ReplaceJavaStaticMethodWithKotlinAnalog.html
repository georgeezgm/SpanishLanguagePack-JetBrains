Kotlin 함수와 대체할 수 있는 Java 메서드 호출(예: System.out.println())을 보고합니다.

코드를 치환하여 Java 의존성을 제거하고, 관용적인 Kotlin 코드를 만듭니다.

빠른 수정에서는 동일한 Kotlin 호출에 대한 Java 메서드 호출을 치환합니다.

예:


  import java.util.Arrays

  fun main() {
      val a = Arrays.asList(1, 3, null)
  }

빠른 수정 적용 후:


  fun main() {
      val a = listOf(1, 3, null)
  }
