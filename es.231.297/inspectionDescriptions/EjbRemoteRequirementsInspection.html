다음과 같은 EJB 사양 위반 사례를 보고합니다.
  • 메서드가 java.rmi.RemoteException을 던진다고 선언하지 않는 경우
  • 메서드 반환 타입 또는 인수가 RMI/IIOP와 호환되지 않는 경우
  • EJB 서비스 엔드포인트 인터페이스가 상수 선언을 포함하는 경우

예:


public interface EntityRemote extends javax.ejb.EJBObject{
  public int businessExample(String value) throws java.lang.RuntimeException; // 오류: 'RemoteException'을 선언해야 합니다
}
