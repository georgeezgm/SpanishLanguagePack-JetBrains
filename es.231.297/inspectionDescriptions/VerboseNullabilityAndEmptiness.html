하나의 검사로 단일화할 수 있는 null 및 비어 있는지 확인하는 검사의 조합을 보고합니다.

빠른 수정에서는 강조 표시된 검사를 isNullOrEmpty()와 같은 통합된 검사 호출로 대체합니다.

예:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }

빠른 수정 적용 후:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println("List is empty!")
      } else {
          println(list.joinToString())
      }
  }
