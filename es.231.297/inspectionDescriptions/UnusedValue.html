대입된 후 한 번도 사용되지 않는 변수 값을 보고합니다. 이는 다음과 같은 경우에 발생합니다.
  • 변수가 대입 후 한 번도 읽히지 않음
  • 값이 읽기를 위해 액세스되기 전에 다른 대입에 의해 덮어쓰기됨
  • (위의 두 가지 이유 중 하나로) 변수 이니셜라이저가 불필요합니다
  • 변수 자체가 한 번도 사용되지 않음

예:


  class C {
    int field;

    void foo() {
      field = unused_value;
    }

    void bar() {
      use(field);
      foo();
    }
  };
