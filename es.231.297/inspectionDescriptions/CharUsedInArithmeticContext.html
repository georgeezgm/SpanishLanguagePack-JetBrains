더하기 또는 빼기 표현식에 사용된 char 타입의 표현식을 보고합니다.

이러한 코드가 반드시 문제인 것은 아니지만 버그가 발생할 수 있습니다(예: 문자열이 필요한 경우).

예: int a = 'a' + 42;

빠른 수정 적용 후: int a = (int) 'a' + 42;

String 컨텍스트의 경우: 

int i1 = 1;
int i2 = 2;
System.out.println(i2 + '-' + i1 + " = " + (i2 - i1));

빠른 수정 적용 후:

System.out.println(i2 + "-" + i1 + " = " + (i2 - i1));
