public 중첩 클래스를 보고합니다.

예:


  public class Outer {
    public static class Nested {} // 경고
    public class Inner {}         // 경고
    public enum Mode {}           // 설정에 따른 경고
    public interface I {}         // 설정에 따른 경고
  }

검사 구성:

  • 'public' 내의 열거형 무시 옵션을 사용하여 public 내의 열거형을 무시합니다.
  • 'public' 내의 인터페이스 무시 옵션을 사용하여 public 내의 인터페이스를 무시합니다.
