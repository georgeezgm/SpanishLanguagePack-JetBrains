EnumMap 생성자 호출로 바꿀 수 있는 hashMapOf 함수 또는 HashMap 생성자 호출을 보고합니다.

EnumMap 생성자를 사용하면 코드가 더 단순해집니다.

빠른 수정에서는 함수 호출을 EnumMap 생성자 호출로 바꿉니다.

예:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = hashMapOf()

빠른 수정 적용 후:


  enum class E {
      A, B
  }

  fun getMap(): Map<E, String> = EnumMap(E::class.java)
