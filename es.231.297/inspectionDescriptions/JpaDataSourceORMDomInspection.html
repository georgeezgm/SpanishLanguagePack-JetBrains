구성된 데이터 소스 내부에서 해결되지 않는 Persistence ORM XML 설명자를 보고하고 데이터 소스를 새로고침하거나 다른 데이터 소스를 할당할 것을 제안합니다.

데이터베이스 테이블 예:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
  );

JSP 파일 예:


  <?xml version="1.0" encoding="UTF-8"?>
  <entity-mappings xmlns="http://java.sun.com/xml/ns/persistence/orm" version="1.0">
    <entity name="JavaEntity" class="JavaEntity">
        <table name="UNKNOWN_TABLE"/> // 오류: 해결되지 않은 테이블
        <attributes>
          <id name="id">
            <column name="ID" nullable="false"/>
          </id>
          <basic name="UNKNOWN_COLUMN"> // 오류: 해결되지 않은 열
            <column name="UNKNOWN_COLUMN" length="12"/>
          </basic>
        </attributes>
    </entity>
  </entity-mappings>

참고: 'Assign datasource'(데이터 소스 할당) 액션을 통해 데이터 소스를 구성할 수 있습니다
