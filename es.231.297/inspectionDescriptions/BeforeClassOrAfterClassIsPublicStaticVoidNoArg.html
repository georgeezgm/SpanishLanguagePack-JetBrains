public 및 static으로 선언되지 않거나, void를 반환하거나 인수를 취하지 않는 JUnit 4.0 @BeforeClass 또는 @AfterClass 메서드를 보고합니다. 해당 메서드는 실수로 생성되기 쉽지만 JUnit 테스트 러너를 통해 실행되지 않습니다.

예:


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) {
    }
  }

빠른 수정이 적용된 후 메서드는 다음과 같이 변경됩니다.


  class MainTest {
    @BeforeClass
    public static void beforeClass() {
    }
  }
