@Nls 어노테이션이 추가되어 있으며 대부분의 플랫폼 UI 가이드라인에 존재하는 대문자 표기 규칙을 준수하는 대문자 표기 매개변수를 포함하는 매서드 매개변수 및 반환 값 내 문자열을 보고합니다.

예:


  void setTitle(@NlsContexts.DialogTitle String title) {}
  setTitle("This is sentence capitalization but should be title");

빠른 수정 적용 후:


  setTitle("This Is Sentence Capitalization but Should Be Title");
