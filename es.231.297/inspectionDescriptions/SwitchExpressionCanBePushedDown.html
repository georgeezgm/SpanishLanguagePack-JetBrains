브랜치가 공통된 하위식을 포함하므로 switch가 내부로 이동될 수 있는 switch 식 및 구문을 보고합니다. 이 액션은 코드를 단축합니다. 많은 경우 결과 switch 식을 별도의 변수나 메서드로 빼내는 것이 좋습니다.

예:


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

빠른 수정 적용 후:


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

이 검사는 화살표 문법으로 강화된 switch에만 적용됩니다.
