사용되지 않는 선언을 보고합니다.

빠른 수정에서는 사용되지 않는 선언을 제거합니다.

아래 옵션에서 검사가 공용 및/또는 로컬 선언에 대해 작동해야 하는지 여부를 제어할 수 있습니다. 로컬 선언의 경우 검사 활성화 및 비활성화 외에도 다음 세 가지 컴파일러 옵션 중 하나의 존재에 종속되게 만들 수도 있습니다: -Wunused:locals, -Wunused:linted 또는 -Xlint:unused.

예:


  def test(): Int = {
    val unused = 3
    3
  }

빠른 수정 적용 후:


  def test(): Int = {
    3
  }
