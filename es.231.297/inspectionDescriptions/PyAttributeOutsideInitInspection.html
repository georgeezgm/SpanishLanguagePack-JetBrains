인스턴스 속성 정의가 __init__ 메서드 외부에 있는 경우 문제를 보고합니다.

예:

    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

빠른 수정이 적용되면 코드 샘플은 다음과 같이 변경됩니다: 

    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
