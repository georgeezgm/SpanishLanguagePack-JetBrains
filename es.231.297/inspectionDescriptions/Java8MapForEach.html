for(Entry<?,?> entry : map.entrySet()) {...} 또는 map.entrySet().forEach(entry -> ...)를 map.forEach((key, value) -> ...)로 대체하도록 제안합니다.

예


  void print(Map<String, Integer> map) {
    map.entrySet().forEach(entry -> {
      String str = entry.getKey();
      System.out.println(str + ":" + entry.getValue());
    });
  }

빠른 수정 적용 후:


  void print(Map<String, Integer> map) {
    map.forEach((str, value) -> System.out.println(str + ":" + value));
  }

루프를 보고하지 않음 옵션이 활성화된 경우 entrySet().forEach() 사례만 보고됩니다. for 루프에도 빠른 수정 액션이 지원됩니다.

이 검사는 프로젝트 또는 모듈의 언어 수준이 8 이상인 경우에만 보고합니다.
