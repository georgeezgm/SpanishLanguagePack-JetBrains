'plugins {}' 블록 앞에 허용되지 않는 구문을 탐지합니다.

Gradle Plugin DSL의 제한 때문에 'plugins {}' 블록 앞에 제한된 Groovy 구문 집합만 사용할 수 있습니다. 유일한 옵션은 'buildscript {}', 'pluginManagement {}' 및 다른 'plugins {}'입니다.
Gradle 문서를 참조하세요.

예:


import foo.bar.Baz
plugins {} // reports 'plugins'
plugins {
  foo() // reporst 'foo'
  id 'java'
}
