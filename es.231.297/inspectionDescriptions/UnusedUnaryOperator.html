사용되지 않는 표현식에 있는 숫자 타입에 대한 단항 연산자를 보고합니다.

단항 연산자를 괄호 없이 사용하면 이전 표현식을 망가뜨립니다. 결과적으로 여러 줄로 구성된 수학적 표현식에서 오해의 소지가 있습니다.

예:


  fun main() {
      val result = 1 + 2 * 3
                  + 3              // <== '+ 3'이 'result' 변수에 속하지 않으며 사용되지 않습니다
      println("Result = $result")  // 결과는 기대 값인 '10'이 아니라 '7'입니다
  }
