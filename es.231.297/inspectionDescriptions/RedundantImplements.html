클래스의 implements 목록의 인터페이스 또는 상위 클래스에 의해 이미 구현되었거나 해당 클래스의 상위 인터페이스에 의해 확장된 인터페이스의 extends 목록을 보고합니다. 그러한 선언은 불필요하며 안전하게 제거할 수 있습니다.

예:


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

빠른 수정 적용 후:


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

extends 또는 implements 목록의 Serializable 또는 Externalizable에 대해 보고하지 않으려면 이 옵션을 사용합니다.
