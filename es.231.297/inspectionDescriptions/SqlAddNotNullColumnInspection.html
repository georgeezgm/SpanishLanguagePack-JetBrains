DEFAULT 값 없이 NOT NULL 열을 생성하려는 시도를 보고합니다.

예(Microsoft SQL 서버):

CREATE TABLE foo (a INT, b  INT)

ALTER TABLE foo ADD c INT NOT NULL;

디폴트로 열은 NULL 값을 가집니다. 이 예에서는 열이 NULL 값을 수락하지 않도록 강제하는 NOT NULL 제약을 사용합니다. NULL 값 사용을 금지할 경우, 새 레코드 생성 시 SQL이 사용할 수 있는 DEFAULT 값을 설정해야 합니다. 

ALTER TABLE foo ADD c INT NOT NULL DEFAULT 42;

디폴트 값 추가 빠른 수정을 사용하여 DEFAULT 값을 빠르게 추가할 수 있습니다.
