읽기 전용 Collection에 있는 증분 대입(+=) 표현식을 보고합니다.

읽기 전용 <Collection의 증분 대입(+=) 표현식은 새 컬렉션을 임시로 할당하므로 성능이 저하될 수 있습니다.

타입을 가변으로 변경 빠른 수정을 사용해 코드를 자동으로 수정할 수 있습니다.

예:


  fun test() {
      var list = listOf(0)
      list += 42 // 여기에 새 목록이 할당되어 있습니다. list = list + 42
  }

빠른 수정 적용 후:


  fun test() {
      val list = mutableListOf(0)
      list += 42
  }
