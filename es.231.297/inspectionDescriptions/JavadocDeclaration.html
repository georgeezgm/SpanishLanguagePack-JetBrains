다음 문제가 있는 Javadoc 주석 및 태그를 보고합니다.
  • 잘못된 태그 이름
  • 불완전한 태그 설명
  • 중복된 태그
  • 누락된 Javadoc 설명

예:


  /**
   * 잘못된 태그 이름
   * @poram param description
   */
  public void sample(int param){
  }

예:


  /**
   * 자기 참조 {@link #sample(int)}
   */
  public void sample(int param){
  }

빠른 수정에서는 알 수 없는 Javadoc 태그를 사용자 정의된 추가 태그 목록에 추가합니다.

추가 Javadoc 태그를 정의하려면 아래 텍스트필드를 사용하세요.

중복된 'throws' 태그를 무시하려면 첫 번째 체크박스를 사용하세요.

누락되거나 불완전한 첫 문장이 설명에 있는 문제를 무시하려면 두 번째 체크박스를 사용하세요.

자신을 가리키는 참조를 무시하려면 세 번째 체크박스를 사용하세요.
