me, this, self와 같은 리시버 이름을 보고하거나 이 타입의 다른 리시버 이름과 상이한 이름을 보고합니다.

예:

func (self *MeterSnapshot) Rate5() float64 { return math.Float64frombits(self.rate5) }

github.com/golang의 Receiver Names에 따르면, 'me', 'this' 또는 'self'와 같은 일반적 이름을 사용해서는 안 됩니다. 이러한 식별자는 객체 지향 언어에서 전형적으로 쓰이며 메서드에 특별한 의미를 부여할 수 있습니다.
