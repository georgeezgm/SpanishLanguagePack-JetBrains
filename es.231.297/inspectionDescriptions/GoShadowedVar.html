외부 스코프에 있는 선언과 겹치는 변수 선언을 보고합니다.

이 경우 변수의 의미는 스코프에 따라 달라지므로, 혼란을 줄 수 있으며 의도하지 않은 결과로 이어질 수 있습니다.

예:

for i := 0; i < len(nums); i++ {
    for i := 0; i < len(nums); i++ {
    }
  }

내부 루프의 변수 i가 가려집니다. 이를 방지하기 위해 내부 루프의 변수 이름 변경을 고려하세요.

for i := 0; i < len(nums); i++ {
    for j := 0; j < len(nums); j++ {
    }
  }
