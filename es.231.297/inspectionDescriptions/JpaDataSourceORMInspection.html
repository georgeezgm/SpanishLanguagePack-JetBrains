다음 Persistence ORM 어노테이션 내부의 참조가 구성된 데이터 소스 안에서 해결될 수 없는 경우를 보고합니다.
  • Table
  • Column
  • [Any]To[Any]
  • JoinColumn / JoinTable
  • NamedQuery / NativeQuery
  • Embeddable

데이터 소스를 새로고침하거나 다른 데이터 소스를 할당할 것을 제안합니다.

데이터베이스 테이블 예:


  CREATE TABLE DATA_TABLE (
    ID INTEGER,
    DATA VARCHAR(100)
);

JPA 엔티티 클래스 예:


  @Entity
  @Table(name = "UNKNOWN_TABLE") // 해결되지 않은 참조
  public class JavaEntity {
    @Column(name = "ID") // 정의되지 않은 테이블 내 열을 해결할 수 없음
    Integer id;
  }

참고: '데이터 소스 할당' 액션 또는 엔티티 클래스에 해당하는 여백 아이콘을 통해 데이터 소스를 구성할 수 있습니다
