if, while, for 또는 do 문 또는 조건식에서 조건으로 사용되는 대입을 보고합니다.

의도된 경우도 있으나 이러한 사용은 혼란을 일으키며, 오타를 나타낼 수 있습니다(예: == 대신 = 사용).

빠른 수정은 =를 ==로 바꿉니다.

예:


  void update(String str, boolean empty) {
    // 경고: 'empty'가 str.isEmpty()와 비교되지 않고
    // 재대입되었습니다
    if (empty = str.isEmpty()) {
      ...
    }
  }

빠른 수정 적용 후:


  void update(String str, boolean empty) {
    if (empty == str.isEmpty()) {
      ...
    }
  }
