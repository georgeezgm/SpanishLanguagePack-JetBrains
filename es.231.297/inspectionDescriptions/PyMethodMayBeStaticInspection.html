클래스 인스턴스를 생성할 필요가 없어 static으로 만들 수 있는 모든 메서드를 보고합니다.

예:

class MyClass(object):
    def my_method(self, x):
        print(x)

메서드로부터 함수 생성 빠른 수정이 적용되면 코드는 다음과 같이 변경됩니다:

def my_method(x):
    print(x)


class MyClass(object):
    pass

메서드를 정적으로 만들기 빠른 수정을 선택한 경우 @staticmethod 데코레이터가 추가됩니다:

class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
