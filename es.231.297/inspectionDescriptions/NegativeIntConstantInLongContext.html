long 컨텍스트에서 음수인 int 16진수 상수를 보고합니다. 이러한 상수는 묵시적으로 long으로 확장되어, 상위 비트가 0이 아닌 1이 됩니다(예: 0xFFFF_FFFF는 0xFFFF_FFFF_FFFF_FFFFL이 됩니다). 이는 의도되었을 가능성이 낮으며, 의도되었다 하더라도 명시적 long 상수를 사용하는 것이 더 명확합니다.

예:


  // 경고: 이는 int 상수 -1이며 long으로 확장되어
  // 0xFFFF_FFFF_FFFF_FFFFL이 됩니다.
  long mask = 0xFFFF_FFFF;
