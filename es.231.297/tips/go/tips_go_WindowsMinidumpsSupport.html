Windows minidump 파일은 실행 중이거나 충돌 시점의 프로그램의 상태를 기록합니다. 미니 덤프 파일을 읽으려면 애플리케이션의 바이너리와 덤프 파일이 있어야 합니다.

GoLand에서 미니 덤프를 열고 탐색할 수 있습니다. 미니 덤프를 열려면 실행 | 코어 덤프 열기 로 이동합니다. 실행 파일과 덤프 파일이 있는 경로를 지정합니다. 확인 을 클릭합니다.

참고: Microsoft Windows 운영 체제에서만 Windows 미니 덤프를 생성할 수 있습니다. 그러나 지원되는 모든 운영 체제에서 애플리케이션을 분석하고 디버그할 수 있습니다.
