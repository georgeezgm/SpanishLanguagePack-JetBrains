Consolas de bases de datos, archivos reutilizables y buffers reutilizables

Consolas de bases de datos son archivos SQL que ya están adjuntos a una fuente de datos. Cuando crea una fuente de datos, se crea automáticamente una consola de base de datos. Para abrir la consola, haga clic derecho en la fuente de datos y seleccioneNavegación | Saltar a la consola de consultas | Abrir consola predeterminada. Para crear una nueva consola, haga clic derecho en la fuente de datos y seleccioneNuevo | Consola de consultas.

Archivos borrador Son archivos SQL que no están adjuntos a una fuente de datos. Consulte los archivos borrador como notas temporales o borradores para obtener algunas ideas de código fuera del contexto del proyecto. Para crear un archivo borrador, navegue hastaArchivo | Nuevo | Archivo borrador (o presione&shortcut:NuevoArchivoScratch;).

Buffers de raspado Son archivos de texto simples sin ninguna función de asistencia de codificación. Los buffers de scratch se pueden utilizar para listas de tareas simples y notas para usted mismo. No hay un elemento de menú dedicado para la acción de crear un nuevo buffer temporal, pero puedes usar elBuscar acción surgir (&shortcut:GotoAction;) y ejecute elNuevo buffer temporal acción.
