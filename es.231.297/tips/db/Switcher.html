Conmutador para navegación

Usa el conmutador&atajo:Conmutador; para navegar entre archivos abiertos y ventanas de herramientas. MantenerControl presionado para dejar abierta la ventana emergente del conmutador. Para la navegación, utilicePestaña yMayús+Tab.

PrensaBorrar oRetroceso para cerrar una pestaña del editor u ocultar una ventana de herramientas.
