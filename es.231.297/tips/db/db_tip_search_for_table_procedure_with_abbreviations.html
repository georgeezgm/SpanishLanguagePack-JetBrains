Buscar una tabla o un procedimiento

Cuando busca una tabla o un procedimiento con&shortcut:GotoClass;, puedes escribiresquema.tabla ofuente.de.datos.procedimiento en el campo de búsqueda. Además, la búsqueda funciona cuando se acortan las palabras. Por ejemplo, para buscar elactor tabla en una fuente de datos llamada PostgreSQL, escribapost.actor.
