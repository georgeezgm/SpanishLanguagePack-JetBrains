DDL, datos y ubicación del objeto de base de datos enExplorador de bases de datos

Prensa&shortcut:Ir aDeclaración; para abrir el DDL del objeto seleccionado. Por ejemplo, si presiona este acceso directo en el nombre de una tabla en la instrucción SELECT, verá el DDL de esta tabla (la instrucción CREATE TABLE).

Prensa&shortcut:EditarFuente; para abrir los datos del objeto. Por ejemplo, si presiona este acceso directo en el nombre de una tabla en la instrucción SELECT, verá el contenido de la tabla.

PrensaAlt+Mayús+B para navegar hasta el objeto en elExplorador de bases de datos ventana de herramientas.
