Volcar a fuente de datos DDL

La fuente de datos DDL es una fuente de datos virtual con un esquema basado en varios scripts SQL. Almacenar estos archivos en el Sistema de control de versiones (VCS) es una forma de mantener su base de datos bajo el VCS.

Para crear una fuente de datos DDL, haga clic derecho en una fuente de datos en elExplorador de bases de datos y seleccioneImportación/Exportación | Volcar a fuente de datos DDL.
