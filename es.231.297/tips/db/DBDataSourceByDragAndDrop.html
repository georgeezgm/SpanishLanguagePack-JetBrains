Archivos SQL como fuente de datos DDL

Los archivos SQL que contienen declaraciones CREATE se pueden utilizar como fuentes de datos. Como resultado, puede crear diagramas, hacer referencia a todas las tablas, columnas y otros objetos definidos en dichos archivos en el editor.

Para crear una fuente de datos DDL, arrastre el archivo DDL a laExplorador de bases de datos (Ver | Ventanas de herramientas | Explorador de bases de datos) desde su administrador de archivos.
