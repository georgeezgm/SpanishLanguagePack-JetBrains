Buscar en todas partes

PrensaCambio dos veces para buscar archivos, acciones, símbolos, elementos de la interfaz de usuario, ramas de Git y comentarios en todo el proyecto. Presionando dobleCambio Nuevamente, ampliará la búsqueda a elementos que no sean del proyecto.

Utilice pestañas o atajos directos&shortcut:GotoClass; para objetos de base de datos,&shortcut:Ir a archivo; para archivos,&shortcut:GotoSymbol; para símbolos y&shortcut:GotoAction; para acciones que limiten los resultados de su búsqueda.
