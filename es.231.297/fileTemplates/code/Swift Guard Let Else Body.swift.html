선택적 값을 guard 문으로 래핑 해제한 후 else 블록을 채웁니다. 템플릿을 편집할 수 있습니다. Swift 표현식 및 주석과 함께 그 대응하는 값으로 확장되는 사전 정의된 변수를 사용할 수 있습니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${RETURN_TYPE}   주변 함수의 반환 유형(있는 경우)
${DEFAULT_RETURN_VALUE}   제안된 디폴트 반환 값(빈 값일 수 있음)
Swift File Header 템플릿(Includes 탭)의 모든 사전 정의된 변수도 사용 가능합니다. (예: ${USER}, ${DATE} 등)
