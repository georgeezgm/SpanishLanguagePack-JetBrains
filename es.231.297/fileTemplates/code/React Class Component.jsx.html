구성 요소 추출 리팩터링과 같이, React 클래스 구성 요소가 생성될 때 사용되는 기본 제공 템플릿입니다. 템플릿을 사용자 정의할 수 있습니다. 예를 들어, PureComponent 대신 Component를 사용하거나 prop 타입 정의를 변경할 수 있습니다. 사전 정의 변수도 사용할 수 있습니다. 이러한 변수는 템플릿이 적용될 때 해당 값으로 바뀝니다.
사전 정의된 변수는 다음의 값을 사용합니다:
${NAME}   구성 요소 이름
${COMPONENT_BODY}   구성 요소 본문
${PROP_TYPES}   속성 타입 정의 (속성-타입 모듈 사용)
${PROP_TYPES}   속성 값이 제공되지 않을 때 React가 사용할 기본 속성 값
